 <p>
  明年1月11号，台湾除了要选出新总统外，也要选出新国会。2020年立委参选情况热烈，登记政党数量与参选人总数双双创下历史新高。许多小党选择与大党合作，在部份区域，立委以不提名相互礼让，但到了最后关键期，弃保声响起，小党纷纷喊出“亡柯感”、“亡昌感”，如何在蓝绿夹杀中活下来，小党都各出奇招。
 </p>
 <p>
  2020年台湾选举分为总统、立委、政党票三张选票。立委分为区域立委和不分区立委，区域立委是在各划分的选区内竞争单一名额，不分区立委则根据政党票得票比例分配席次。
 </p>

  中选会表示，2020年共有412位区域立法委员候选人完成登记。而全国不分区及侨居国外国民立委选举，有19个政党提出共217位候选人名单，为史上最高。
  <br/>
  <br/>
  台湾6家主要民调显示，政党票方面，国民党支持度原本领先，后来因为国民党推出的不分区名单，出现有曾到中国大陆听习近平训话的亲中退役将领，及声援香港暴警的女教授，爆发争议，政党票支持度跌到民进党之后。而由台北市长柯文哲组党的台湾民众党则位居第三。太阳花运动后兴起的最强第三势力时代力量，则因内讧分裂，支持度跌到第四，亲民党则在宋楚瑜宣布参选总统后，支持度向上爬升。

  主张台湾主权的小党如时代力量、台联、台湾基进、一边一国、喜乐岛联盟等，不只面对同质性高导致的自相残杀局面，部份在跟大党合作礼让求胜后，到最后被“弃保”，出现生存危机。民进党就已经公开呼吁选民，想投小党的人要“大义灭亲”，也就是总统、立委、政党“三张票”都要支持民进党，集中选票。
  <br/>
  <br/>
  <b>
   时代力量V.S.民众党割喉战 “亡昌感”对上“亡柯感”
  </b>
 </p>
 <p>
  <br/>
  小党之间因此相互厮杀。民众党台北市长柯文哲的心腹蔡璧如近日放话说，时代力量支持度只有不到5％，排不分区名单第四名的黄国昌，根本不会上，只会浪费票，票应让给台湾民众党。而台湾民众党也定调要打“亡柯感”，也就是如果民众党选不好，柯文哲政治生命也无法延续。柯文哲11号就首度松口坦言，2024年的总统大选，如没意外的话就会参选。

  </b>
  <br/>
  <br/>
  台联党文宣部主任潘厚勋接受自由亚洲电台采访时说：“台联从2001年由李登辉前总统催生、成立，到现在已经有19年历史，曾经台联在立法院有席次，表现也非常出色。这次选情，蓝绿两大党竞争激烈，台联也是夹缝中求生存，我们能做的就是不断打真正效忠台湾、守卫台湾，来突显台联的特色。”

  台媒报导，时力估计席次“坐5望6抢7”；民众党评估至少可拿4席；亲民党在总统参选人宋楚瑜的母鸡效应下，喊出“拚6席抢8席”。统派新党认为至少拿到2席，目标则是拚到4席。
  <br/>
  <br/>
  由于第三势力异军突起，各方预估，蓝绿两大党国会过半难度高，可能出现民进党、国民党、民众党、时代力量都跨过门槛，成为“四党不过半”，或是加上亲民党成为“五党不过半”，第三势力将扮演关键角色。

  </b>

  <br/>
  不分区立委的席次如何评估？台湾师范大学政治学系教授范世平接受自由亚洲电台采访就说：“国民党现在比较麻烦，因为他本来安全名单应该排在14名，最近因为国民党整个形象的问题，加上韩国瑜的民调一直下滑，就变成国民党的支持者会不会去投票，现在是个关键。”
  <br/>
  <br/>
  至于民进党的不分区立委席次是否可能过半？范世平说：“
  <br/>
  （前民进党大老）沈富雄这两天的说法是会过半，但是我觉得没有那么简单，我觉得在边缘，我认为不会过。”
