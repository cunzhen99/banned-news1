 <p>
  香港反送中运动中，多次大规模抗议游行的组织者民间人权阵线的总召集人岑子杰在香港街头被人用榔头袭击，震惊香港社会。日前，民阵获得美国的“公民力量奖”。前来领奖的民阵前副召集人梁颖敏接受了本台记者郑崇生的专访，谈到他们的忧虑、期待和反送中运动今后的可能走向。
 </p>
 <p>
  记者：反送中运动走到现在，对于示威者而言你们现在最大的关切和担忧是什么？
  <br/>
  <br/>
  梁颖敏：我们的召集人岑子杰先生在前几天受到暴力对待。这已经是他第2次遭遇暴力，而且这一次是有人拿铁锤打他的头，这可是会致命的袭击。为什么香港会有这样的袭击，而且不是对岑子杰一个人，还有我们在香港的别的示威者都遭遇这样的对待？
  <br/>
  <br/>
  现在香港的政府、香港的警察没有把那些打人的暴徒绳之以法。他们也没有对警察内部一些有暴力行为的前线警察（问责）。所以他们没有做到他们应该做的事情，所以我觉得会令这些袭击行为越来越多。（我们获得）这个奖是一个很好的时机，因为我们很希望世界可以对香港政府、香港警察说，你们以为保护那些亲北京亲政府的暴徒就可以让他们没有后顾之忧，但是世界就是在看着香港。美国国会众议院已经通过一些法案保护香港。所以无论是这个奖项或者是美国国会近来的举动都是在发出同一个讯息，就是说世界在看着香港，美国也在看。
 </p>
 <p>
  <br/>

 <p>
  记者：美国国会众议院已经通过了《香港人权与民主法案》。那你觉得接下来参议院通过的几率大吗？
  <br/>
  <br/>
  梁颖敏：我之前有跟一些参议员见过面。从我得来的讯息，我觉得参议院通过法案的几率还蛮大的。但是一天没有通过，我们一天都不能够松懈，所以我们这次来美国，主要就是跟不同的政治人物去见面，去解释香港的情况。我觉得只要他们能够充分了解香港的情况，帮助整个世界去维护我们香港作为在中国内唯一一个自由、法治和民主的桥头堡（的地位）是非常重要的。我觉得只要他们了解这一点他们都会支持香港，也会帮助通过那个法案。
  <br/>
  <br/>
  记者：岑子杰受到攻击后，对民阵日后的发展以及运动的发展方向会有什么样的影响？
  <br/>
  <br/>
  梁颖敏：岑先生其实已经对外发了一个声明。他除了鼓励我们继续这个运动之外，还鼓励我们不要为他担心，更不要因为将对他的担心变成愤怒而去攻击别的人，因为我们很希望继续维持民阵是一个和平的抗议组织。我们希望能够从一个和平的方向去实行这个运动。从8月31号以来（我们）申请的所有游行都被拒绝了，但是我们一直还在持续努力去申请，还有上诉。我们希望在制度里面我们尽量用尽我们的方法去申请，但是如果我们真的不幸被拒绝的话，我们看到香港人还是会不怕困难，不怕危险，继续走出来。我相信我们香港人未来还是会维持这份勇敢，还会越来越勇敢。我们希望全世界都会跟香港站在一起，跟我们站在一起。
  <br/>
  <br/>
  记者：郑崇生 责编：申铧 网编：郭度
