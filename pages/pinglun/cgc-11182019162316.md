 <p>
  香港的反专制抗争持续五个多月了。在此期间，面对中共的暴行，港人团结、有智慧、和平无畏的决绝抗争，让全世界刮目相看，敬佩有加。
  <br/>
  <br/>
  中共在香港对抗议者从殴打、谋杀、纵容黑警强奸到直接实弹开枪射杀的现场实况视频传遍全球。尤其是包围攻打大学，充分暴露了共产专制暴政的丑恶嘴脸。善良的西方普通人过去对中共的邪恶根本无法想象，发生在香港的一切帮助他们刷新了认知。如今，多种应对共产暴政危害人类文明的方案正在世界各个民主国家的政府、立法机关及民间智库中酝酿着，中共受到惩治的日子不久就会来临。
  <br/>
  <br/>
  从2019年11月12日开始，中共指使港府组织全副武装的防暴队进攻香港中文大学，随即又对其他大学也展开攻打。尤其是在刚刚过去的周末对香港理工大学的围困攻打中，就连前来救助伤员的医护人员也不放过的恶行更加令人愤慨。包围大学的恶警竟然疯狂地喝斥、驱赶前来调停的立法会议员们，更令世人震惊不已。
  <br/>
  <br/>
  一个政权出动武装围困攻打大学，这样的事情空前绝后，就连当年二战时德国纳粹攻佔欧洲、轰炸英国时也没有做过，可是今天的中国共产党在香港做了！在全校师生坚决阻击和香港各界市民的及时声援支持下，其强佔中文大学的阴谋未能得逞。
  <br/>
  <br/>
  然而，共匪爪牙攻打香港理工大学时不仅动用了水炮车，还出动了装甲车、动用了声波武器，而学生们只能无奈地用燃烧瓶和弓箭反击。遭共匪层层围困的学生们多次突围均未能成功，共匪爪牙扬言说，所有理工大的学生都将被抓捕。尽管香港民众自发地开着私家车在外围又包围了中共恶警，并成片的响起了愤怒的车喇叭声，可是善良的民众终不能抵挡野蛮的共匪爪牙，理工大学遭劫了。从视频可以见到大学周围的街道都是一片烟雾弥漫，可想而知，中共爪牙们释放了多少催泪瓦斯！
  <br/>
  <br/>
  今天的好消息是香港高院刚刚作出了“《禁止蒙面法》违宪”的裁决。据此可以追究林郑月娥枉法滥权的责任。另外，美国参议院外交委员会主席卢比奥曾发推说，《香港民主与人权法案》最快有可能今天会在参议院获得通过。侭管如此，香港的形势刻不容缓，需要国际社会立即采取有效的具体行动，制止中共在香港施暴。
  <br/>
  <br/>
  国际社会面对毫无底线连大学都要攻打的灭绝人性、连纳粹都不如的邪恶中共政权，仅有能决定香港国际金融地位的《香港民主与人权法案》去针对中共对人权的迫害，以及有对中共进行惩罚的《马格尼茨基人权问责法》是不够的。应该推动全球民主国家形成共识，将纳粹不如的中共政权列入反人类组织，联合国应批准将其交海牙国际法庭以反人类罪进行审判。
