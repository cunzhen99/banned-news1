 <p>
  在中国政府不断压制维权律师的情况下，中国举办首届“世界律师大会”的消息引发多个国际团体及个人的联署抗议。有评论说，北京官方此举颇有“往自己脸上贴金”的意思。
  <br/>
  <br/>
  由中华全国律师协会主办的“世界律师大会”星期一在广州开幕。约800名来自57个国家的法律界代表到场参加。据司法部介绍，会议旨在“更好地宣传展示中国法治建设和律师事业发展取得的历史性成就”。
  <br/>
  <br/>
  同日，香港“中国维权律师关注组”连同无国界律师组织、德国律师协会、日内瓦律师协会、国际律师协会人权部、台湾民间司法改革基金会等21个团体和个人发表公开信，批评中国利用其法律制度滥捕、任意拘留以及强迫失踪数以百万计的人，并阻止他们获得法律援助及公平公正的审讯。
  <br/>
  <br/>
  广东维权律师隋牧青告诉记者，这场与会的中国代表都是政府的官员以及与政府关系密切的人员，普通律师很难有机会参加。
 </p>
 <p>
  <br/>
  <div class="image-inline captioned" style="width:800px;">
   <div style="width:800px;">
    <img alt="隋牧青一家（受访者独家提供）" src="https://www.rfa.org/mandarin/yataibaodao/renquanfazhi/yf1-09212018095813.html/968b726797524e005bb6-53d78bbf800572ec5bb663d04f9b.JPG" title="隋牧青一家（受访者独家提供）"/>
   </div>

    </span>

 </p>
 <p>
  “听（中国）政府话的那些人（才可以参加）。人权律师或者是稍微对当局持批评态度的人都是不可能去参加。”
  <br/>
  <br/>
  他还提到，人权律师更是在大会举行期间成为被维稳的对象。
  <br/>
  <br/>
  “我原来以为开会期间我到外地也算是一种自我放逐。我都到了机场，快要登机了就被（警察）拦截下来。等到明天（10日）会议基本结束了我才可以外出。”
  <br/>
  <br/>
  香港的“中国维权律师关注组”9日的联署信详细列举包括高智晟、王全璋、余文生、江天勇等多位维权律师被迫失踪、长期羁押、酷刑、强迫认罪、全天候监视的情况。另外还提到官方使用行政处罚手段对付代理政治敏感案件的律师，例如吊销他们的律师执照。
  <br/>
  <br/>
  隋牧青律师就是因接手“六四天网”创办人黄琦的案件而在去年突然被吊销律师执业证。
  <br/>
  <br/>
  他认为，中国作为其中一个对人权律师迫害最严重的国家承办律师大会无疑是一个很大的讽刺。
  <br/>
  <br/>
  “中国是一个践踏律师职业最严重的国家。我相信中国拥有最多被吊牌、注销执业证的律师。抓捕律师等情况也是最严重。”
  <br/>
  <br/>
  与此同时，中国著名维权律师高智晟的妻子耿和也向参与这次“世界律师大会”的各国代表喊话。
 </p>
 <p>
  <br/>
  <img class="image-inline captioned" src="../../../../resolveuid/365f610d-a4ae-41d3-b44d-c8d1736443c7"/>
 </p>
 <p>
  “在中国，律师事务所可以被随便关门，律师可以被随意吊照、抓捕、失踪、酷刑。在这样一个国家举行律师大会是很讽刺的，而且还在世界人权日（12月10日）举办，这体现出中国政府的傲慢，对国际社会的挑衅以及向国际律师进行一种贿赂。”
  <br/>
  <br/>
  耿和接受本台记者采访时分析说，北京政府或是迫于国际社会对中国侵犯人权行为谴责声浪过大，因此才要举办这场大会来挽回点面子。
  <br/>
  <br/>
  “中国近年来打压律师的力度不断升级，从高智晟开始到‘709’，并继续扩大。我们家属不遗余力地去寻找自己的丈夫，我想北京当局应该是脸面丧尽了，所以想通过开律师大会来为自己镀金。”
  <br/>
  <br/>
  另外，“709案”的律师家属李文足、王峭岭、原珊珊和许艳纷纷在社媒发声，呼吁外界关注中国人权律师被打压的现状。
  <br/>
  <br/>
  （记者：韩洁 责编：申铧 网编：郭度）
  <br/>

 </p>
