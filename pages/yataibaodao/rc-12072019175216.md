 <p>
  2020年一月台湾将举行总统大选。国民党总统参选人韩国瑜的妻子李佳芬，12月7日在她“代夫访美”的首场侨界造势活动中形容，中华民国2020年的总统选战，充满霸凌抹黑的负面攻击，更戏称自己甚至变成“最爱干政夫人”。她呼吁在美国的台湾侨胞，明年1月11日要踊跃回台投票。
  <br/>
  <br/>
  超过700名支持国民党的台湾在美侨胞高呼胜选与加油口号，热情欢迎李佳芬。华盛顿是李佳芬这趟访美行的首站。她站上讲台后深深一鞠躬，向支持侨胞们致意，也首次对这次选举开始以来，全家遭放大镜检视，讲述自己的心境。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  李佳芬：”我常跟我先生说，我嫁给你我什么没有，但我坐最多的就是‘云霄飞车’（过山车），我不知道他的人生为什么起伏这么大。2020年这场总统大选，大家都说这是有史以来似乎是‘最肮脏的选举’，我们个人被毁谤都事小，但是，这个国家能不能长期存在，才是决定了千秋大业。“
  <br/>
  <br/>
  过去两个月，从家人过去从事砂石业是否有靠着当时担任立委的韩国瑜关说，到韩家人在台北买预售屋引发炒房争议话题，韩国瑜选总统让相关家族过去的经历，一再被仔细审视。
  <br/>
  <br/>
  最新消息还指出，因为李佳芬对韩国瑜竞选办公室发言人叶元之的表现有疑虑，找来了刚请辞高雄市新闻局长的王浅秋担任总发言人，却也再次引发外界对李佳芬是否有”夫人干政“的讨论。
  <br/>
  <br/>
  李佳芬面对台下侨胞也自嘲，这场选战已经让她变成全台湾”最爱干政“的夫人般，但她不在乎外界怎么看，也不怕霸凌，因为侨胞给他们夫妇俩很多的温暖与关怀。
  <br/>
  <br/>
  陪同李佳芬访美的叶元之特别上台澄清，他要陪韩国瑜团队打完这场选战的决心不变。
  <br/>
  <br/>
  而韩国瑜则在事先录制好的影片中呼吁侨界，明年1月11日一定要返乡投票，捍卫中华民国生存与民主发展，并重申他的四大诉求。
  <br/>
  <br/>
  韩国瑜：“拜托大家要全力地保卫中华民国，第二，要同样的热爱中华文化，第三，我们要坚持民主自由，第四，莫忘世上苦人多。”
  <br/>
  <br/>
  5个星期后，台湾就将迎来又一次的民主选举，李佳芬代夫出征访美，主要聚焦与侨界互动，呼吁大家返乡投票。她这趟的访美行催票，前外交部长胡志强夫妇也随行陪同。
  <br/>
  <br/>
  曾担任台湾驻美代表的胡志强笑称，因为太太邵晓铃是“钢铁韩粉”，坚持要陪李佳芬来，他只好“妇唱夫随”，重返华盛顿。胡志强也打出有“芒果干”之称的“亡国感牌”，强调这次选举是中华民国的“生死关头”，呼吁大家一定要返乡投票。
  <br/>
  <br/>
  记者:郑崇生  责编：石山  网编：景铭
 </p>
