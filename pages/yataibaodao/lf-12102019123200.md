 <p>
  因中国华为公司财务长孟晚舟在加拿大遭逮捕，加拿大前外交官康明凯（Michael Kovrig）及商人斯帕弗（Michael Spavor）随即被中国关押，至今已满一年。加国发声明批评中国当局任意拘押两人，中国则强硬反击，表达强烈不满。
  <br/>
  <br/>
  10天前，孟晚舟发公开信说明自己过去一年来在加拿大的生活，正在等待法庭聆讯的她，可以自由外出，在家读书作画；但一年前就被中国关押的康明凯和斯帕弗却获得截然不同的待遇。

  加拿大外交部长商鹏飞的声明说：“康明凯和斯帕弗已经被中国任意拘留一年，一直未获得与律师和家人见面的机会。他们依然被列为加国政府最优先处理的事项。我们将继续不懈地努力，以确保他们立即获释。”
  <br/>
  <br/>
  中国驻加拿大使馆随後批评加拿大外交部：“对两人案件说三道四。康明凯丶斯帕弗因危害中国国家安全犯罪，依法逮捕，根本不存在所谓『任意』拘押。中方敦促加方停止发表不负责任的言论。” 中国外交部发言人华春莹则表示，两人为境外刺探国家秘密，已侦查终结，将移送检察机关审查起诉。中国司法部门严格依法办案，保障两人的合法权利。
  <br/>
  <br/>
  渥太华大学政治学教授史国良(Scott Simon)说，康明凯和斯帕弗难以被释放了。他并认为加拿大政府立场不够强硬，例如上个月新斯高沙省省长高调访问中国，上星期曼尼托巴省温尼伯市长还与中国驻加拿大新大使会面，均强调友好气氛，但事实上，当中国任意羁押加拿大公民，甚至践踏他们人权，这绝对是敌对方。
  <br/>
  <br/>
  “我们应该要暂时和中国保持距离，让两国关系稍微冻结一下，让中国知道我们是严肃的，我们希望被中国逮捕拘禁的加拿大公民能回来，不只是康明凯和斯帕弗，还有谢伦伯格(Robert Lloyd Schellenberg)丶玉山江(Huseyincan Celil)等人。”

  目前康明凯被关在北京南部郊区的一个监狱，斯帕弗被关在丹东监狱。加拿大《环球邮报》记者走访了康明凯和斯帕弗被关押的监狱，透过狱友们的描述，披露两人前6个月遭单独囚禁，如今暂时好一点，但饮食条件很差，只有稀薄的大豆肉汤丶玉米粥丶白菜萝卜丶馒头等。狱中没有丝毫隐私，24小时开着灯，睡觉也不能盖着头。两人唯一的休闲可能就是看书了，每个月可获得一次加拿大领事探访30分钟。
  <br/>
  <br/>
  曾任中国维权律师丶自己也蹲过看守所的祝圣武说，外界看到的仅是实体上恶劣的条件，但更糟糕的是精神上的虐待。中国当局会任意扣罪名拘捕无辜者，而他根本不知道自己接下来要面对什麽，不知道司法程序是如何进行的。
  <br/>
  <br/>
  “在中国谈程序权利就是一个笑话，共产党认为什麽程序权利啊？我不杀了你，你就应该感恩戴德。这就造成很大的压力，虽然我知道我死不了，但我不知道什麽能出去啊！你有可能明年出去丶可能明天，但也可能10年丶20年之後。”
  <br/>
  <br/>
  2014年，加拿大公民高凯文（Kevin Garratt）也曾因加中政治因素被关在丹东监狱内，他被关了19个月之後，於2016年才获得释放。他当时与被控谋杀丶腐败和偷窃者共处一个牢房，但无论是犯甚麽罪，都没有人知道自己的审判情况，他们都是活在不确定的黑暗中。高凯文说：“你承受着永远不知道会发生甚麽的压力。除了等待，还是等待，一切都是未知。”
