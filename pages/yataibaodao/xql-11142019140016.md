 <p>
  中国经济下滑趋势仍在持续之际，中国有两位体制内专家呼吁应该继续改革，进行结构性调整。不过有评论人士认为，中国当局恐怕未必会采纳。
  <br/>
  <br/>
  由于经济增长持续下跌，中国央行上个月再次下调中期贷款利率，中国国务院并提前公布明年地方债务额度，也降低了地方公共建设的最低资金要求。
 </p>
 <p>
  中国前总理朱熔基的儿子，本身也曾任中国最大投资银行中金公司CEO的朱云来，最近在财经杂志主办的一个研讨会上表示，用大量发行货币的方式，无法挽救中国的经济。这种短期的刺激手段，恐怕只会让中国的经济长期陷入困境，并错过了本身结构性改革的机会。
  <br/>
  <br/>
  与此同时，中国进出口银行前董事长李若谷也表示，中国应该落实中共十八届三中全会的决议，落实改革开放的承诺。他认为，这是美中贸易纠纷的关键问题。
 </p>
 <p>
  <br/>

 <p>
  不过，在美国的经济学者秦伟平认为，朱云来和李若谷的这些呼吁，恐怕不会对中国决策者形成影响。
  <br/>
  <br/>
  “朱云来和李若谷在中国政坛甚至是经济领域都没有什么影响力，都是边缘化的人物，不会对当局政策有太大影响。中国政府先关心的表面上看是经济，但实际上最重要的是政权的安全。他们在考虑如果经济出问题，如何能够维持社会的稳定。”
  <br/>
  <br/>
  美国南卡大学贝肯商学院教授谢田（Frank Xie）也认为，中国政府目前最大的考量是政权稳定，经济改革并非重点。他表示，中国经济下滑危机十分严重。
  <br/>
  <br/>
  “中国经济实际上已经是有点山穷水尽，随时会出问题。尤其在金融方面，坏账的问题、流动性的问题，银行出现挤兑等等，随时可能爆发更大危机。”
  <br/>
  <br/>
  他分析说，造成中国经济下滑的原因很多，过去多年来中国经济在政府干预下结构失调，以及美中贸易战都有影响。
  <br/>
  <br/>
  美中贸易谈判虽然传出第一阶段协议进展顺利的消息，但双方官方都维持谨慎态度。中国海关宣布从11月4日开始解除美国禽肉的进口限制，恢复进口美国禽肉。
 </p>

 <p>
  但中国进出口银行前董事长李若谷认为，美中之间的冲突，不能指望美国来帮助中国解决，只能靠中国继续改革开放，和落实以前作出的承诺。
  <br/>
  <br/>
  秦伟平表示，目前中国经济状况十分糟糕，北京最可能采取的应对措施，可能还是增发货币和加杠杆。
  <br/>
  <br/>
  “中国政府掌控了所有的资源，他们涉及到执政安全的考虑，虽然知道银行放水带来的问题，但没有其他好办法，只能是过一天算一天了。以后的问题以后再面对。”
  <br/>
  <br/>
  秦伟平表示，经济起伏周期是自然规律，但中国如果用错药方，将产生长期的不良影响。谢教授也认为，中国改革开放之后经历三十年的持续增长，并未经历过真正的大型经济危机，中国政府缺乏弹性的体制，可能将严重限制政府的经济和金融政策选择空间，这体现了政治体制僵化的深层问题。
  <br/>
  <br/>
