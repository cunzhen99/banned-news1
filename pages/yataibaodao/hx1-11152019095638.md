 <p>
  星期五在台北一场有关中国解放军军力的闭门研讨会上，有学者指出，解放军近二十年，投入大量人力物力研发新科技武器装备，战术战法也大幅提升，台湾应加强发展不对称战力。此外，美国国会美中经济暨安全审查委员会（USCC）年度报告，则建议美国国防部进行研究，遏止北京企图以武力统一台湾的计划。
  <br/>
  <br/>
  面对解放军日渐提升的军力和战备，台湾应如何应对？前国防部长杨念祖指出，解放军资源多，台湾国防应该发展短小精干军、不对称战力，阻止敌人用兵，让对方不能轻易渡海攻打台湾。由于解放军的扩张已经是全球关注的焦点，台湾15号一场重要的闭门研讨会中，就有来自全球各地的军事专家，参与探讨“解放军在境外用武”这个主题。
  <br/>
  <br/>
 </p>
 <p>
 </p>
 <p>
  台湾政治大学台湾安全研究中心研究员兼副主任胡瑞舟接受自由亚洲电台采访提到，解放军部队约200万现役军人，台湾部队约21万。军事预算上，中国大陆一年约1800多亿美金，台湾一年约101亿美金，军事力量严重失衡。因此台湾要有一支可靠军队，让对岸评估，台湾就算被打败，也令对方伤痕累累惨胜，如此就会降低北京动武的念头，也最好能从政治面解决问题避免战争。
 </p>
 <p>
 </p>

  这场由台湾的高等政策研究协会，和美国智库兰德公司主办的“中国人民解放军国际研讨会”，已经进入第30年。对于解放军的研究，有第一手的了解。来自美国的学者，就对解放军近20年军力提升，作了研究报告。
  <br/>
  <br/>
  <b>
   解放军
  </b>
  <b>
   20
  </b>
  <b>
   年来精进新科技武力和战术
  </b>
  <b>
  </b>
  <b>
   全球密切关注
  </b>
  <br/>
  <br/>
  胡瑞舟转述研讨会内容说：“例如说超音速的导弹，及发展出远程投射，甚至一枚核弹发射出去，可攻击十个目标。这都是比较特别的，包括导弹设计过程，它有不同速度，会增加你拦截的困难，这个都是非常突出的。解放军的反舰飞弹，五倍于音速，拦截非常困难。”
  <br/>
  <br/>
  研讨会中也讨论了解放军战术战法的配合，胡瑞舟说：“例如饱和攻击配合他们军民融合的科技发展，包括他们的无人机使用。有时看到他们的庆典展现无人机非常壮。我们要想到，如果用于军事用途，对我们造成非常大的威胁，拦截非常困难。”
 </p>

  <b>
   美国会建言美国防部研究阻止中共武力犯台
  </b>
  <br/>
  <br/>
  此外，台湾中央通讯社15日报导，美国国会美中经济暨安全审查委员会（USCC）年度报告指出，台湾对“一国两制”的担忧因香港抗议而扩大，报告也建议国防部进行研究，遏止北京企图以武力统一台湾的计划。报告在提及台湾时，建议国会指示美国国务卿提交报告，说明美国针对北京企图孤立台湾民主领袖所采取的因应计划或行动，加强支持台湾参与国际社会，并依据台湾旅行法扩大美台官员交流。
 </p>
 <p>
  报告也提到美国应制定战略，对与台湾维持外交关系的国家扩大发展与安全援助、多边合作，并建议国会指示美国国家情报总监办公室针对台海局势，从台湾、中国、日本与韩国高科技产品供应影响进行研究。委员会指出，这项研究应向国会各相关委员会提交简报，并针对中国企图在15年内武统台湾的计划，提出遏止的对策。
  <br/>
  <br/>
  <b>
   台国防部：国军有自我防卫能力
  </b>
  <b>
  </b>
  <b>
   确保国家及区域稳定
  </b>
  <br/>
  <br/>
  对此份报告，台湾国防部发言人史顺文接受自由亚洲电台采访回应说：“中共从来都没有放弃武力犯台，国军会持续掌握敌情的威胁，做好自我防卫的准备。我们有能力、有信心能够确保国家安全及区域稳定，让国家的自由民主可以永续地发展。”
  <br/>
  <br/>
