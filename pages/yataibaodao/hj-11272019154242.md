 <p>
  香港的“反送中”运动持续至今超过5个月，香港警察暴力滥权的问题一直是外界的关注焦点之一。香港保安局局长星期三就警方执法问题接受立法会议员质询时透露，已接获超过1千宗涉警投诉。
 </p>
 <p>
  <br/>
  李家超27日就警方针对连续数月的抗议行动，作出执法情况总结。
  <br/>
  <br/>
  <b>
   香港警察执法总结 发射约1万枚催泪弹
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>
  <img class="image-inline captioned" src="../../../../resolveuid/65a7db95-14d4-49a4-a780-30cff63d9762"/>
 </p>
 <p>
  他表示，自6月初至今，警方发射了约1万枚催泪弹、4800颗橡胶子弹以及约19发实弹，其中有3发打中示威者。共有超过2600人受伤送院，超过470人为警务人员。
  <br/>
  <br/>
  与此同时，警方一共拘捕超过5800人，其中923人已被立案控告。
  <br/>
  <br/>
  李家超还提到，截至本周一，香港警队投诉警察课接获1,261宗与“反送中”示威有关的投诉警察个案，包括指警务人员行为不当、不礼貌、殴打等，涉及1,647项指控，全部尚在调查。目前没有警员受到纪律处分。
  <br/>
  <br/>
  <b>
   民间组织：香港警察暴力执法情况严重
  </b>
 </p>
 <p>

  </b>

 </p>
 <p>
  香港“反送中”抗争运动始于民众要求政府撤回《逃犯条例》修订案，但由于当局无视民意以及警方的过度使用武力，导致警民之间的冲突越演越烈。香港警方暴力镇压示威者的举动也受到国际社会的广泛谴责。
  <br/>
  <br/>
  在信息不公开、政府公信力受损、警方不被追责的情况下，有民间团体自发搜集、整理、并发布《香港警察滥权实录资料库》，记录“反送中”运动期间的警察滥权案例。
  <br/>
  <br/>
  资料库收录的个案超过1400 个，涵盖了勾结与包庇黑势力类、妨碍司法公正类、性暴力类、恐吓、威胁、言语暴力类、插赃嫁祸示威者、攻击记者及妨碍新闻自由类等十三个范畴。
  <br/>
  <br/>
  据了解，这些个案大部分附有影片及新闻来源，并设有中文、英文和日文三个语言版本。
  <br/>
  <br/>
  （记者：韩洁 责编：申铧 网编：郭度）

 </p>
