 <p>
  2019年世界形势动荡不安，不少国家和地区也发生了专制政府和普通民众的激烈对抗。非政府机构美国的人权基金会邀请世界各地的人权捍卫者在纽约齐聚一堂，召开奥斯陆自由论坛，共同为自由发出呼吁。香港著名歌手何韵诗和前北大商学院教授鲍尔丁（Christopher Balding）出席并发表演说。
  <br/>
  <br/>
  何韵诗在所有十三位来自世界各地的人权卫士中，最后压轴出场。她在演说中，用提高的声调两次强调，让我们一起来斗争，为了自由、公正、人权与和平而斗争。她还号召在座的来自世界各地的代表与香港人站在一起，并肩战斗。
  <br/>
  <br/>
 </p>
 <p>
 </p>
 <p>
  她所强调的斗争，与上个月中国国家领导人习近平在中央党校演说时，五十多次重复的“斗争”二字，显然大异其趣。习近平所强调的是，中国共产党为了党和国家、为了民族而斗争。
  <br/>
  <br/>
  何韵诗说，香港“反送中”运动的核心就是习近平所领导的不尊重人权的政府与热爱自由、法治的香港人民之间的斗争。她总结了香港目前所处的困境，示威民众遭到的严酷打击。她同时用忧心的语调质问香港政府，你们该如何收拾目前的残局？难道是继续更深的压迫？她严肃地说，除非目前的问题得到真正的解决，香港的局势不会回到以往的常态了。
  <br/>
  <br/>
  何韵诗对在场不同国界的人士强调，香港的反送中运动已经是一场具有全球意义的斗争。它已经在印尼、加泰罗尼亚等地产生影响。她呼吁，每个人即使是采取微小的行动，也能带来巨大的改变。

 <p>
  本次论坛另一位演说嘉宾鲍尔丁的经历似乎印证何韵诗的这句话。鲍尔丁曾在北京大学汇丰商学院任教九年，但因为抵抗学校的政治限制而在去年被解雇。鲍尔丁目前在越南一所美国大学任教，而他最新的研究项目是中国的电信巨头华为公司和监控制度。
  <br/>
  <br/>
  他在演说中指出，目前中国是世界上对个人实施监控最严厉的国家。根据他收集到的数据，到2020年末，中国平均每两人将受到一台监控摄像头的监控。但这只是平均数，在其中一些地方，政府对个人的监控程度更为严厉。基于2019年的资料，目前世界上受监控最严厉的城市，前十名中有八个是来自中国，前三名分别是重庆、深圳和上海。
  <br/>
  <br/>
  他强调，中国已经成为了一个高科技“威权”，而华为公司正是这个威权政府的重要帮手。他系统研究了华为公司雇员的背景资料，发现其中有不少是在中国人民解放军的网络战争部门兼职，还有不少是在国家安全部门兼职，并对迫害人权的案例负有直接的责任。他揭露，华为公司从新疆的监控体系中拿到了不少合同。
  <br/>
  <br/>
  他进一步指出，华为公司目前正处于美中贸易战的前线位置，但在华为问题上产生的争议，并不仅仅是有关两国的竞争力的问题。问题的本质是技术的限制，以及象华为这样的公司将怎么影响我们的生活。

  参与本次论坛的人权卫士代表还包括在委内瑞拉街头抗争中，拉响小提琴鼓舞抗争者士气，因而被捕入狱的音乐家阿迪加(Wuilly Arteaga)，和朝鲜最高级别的外逃官员、前驻英国公使太永浩等等。
  <br/>
  <br/>
  在全场演说结束的欢呼声中，何韵诗演唱了她的新歌《极夜后》。歌词中说，“那一线光，即将散落，寸土尺方，风中依稀，听到了号角；这片地，终再看到，那艳阳高飞。”
