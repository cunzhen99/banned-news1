 <p>
  一向反对外国干涉内政的中国，日前疾言厉色地警告瑞典政府，不得派官员出席瑞典笔会（Svenska PEN）给遭中国关押的瑞典公民桂民海的颁奖典礼。在不干涉他国内政上，中国是不是双重标准？中国强势外交在瑞典当地又引发什么讨论或回响？
 </p>
 <div>
 </div>
 <div>
 </div>
 <div>
  中国驻瑞典大使桂从友2017年上任后，不时让中国在冷冽的北欧变成热门话题。最新焦点是瑞典笔会15日将2019年的图霍夫斯基奖（Tucholsky Prize）颁给仍遭中国监禁的瑞典公民桂民海，表彰他为言论自由发声，这让中国很生气。
  <br/>
  <br/>
  桂从友则再次展现强势外交的风格。他接受瑞典公共媒体瑞典电台（Radio Sweden）访问时用中文说，瑞典文化部长林德（Amanda Lind）出席给桂民海的颁奖典礼，将会有严重的反效果。
  <br/>
  <br/>
  瑞典电台的英文报导指出，桂从友还威胁瑞典政府，中国将不得不做出反制措施，两国正常交流与合作，将遭受严重阻碍。
 </div>

  瑞典首相勒文（Stefan Lofven）接受瑞典电视台访问时则不甘示弱地回击，瑞典政府“不吃中国威胁这一套”，不会向中国的恐吓低头，瑞典是有言论自由的国家。
  <br/>
  <br/>
  居住于瑞典的独立中文笔会前任秘书长张钰告诉自由亚洲电台，中国要求他国不得干预内政，北京却干预他国内政，这种双重标准，他一点都不意外。
  <br/>
  <br/>
  张钰：“这个大使是特别的具有侵略性，老向瑞典政府施加压力。实际上，瑞典民间或媒体还认为（瑞典）政府做得相当不够，都不太敢说这个事儿。当然，他们估计得从两个方面考虑，一是国家关系利益上的；另一方面，也担心说得厉害后，对桂民海本人也不利。就是中国大外宣搞得越过分的话，当地人就会反感。”
  <br/>
  <br/>
  <b>
   干预他国内政    万之：本质是意识形态斗争
  </b>
  <br/>
  <br/>
  中国大外宣的强势作为，不一定给中国的国际形象加分，而同样定居在瑞典的华裔异见作家万之则告诉自由亚洲电台，瑞典1999年原本要派他的瑞典籍妻子担任该国驻北京的文化参赞，却遭北京反对，就是因为担心他会跟他的妻子回到中国。万之说，他的例子证明，中国早就在干涉他国内政，而他认为中国大使馆反对瑞典官员参与颁奖给桂民海的活动，是有更深层考量。
  <br/>
  <br/>
  万之：“比如说，它（中国）把刘晓波关在监狱里，（引来）全世界要抗议，中国人会说这是干预我内政，然后，西方就给刘晓波发这个奖（诺贝尔和平奖），这个是一种意识形态的对抗，我觉得，这不是干涉内政的问题。”

 <div>
  桂从友受访时还声称，瑞典官员执意出席颁奖活动，会让瑞典在中国的形象受损。然而，去年中国游客不文明的行为，中国形象在瑞典民间早有变化。
  <br/>
  <br/>
  去年曾在网上疯传一个视频，一名中国游客用英文高喊“这是杀人行为、不人道的种族歧视”。这名曾姓男子去年半夜到访斯德哥尔摩，因酒店不愿提供提早入住，最后，他的父母遭瑞典警方抬出酒店，这些画面，当时在中国及瑞典两地都引发讨论。瑞典政治脱口秀节目借此讽刺中国游客，还遭中国大使馆抗议是“辱华行为”，甚至演变成两国之间你来我往的外交照会交涉。
  <br/>
  <br/>
  桂民海原是香港铜锣湾书店股东，因铜锣湾书店出版批评中国领导人习近平的书籍一案，他2015年在泰国被绑架回中国，至今仍被关押。
  <br/>
  <br/>
  中国政府指出，桂民海因涉嫌泄露国家机密罪而入狱。
  <br/>
  <br/>
  他的女儿安琪拉∙桂在英国剑桥大学攻读博士，安琪拉∙桂在网路上发起“释放桂民海”的活动。她在个人推特上说，自己因身体不适，未能前往瑞典领奖。
  <br/>
  <br/>
  截至发稿，中国驻瑞典大使馆未回应自由亚洲电台查询，而其英文官网上则于11月7日公布英文新闻稿指称，颁奖给桂民海“纯粹就是场闹剧”。
  <br/>
  <br/>
  据介绍，瑞典笔会1985年设立图霍夫斯基奖，每年向受迫害与威胁、或是对人权有贡献的作家或出版人员予以奖励，奖金是15万瑞典克朗，以纪念第一次世界大战时，从德国逃到瑞典的反战记者与作家图霍夫斯基（Kurt Tucholsky）。
  <br/>
  <br/>
  自由亚洲电台记者郑崇生华盛顿报导 责编：申铧 网编：郭度
