 <p>
  愣在焕然一新的废墟前
  <br/>
  唯色
  <br/>
  <br/>
  愣在焕然一新的废墟前，
  <br/>
  不，废墟已化为乌有，
  <br/>
  替代物是一座仿若寺院的建筑，
  <br/>
  如同海市蜃楼的幻现，
  <br/>
  更似将要派上用场的赝品。
  <br/>
  <br/>
  似乎只有他们没变，
  <br/>
  像是长不大或停止了生长，
  <br/>
  依然是多年前的跑来跑去，
  <br/>
  依然是多年前的叽叽喳喳，
  <br/>
  依然是多年前的好奇心按捺不住，
  <br/>
  扑过来要我拍照，要看镜头里的小伙伴。
  <br/>
  <br/>
  我指着那边说：废墟呢？
  <br/>
  你们知道那废墟以前是寺院吗？
  <br/>
  知道那寺院怎么成了废墟吗？
  <br/>
  知道那废墟又怎么成了这貌似的寺院吗？
  <br/>
  我是如此急切，甚至连地上跑过去的狗，
  <br/>
  半空中飞过去的鸟，也想一把抓住问几句。
  <br/>
  <br/>
  他们充耳不闻，竭力从我手中挣脱，
  <br/>
  兀自玩耍，好像这里原本就是游乐场。
  <br/>
  有几个张开双臂，绕着镜面反光的太阳灶不停转圈，
  <br/>
  “我想拿这些金属片做一只翅膀……”[1]
  <br/>
  回忆与眼前重叠，受挫的内心渐渐平复，
  <br/>
  是的，喜德林[2]的孩子们安慰了我。
  <br/>
  <br/>
  这些现世中的生命，其实我比他们更知道
  <br/>
  眼前崭新的建筑有着怎样的过去，
  <br/>
  须从赞普[3]松赞干布[4]向佛的时代说起，
  <br/>
  之后的故事纠结本土风云，可以省略不提，
  <br/>
  直至世事反转沦为废墟，又被推倒重筑，
  <br/>
  另有操纵者很深的用意……
  <br/>
  <br/>
  但在他们的眼中，我可能更像一个好奇的
  <br/>
  观光客，其中几个，我多年前就见过，
  <br/>
  那时候多么顽皮，现在长成了少年，
  <br/>
  稍微收敛了贪玩的习性，
  <br/>
  却对生于此地的变故，
  <br/>
  似乎并不知情。
  <br/>
  <br/>
  不过我知道的，比他们多得多又如何？
  <br/>
  我知道的，比许多人多得多又如何？
  <br/>
  我知道的再多，或许并不应该写下来，
  <br/>
  当我写下喜德林的前世今生，
  <br/>
  它就中了真理部的魔咒。
  <br/>
  2018-4-14，拉萨
  <br/>
  <br/>
  注释：
  <br/>
  [1] 2013年夏天，艺术家艾未未在推特上看到我拍摄的喜德林废墟，受废墟前多个聚热烧水的太阳灶启发，让我协助用新的太阳灶更换了旧的太阳灶，并用这些旧的太阳灶做了名为“断翅”的作品。后来他在一个访谈中说：“……我想拿这些金属片做一只翅膀，像羽毛一样……让我遗憾的是，那座废墟被拆掉了……一段历史就这么没了。”
  <br/>
  [2] 喜德林：བཞི་སྡེ་གྲྭ་ཚང་།（Shide Tratsang），又称喜德扎仓，藏传佛教格鲁派在拉萨重要的经学院。毁于1959年的中共军事镇压、1966年文化大革命及之后的岁月。2016年废墟被拆除，重盖仿寺院建筑式样的房子，据说可能设成“爱国主义”主题的展览馆。
  <br/>
  [3] 赞普：བཙན་པོ།（Tsanpo）,君王，指吐蕃（图博）帝国君王。
  <br/>
  [4] 松赞干布：སྲོང་བཙན་སྒམ་པོ། （Songtsen Gampo），公元七世纪人物，吐蕃（图博）帝国第三十三代赞普，实际上是立国之君，以缔造帝国、建造布达拉宫并将佛教引入图伯特、修建大昭寺和小昭寺而著名于世，被藏人认为是观世音菩萨的化身。
