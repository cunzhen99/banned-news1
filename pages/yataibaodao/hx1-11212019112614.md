 <p>
  香港警察连日来与在大学校园里的抗议者发生暴力对峙，导致不少大学宣布学期提前结束。目前出现了在港师生赴台湾紧急避难访问潮。光是台湾大学提供的免费赴台1个月访问专案，就有20个国家近600名学生提出申请。
  <br/>
  <br/>
  在香港多所大学形同“战场”的危急时刻，台湾最高学府台湾大学上周四（14日）率先伸出援手，提供在港大学生免学杂费访问一个月的专案，交通大学、清华大学、政治大学、成功大学、中山大学等校，也纷纷宣布跟进。“在港学生赴台访问潮”明显涌现。

  <br/>
  <b>
  </b>

  <b>
   台大专案名额无上限
  </b>
  <b>
  </b>
  <b>
   香港本地生申请量日增
  </b>
  <b>
   60%
  </b>
  <br/>
  <br/>
  台大副校长周家蓓21日接受访问就说：“这个数字每天都在提升，今天早上的数字，总共申请562位学生，其中本国（台湾籍）的学生大概有289位，在香港本地学生大概有168位，其他大概还有21个国家加起来也有100位的学生。昨天香港本地学生105位，今天跳到168位，几乎增加了60%的香港本地学生，所以确实有独特的需求。”
 </p>

  周家蓓强调：“这个是一个专门、特殊的管道，以往从来没有在学期当中有同学转进来，而且是这么大量。”
  <br/>
  <br/>
  周家蓓说，香港有11所大学跟台大是姐妹校，但此专案对象，是只要在香港各大学念书，具学生身分，都会接纳，不在乎成绩表现，不收取任何学杂费，名额无上限，25日申请截止。学校会先腾出新进教师的宿舍给自香港到台湾的港生或外籍生，必须缴住宿费。访问时间是12月初至明年1月中学期结束，不能获得学分。
  <br/>
  <br/>
  周家蓓强调：“台湾大学这么样主动、很善意地做这个呼吁，是希望在香港念书的各国年轻人能够不要荒废这一个月。虽然他们的学校也可能有网路课程，但是他们基本上就没有办法到校园里面去了。所以台湾大学很欢迎他们来台湾大学接受一个月台大正规课程教育。但在这个月过去之后，我们还是觉得他们应该思考回到他们自己原来的学校，但是如果学生真的有他们思考后的决定，我们也表示欢迎。”

 <p>
  申请台大专案访问生562人当中，最多的是来自香港中文大学，另外还有香港城市大学、香港理工大学、香港大学、以及香港科技大学等。
  <br/>
  <br/>
  位于新竹的台湾理工科名校清华大学，也对在香港的学生同样提供免费访问的专案。
  <br/>
  <br/>
  清华大学全球长严大任21日接受自由亚洲电台访问，也以“如火如荼”形容咨询人士的踊跃：“如火如荼进行中。香港局势变化很快，上礼拜五，设立清华香港的热线处理这件事，已接到150多通询问电话，约有40多人表示要到台湾清大访问。”
  <br/>
  <br/>
  严大任透露，校方也接到在香港教书的老师透过友人询问，间接表态想申请到清华大学任教。

  严大任提到，在香港的学生，如果想到台湾清大就读有三种入学管道，包括没有学分的访问、可抵免学生的交换生，以及转换学籍的转学生。他说，目前访问专案属于“救急”。
  <br/>
  <br/>
  新竹交通大学也表示，已接获约百通谘询电话，有香港中文大学的台湾籍学生申请旁听，也已经入住宿舍，等确认修读课程后办理加签。另有香港的教授询问短期交流细节。

  <br/>
  <br/>
  台南的成功大学也紧急出台“在港学生来台学习衔接措施”。成大21号表示，已接获近50次询问，也有香港教师询问可否转换教职到成大。成大表示，除了现有提供旁听、随班附读等学习衔接措施外，也提供20个独立工作室。建筑系则备有50个短期研究室，规划设计学院和工学院的实验仪器设备，现已逐步开放申请，以方便学生作业需求，成大还安排了200多个床位。
