 <p>
  中国大陆日前推出“对台26条措施”，提供所谓“同等国民”待遇，以此吸引台湾人才的意图明显。11月5日，美国与台湾政府联合创建“人才循环大联盟计划”，提升打造台湾的软实力。
  <br/>
  <br/>
  美国在台协会（AIT）、台湾人才发展协会及经济部工业局，5日在台北共同举办首届“人才循环大联盟高峰会”。
  <br/>
  <br/>
 </p>
 <p>
 </p>
 <p>
  台湾的副总统陈建仁致词指出，根据瑞士洛桑管理学院最新公布的《2018IMD世界人才报告》(IMD World Talent Report 2018)，台湾在亚洲人才发展当中排名第4，领先日本与韩国，一直扮演专业技术提供者的角色，并领导亚洲技术与经济的发展。
 </p>
 <p>

  <b>
   陈建仁：台湾面临如何留才揽才挑战
  </b>
  <br/>
  <br/>
  陈建仁提到，台湾面临要如何育才、留才与揽才的挑战，台美共同发展“人才循环大联盟计划”，不仅可健全台湾人才发展环境、促进双方人才交流，并建立跨国人才库。双方将提供给对方的专业人士、交流生、高阶职员互相进修实习机会，以打造台湾成为亚洲软实力典范为目标。
  <br/>
  <br/>
  <b>
   AIT
  </b>
  <b>
   盼人才计划重建台湾奇迹
  </b>
  <br/>
  <br/>
  美国在台协会处长郦英杰提出，美台合作创立的“人才循环大联盟计划”有4大目标，包括以数位方式，重建台湾过去的奇迹与成功，解决人才外流问题，持续培养人才资源及扩大台湾在全球的能见度。
 </p>
 <p>
 </p>
 <p>
  <div class="image-inline captioned" style="width:1500px;">

  郦英杰说，台湾很多企业创办人都是从美国回台湾就业，这些菁英在国外取得宝贵经验后，回台开创一条光明的道路，“人才循环大联盟计划”希望能复制这样的经验。至于解决人才外流问题，就是在台湾、美国及其他经济体之间能有很好的人才互动与流通。
  <br/>
  <br/>
  台积电创办人张忠谋会中对上千名参与者演讲，分享自己在美国求学、就业经验。
  <br/>
  <br/>
  张忠谋提到，他自己就是经过在美国和中国的循环跟训练之下的最大受益者。
 </p>

  <b>
   张忠谋：从美国带回
  </b>
  <b>
   “
  </b>
  <b>
   世界观
  </b>
  <b>
   ”
  </b>
  <br/>
  <br/>
  张忠谋说，他在美国36年，固然大学教育在美国，但他觉得有学到比技术更重要的东西并带回台湾，创办台积电。
  <br/>
  <br/>
  张忠谋说：“我还有学到比大学教育更重要的是世界观，我学到的是谦卑。台湾实在是一个相当小的地方，我常看到一个人考取台湾大学，那个好像就是他一生的颠峰，为什么我这样想？因为我看很多五、六十岁的人，还是总在叫‘我在台大的时候’。我在美国可以说，从来没有看过一个五、六十岁的人说，我在哈佛、在耶鲁的时候。”
 </p>
 <p>
 </p>
 <p>

  <br/>
  张忠谋说，在美国就知道有很多好的大学，所以并不觉得上过哈佛有什么。自己是MIT（美国麻省理工学院）毕业，但毕业之后就不提MIT了，知道世界很大、人才很多，就学会谦卑，就不会想提在台大如何如何。
  <br/>
  <br/>
  张忠谋说：“我从美国带回来最重要的是世界观，创办台积电，我的主要的资产、智慧财产是世界观，而不是专门的知识 。”
  <br/>
  <br/>
  张忠谋提到，他在台积电企业内推动人才循环，目前台积电有20多名副总以上的高阶主管至少一半都是在台湾念完大学，转赴美国念研究所、就业后再回台，都是从基层做起，一步步升到高阶主管职位。他也提到，台积电最早的3位总经理都是美国人，不是华人，他们离开台积电后都回到美国创业、工作、投资，有很好的发展。
