 <p>
  香港反送中运动届满半年之际，12月8日，世界人权日当天，有80万香港人走上街头，持续表达“五大诉求、缺一不可”的坚持。这半年来，香港人的坚持不懈达成哪些目标？香港社会又起了什么样的变化呢？
  <br/>
  <br/>
  在世界人权日当天，又有80万香港人一起走上街头高呼“五大诉求、缺一不可”的口号。香港民间人权阵线发起了这次合法游行集会。游行持续到当天夜里，游行民众全把手机照明灯打开，形成一片闪烁的灯海。
 </p>
 <p>
  而回顾香港人这半年来的坚持，光明并不是一下子就到来。
  <br/>
  <br/>
  “在6月16日，更是有200万的香港人走上街头，这一次之后，港府已经宣布会‘暂缓’修例的工作。” 这是自由亚洲电台记者吕熙在12月8号当天进行视频直播, 回顾这半年来街头运动情况的进展。

 <p>
  6月中旬，香港700多万人口，有近200万人上街后，香港特首林郑月娥才在6月18日首次亲自出面，公开致歉，但她不下台、不宣布撤回修法，仅说已经“暂缓”。香港民众对政府的不信任度，也随着警方暴力及选择性执法的争议升高，尤其是8月31日，港警在地铁太子站无差别式以警棍殴打、拘捕民众后，香港警民冲突从街头到商场，从地铁站到私人住宅，暴力冲突不断。
  <br/>
  <br/>
  示威民众喊着要解散警队，5大诉求中，民众还要求“成立独立调查委员会”，但林郑月娥9月4日则是提出“4项行动”回应“5大诉求”，总算宣布正式撤回修订《逃犯条例》草案，不过，对于成立独立调查委员会她在9月底的首场、也是至今唯一一场的社区对话中明确说“不”。
  <br/>
  <br/>
  到了10月，港府还宣布启动《紧急条例》、制定《禁蒙面法》，导致街头暴力升级，警方数次对民众开枪；11月，更引发警方在香港中文大学与香港理工大学的包围与对峙，让香港校园成为火光与催泪烟弥漫的战场般。

 <p>
  香港警方12月9日公布数据显示，这半年来，已拘捕近6000人、超过1/3是学生，比率在开学后明显增长，其中，至少有340人年龄为16岁以下。另外，警方这半年发射1万6千枚催泪弹，各类弹种共4万枚。
  <br/>
  <br/>
  半年来，香港的街头运动，让民主派在11月24日的区议会选举中大获全胜，452个民选议席中，拿下超过8成席次，亲北京的建制派输得空前惨烈。
  <br/>
  <br/>
  与此同时，美国国会参众两院以罕见的速度和接近全票先后通过《香港人权及民主法案》，特朗普总统在美国传统的感恩节前夜将法案签署生效，使得香港的抗议者们深受鼓舞。
  <br/>
  <br/>
  香港人六个月的坚持，受到全世界的关注。他们中有多人蹊跷地死亡，很多人被警察殴打受伤、被抓，又有很多人和不理解的家人决裂，甚至远走他乡……面对强硬的中国政府，他们要想真正实现五大诉求，还有很长的路要走。
