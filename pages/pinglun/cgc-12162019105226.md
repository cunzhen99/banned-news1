 <p>
  2019年12月1日，中共刚刚在广州的广州大道北与禺东西路交界处发生的地面坍塌中不经过救援就直接将掉在里面的石宜、石科伟父子等3人填埋在下面。而几天后的12月12日晚上10点，在福建厦门地铁附近又发生了地面塌陷。更具讽刺意义的是，该地铁工程两周前刚刚通过了有关部门的质量验收。
  <br/>
  <br/>
  广州的地面坍塌发生在大约上午9点多。 石科伟的妻子喻妮接到消息后，11点多就赶到了现场，可是中共有关部门一直让他等着，公安们根本不让她靠近现场，也没采取救援措施。然而到了12点多以后，工程车就开始一味的回填，根本不再考虑救人了。
 </p>
 <p>
  喻妮要求停下水泥灌浆车先把人救上来，活要见人，死要见尸，遭到拒绝。她试图冲到现场，但被警察暴力阻拦，并且强行将她弄到酒店软禁起来。现场的围观者，面对政府活埋人的情景，纷纷谴责中共政府毫无良心，灭绝人性。
  <br/>
  <br/>
  从9点多事故发生，到12点多工程车到达现场，开始回填，中间不过三个小时。从政府调集工程车，到施工车辆到达现场肯定需要一定的时间，这说明中共政府早就做出了放弃救援，直接填埋的决定。根本就是视人民的生命如草芥。
 </p>

 <p>
  两天后，喻妮抱着刚满月的孩子，与她的婆母也就是石宜的妻子，来到已经修好路面的现场当众哭诉，但是已经于事无补。当时众多路人围观，悲愤伤痛而无奈的场面，令人不忍听闻！
  <br/>
  <br/>
  记得好像是一九八九年的夏天，电视新闻上播放日本有个小女孩掉进了直径很小的机井里，由于井口太小无法救援。日本政府就在原来井的旁边又迅速地钻了一眼直径较大，同样深度的井，救援人员下到井底横着将两口井打通，把孩子救上来。当时我的一位邻居大叔说：“这事要是发生在中国，这个孩子根本不可能被救上来”。我问：“那怎么会？”他说：“谁管？”我听了他的话不以为然，觉得平时政府说得那么好，怎么会见死不救呢？现在回想，早在那个时候，我这位邻居大叔就有如此深刻的见解，真是不简单。
  <br/>
  <br/>
  据说，掉入厦门地铁塌陷深坑中的两辆车上的人员全都自行逃生捡回了自己的性命，希望这不是中共有意放出的不实之词。我期待车上人员安全逃生的消息能得到进一步证实，
  <br/>
  <br/>
  在共产党自己监督自己的专制中国发生的所有灾难，不查都是天灾，一查几乎全是人祸。共产党绑架中国七十年，使得全社会道德沦丧，人心不古。不幸的是，在所有的人祸中，倒霉的永远是普通民众，或称“老百姓”。有网友看到喻妮娘俩哭诉的视频后，留言说：“他们有没有支持香港的时代革命？之前说不定经常唱社会主义好呢？” 这些问题问的很残酷而不合时宜。可是毋庸置疑的是，其中提及的事务深层相关联系确实十分重要。
  <br/>
  <br/>
  在中共统治下的七十年，无数事实证明，许许多多重大不幸灾难事故发生，不查都说是“天灾”，一查尽是人祸。
  <br/>
  <br/>
  人生不易，无论是谁，未雨绸缪都是十分必要的。平时是否勤于思考，关注社会公正，在灾难还没有轮到自己头上之前，是否具备对所处社会深刻准确的认识，对任何人来说，都是生命攸关的。切不可蒙蒙昧昧度一生，不知哪一刻，就有可能被无处不在的人祸夺去性命。
 </p>
