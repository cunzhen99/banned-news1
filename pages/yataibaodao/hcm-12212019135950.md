 <p>
  台湾总统选举进入最激情的一个周末，焦点则聚集在国民党总统候选人韩国瑜的身上。台湾民间组织Wecare，在高雄发起罢免韩国瑜高雄市长的大游行，号称有50万人上街；而另一端，韩国瑜竞选总部也发起大游行与之对抗，组织者号称有35万参加，双方在高雄街头拚场。台湾警方出动三千名警察维护治安，所幸并没有发生冲突意外。
  <br/>
  <br/>
  支持韩国瑜的游行队伍下午一点从凹仔底公园出发，沿着北高雄的主干道博爱二路等路段行进，最后抵达微笑公园造势场合。游行人士一路大喊：“韩国瑜当选、韩国瑜当选”。不少参与民众配合主办单位宣传的服饰颜色，以蓝白红的国旗色系服装为主，游行的队伍边走边摇国旗，形成一片国旗海，场面相当壮观

  <br/>

   Wecare罢免韩国瑜游行  50万人高喊光复高雄

  而另一头，在南高雄登场的则是“Wecare台湾大游行”，游行队伍从高雄文化中心登场，一路沿着五福路一路行走，游行队伍绵延数公里，群众口里喊着“光复高雄、保卫台湾”、“韩国瑜下台”等口号，游行民众举着一面长210公尺、宽12公尺的巨幅旗帜，上面写着“光复高雄”。
  <br/>
  <br/>
  Wecare高雄发起人尹立在下午4点45分，上台宣读1221wecare台湾大游行宣言，更宣布这场游行参加人数已超过50万人。

  <strong>
   35万份“罢韩”连署书上街 26日进入罢免程序
  </strong>
  <br/>
  <br/>
  游行队伍有一辆相当醒目的卡车上贴有“封条”，上面载着30万份“罢韩”公众连署书。尹立表示，12月26日，会向中选会递送第一阶段的高雄人连署罢免提议书3万份，正式进入罢免程序。未来几个月，将面对韩国瑜市府的选委会，还要进入二阶连署，即要有高雄市民人口10分之1联署，经审核后才能办理罢免投票，更要有至少4分之1投票率，且赞成罢免人数超过反对票，高雄才能有重新选择市长的机会。
  <br/>
  <br/>
  尹立在游行前表示，两边游行的诉求都一样，都希望韩先生赶快离开高雄。在游行队伍，一位模仿韩国瑜造型的高雄市民说出她的心声，“他今天敢抛弃高雄市民，我们就有勇气去讽刺他。”
