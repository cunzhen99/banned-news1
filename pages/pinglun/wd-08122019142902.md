 <p>
  各位听众你们好，我是王丹。中美贸易战现在愈演愈烈，中美双方都祭出了各种手段来报复对方。8月5日开始，中国正式停止采购美国农产品。我们知道，中国是全球最大的大豆买家，而美国的大豆主要出口到中国。中国停止採购大豆，当然会伤害到美国的农民，中共希望的是因此而撼动川普的政治基础。他们寄希望于美国农民因为自己的利益受损，而通过选票让川普下台。
  <br/>
  <br/>
  不过大家也要想到，这样的贸易制裁，其实是一把双刃剑，它伤害的不仅是美国的农民，其实受害者也包括中国自己的老百姓。我们知道，中国每年人均消费59公斤猪肉，相当于每年必须饲养7-8亿只猪供作肉食消费，还有鸡鸭牛羊等，这些部分都要靠大豆或残渣作饲料，此外，大豆还主要用于豆腐，豆浆，榨油等食用消耗，所以，中国人的日常生活和消费对大豆的需求是非常高的。大豆需求是否能够满足，也关系到中国老百姓的切身利益。未来大豆要減少依靠进口的模式，价格必然也要上涨，相关产品的价格也会水涨船高，最终承担代价的，还是中国的消费者。
  <br/>
  <br/>
  或许有人会说，中美贸易战，中国的消费者付出一定代价是应当的。这么说的人，不妨回顾一下中国官方过去的说法。2018年的时候，中美贸易战曾经一度有缓和的迹象，当时中方答应大量购买美国的农产品，当然就包括大豆。大家还记得那个时候中国官方是怎么说的吗？2018年5月24日的《人民日报》曾经发表署名文章，题目为《扩大美国农产品进口符合中方利益》，官方给出的说法是：中国增加美国农产品进口，意味着国内消费者也有了更多选择，消费更趋多元化；中国农业结构调整有了更多动力，农业发展品质将进一步得到提升。因此，《人民日报》文章的结论是：在我国全面深化改革开放、扩大内需的大背景下，满足人民日益增长的美好生活需要、实现经济高品质发展，呼唤更好产品。中方增加自美国进口农产品，是根据双方市场的供求情况，政府引导，基于市场经济原则，由市场主体自主决策的结果。换句话说，当时的中国官方宣称进口美国农产品，不是出于贸易战的妥协的需求，而是为了“满足人民日益增长的美好生活需要”。
  <br/>
  <br/>
  那么问题就来了：难道现在中国人民的这种“对于美好生活的需要”就消失了吗？当然不可能。可是中共现在就完全不顾及这种需要了。如果当初扩大进口美国农产品不是为了贸易战，而是为了中国老百姓的利益，那么现在为何就不顾老百姓的利益了呢？这里显然存在前后矛盾的地方。换句话说，是不是进口美国农产品的考量基础，根本就不是为了中国老百姓的利益，而只是相互攻防的策略性决定。在中共眼裡，中国人的“对美好生活的需要”，不过就是进攻对手的武器而已，是随时可以放弃的。
  <br/>
  <br/>
  当然，美国的川普政府在中美贸易战中採取的制裁措施，也导致了美国的农民的利益的损失。但是我们必须指出的是，美国农民是完全有权利表达不满、甚至通过投票反对川普这样的做法的，我们前面说过，这也正是中共採取这种做法的深远目的。换句话说，美国农民会不会为了中美贸易战付出代价，在某种程度上他们自己是可以进行选择的。相反，中国的老百姓承担贸易战的代价，却完全无法为了维护自己的利益做任何事情。一场贸易战，双方的老百姓都会付出代价，但是对比之下，中国的老百姓更像是中共的人质。以上是自由亚洲电台特约评论员王丹为您做的评论。
