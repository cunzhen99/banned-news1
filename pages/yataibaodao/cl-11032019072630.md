 <p>
  “ 112 ” 是国际紧急求救号码，17国家超过46个城市的港人在11月2日全球串联集会，为处于紧急状态的香港向国际社会求援。旅法港人还针对香港警察滥用水砲车，发起了 “ 向法国製造水砲车说不 ” 的连署。
  <br/>
  <br/>
  呼应香港 “ 112求援国际，坚守自治 ” 活动在11月2日与全球17个国家46个城市的串连，旅法港人撑香港团体也在当天巴黎夏特雷广场(Place du Chatelet)集会，谴责香港警察暴力与滥用水砲车，并发起连署活动，呼吁水砲车出产国法国，禁止向香港出售武器，尤其是水砲车。
 </p>

 <p>
  “这是一个反对香港警察使用水砲车攻击一般市民的请愿活动。其在法国解放报也有一个报导指出，这些水砲车是由法国提供的。不管是他的公司进口，或是用甚麽军事合作的名义，让这些所谓的镇暴工具进到香港，我觉得都是对民主的一种破坏。所以，今天我们在这裡，是希望对这件事情做一个抗议。”
  <br/>
  <br/>
  香港警方向法国购买的3辆水炮车，每个都配置15管水砲，而且每分钟可喷射1200公升的水柱，还可同时射出溷合的催泪剂、染色剂和水等，增加驱逐示威者效率；同时，示威者若被喷上染色剂，警方更容易辨认出示威者的身分，可有效驱离或逮捕。
 </p>

  <br/>
  但是，法国民众却无法以这个“法国製造”而骄傲，夏洛特女士就说:“我是法国人，对于法国输出水炮车到香港，引以为耻。阿拉伯之春的时候，法国建议突尼西亚政府购买水炮车以镇压突尼西亚示威民众，现在又要用这个设备攻击香港人，太不可思议了，我完全不能忍受。香港现今的情况让我难过，所以我来这裡声援香港民众，一起对抗暴政。”
  <br/>
  <br/>
  香港警方以水砲车攻击市民造成伤害的意外层出不穷，例如日前对清真寺与路上行人发射蓝色水砲的事件。香港绿色和平组织根据化验结果指出，警方使用的蓝色水柱含有日内瓦公约禁止使用的催泪剂等有毒化合物，碰触身体后会引起疼痛、红肿、角膜刮伤和喉咙肿胀等症状，严重时甚至可能导致肺炎。

  法国民众马蒂亚斯感同身受的表示:“在法国我们也遭受到同样的镇压，警方也使用武力对付黄背心示威抗议者，法国不只卖水砲车到香港，也卖武器到其他发生战争的国家，对于这样一个标榜人权的国家，实在是绝对的耻辱。”
  <br/>
  <br/>
  由于香港警方镇压行动所用的水砲车都是向法国购买，法国的防暴警察及车厂、水炮製造商都曾到香港交流运用水炮车的战术及经验。所以，在巴黎的集会现场，港人也进行了 “ 向法国製造的水砲车说不”连署，希望让法国人了解水砲车可能对人民造成严重伤害，必需约束制止香港警方的滥用，并要求法国在这个镇压武器的使用专业训练中，增加注意事项与规范。
