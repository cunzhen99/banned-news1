 <p>
  近一周以来，国际各主要数字货币价格普遍下跌。与此同时，中国央行等政府部门也在国内大力打压民间数字货币交易。那么，中国政府的做法是否是国际数字货币市场波动的主要原因呢？
  <br/>
  <br/>
  数字货币价格波动最显著的指标，是比特币的价格。比特币的价格周一开盘是每币7200美元左右。周日晚，最低跌到了6600美元。这是半年以来，比特币的最低价。而一周之前，这个价格还在8400美元，跌幅约20%。与此同时，其他主要的数字货币以太币、瑞波币等也出现了类似幅度的价格下滑。
  <br/>
  <br/>
  美国《华尔街日报》认为，这波行情下调的主要原因是中国政府从上周开始，大力整治国内数字货币市场。由于目前全球数字货币采掘量有70%是在中国，业内分析人士指出，中国的相关政策是影响数字货币市场最主要的因素。
  <br/>
  <br/>
  上周五，中国人民银行上海总部发表声明指出，数字货币的炒作花样翻新、投机盛行，并有融资主体以此为名，向投资者筹集资金或虚拟货币，严重扰乱了经济金融秩序。央行将联合上海市相关部门对本地区的虚拟货币活动展开专项整治。
  <br/>
  <br/>
  无独有偶，就在同一天，深圳市政府也发出通知，提出要防范所谓的虚拟货币“非法活动”。重庆、杭州、内蒙古等地从上周以来，也相继开展了类似的整顿行动。
  <br/>
  <br/>
  严禁民间数字货币交易，是中国政府自2017年以来的一贯政策。据《华尔街日报》报道，央行上海总部和上海市政府自2017年以来，已经强行关闭了13个数字货币初创平台和10家数字货币交易公司。
  <br/>
  <br/>
  但同时，中国央行却在加紧研发自己的数字货币。两周前就有消息传出，央行的数字货币将在三个月内发行，届时将有包括阿里巴巴、腾讯和工商银行等7家机构作为联合发行机构。
 </p>

 <p>
  10月底，中国国家领导人习近平曾在中央政治局集体学习时强调，要加强对数字货币底层技术区块链的研发。自那以来，广州、天津、昆明等地的市政府相继颁布了对区块链的扶持政策，并宣布将加大投资。
  <br/>
  <br/>
  一方面是政府加大对区块链和数字货币的研发和投资，另一方面却不许普通民众自由从事数字货币的交易。这种做法与其他国家差别较大。
  <br/>
  <br/>
  在中国的邻国泰国，《曼谷邮报》最近披露，为了在数字货币竞争中迎头赶上，泰国的证券交易委员会周一表示，正打算修改法律，促进商业企业研发数字货币，同时保护投资者的权益。
  <br/>
  <br/>
  而美国媒体《商业内幕》上周引述美联储主席杰罗姆·鲍威尔（Jerome Powell）表示，美联储正在考虑研发自己的数字货币。与此同时，美国政府并未限制现存数字货币市场的交易，虽然也曾对数字货币风险提出警告。
