 <p>
  对《全球马格尼茨基人权问责法》很多朋友已经不陌生了，但是一些人还未必知道其由来。
  <br/>
  <br/>
  今年美国的“兰托斯人权奖”颁给了比尔.布若德（Bill Browder ），他是俄罗斯人权问责法雏形的推动者。9月26日，在参加兰托斯人权基金会颁奖晚宴时，我有机会与比尔见面交流。在宴会上，他讲述了案件的来龙去脉和如何推动俄罗斯人权问责法的出台。27日上午，我参加了他的颁奖典礼，美国国会史密斯、麦高文等议员到场祝贺。
  <br/>
  <br/>
  当年美国人比尔.布若德在俄罗斯经商，发现很多上市公司腐败严重，与官员互相勾结，欺骗入市的散户，他们拿走所有的利益，使得散户们血本无归。情况很大程度上与现代中国的股市状况相似。于是，他开始展开调查和揭露。俄罗斯当局为了掩盖犯罪真相，开始对比尔进行迫害。俄罗斯律师马格尼茨基挺身而出，介入了比尔的案子，做他的辩护律师。
  <br/>
  <br/>
  于是，俄罗斯当局又开始伺机对马格尼茨基律师进行迫害，把他关进监狱，直到酷刑折磨迫害致死在狱中。之前，当比尔逃离俄罗斯时，曾劝马格尼茨基一起离开，可是他拒绝了。不幸的事很快发生， 2010年他被俄罗斯当局在监狱里酷刑致死。
  <br/>
  <br/>
  比尔回到美国便联系有正义感的国会议员，推动美国对在马格尼茨基被死亡一案中负有责任的官吏进行惩罚，获得了议员们的支持。法案在最后时刻，又增加了几十个字，使得这个法案变成了针对全俄罗斯迫害人权的官员，并获得通过。
 </p>

 <p>
  2012年，作为报复，普京针对美国国会通过该法案下令不准美国人领养俄罗斯的孤儿。普京的反应让支持该法案的议员们觉得很有成就感。
  <br/>
  <br/>
  当时我刚到美国，对此并不知情，只是本能地觉得美国应该有这样一部法律来惩罚独裁政权中参与迫害人权的中共官员。根据中共官员多数都把子女财产送到或藏匿在民主国家的事实，若有这样一部可以禁止他们入境美国......必要时可以冻结他们在美资产的法律，将十分有用。于是，我写了一封信给美国国会，请他们考虑着手立法打击中共迫害人权的恶棍。后来经过几年的反复努力推动，终于有了如今的《全球马格尼茨基人权问责法》。
  <br/>
  <br/>
  人权工作，尤其是改变独裁政权，很难立竿见影，需要有耐心扎实推进，步步为营，坚持下去就一定能取得春种秋收的必然成果。就如同中国的民主之路，虽然一波三折困难重重，可只要你我不断坚持设法推进，必将能找到铲除专制、消灭独裁、建立民主宪政的途径。
  <br/>
  <br/>
  目前，国际环境已经转向，对独裁者的绥靖政策已经成为历史，形势非常有利于我们采取行动。对此，中共也明显感觉到了，这也是为什么中国国内一片肃杀之气，由原来的“稳控”变成了直接的专政。不过，遭劫在数，在数难逃，无论中共怎样挣扎，都无法逆转末日已到的命运。也希望人权迫害者们思量《全球马格尼茨基人权问责法》条款与他们自己的关系，及早收手。
