 <p>
  上个月，瑞典文化部长林德（Amanda Lind）颁发一个奖项给遭中国关押的瑞典公民桂民海后，中国驻瑞典大使桂从友19日表示，中国已取消两个企业代表团访问瑞典的计划。这在瑞典民间和政界引发了什么后续效应？
  <br/>
  <br/>
  瑞典笔会上个月将2019年的图霍夫斯基奖（Tucholsky Prize）颁给仍遭中国监禁的书商桂民海，表彰他为言论自由发声。林德不但代表瑞典官方亲自出席还颁奖，她当时致辞表示，言论自由在瑞典是基本人权。
 </p>
 <p>
  中国当时就很生气，扬言要祭出“反制手段”。经过1个多月后，法新社报导，中国驻瑞典大使桂从友19日宣布，中国取消两个企业代表团访问瑞典的计划。
  <br/>
  <br/>
  根据瑞典官方数据，中国是瑞典的第8大贸易伙伴。
  <br/>
  <br/>
  事实上，在桂从友宣布前，瑞典外交部12月初就已经证实，中国一个访问团的行程暂缓。访问团原定12月10日要到访瑞典首都斯德哥尔摩，讨论双边贸易。
  <br/>
  <br/>
  居住于瑞典的独立中文笔会前任秘书长张钰告诉自由亚洲电台，中国这种反制措施对瑞典经济的立即影响实在有限。
  <br/>
  <br/>
  他说，早在已故的刘晓波2010年获得诺贝尔和平奖时，瑞典朝野看到邻国挪威是如何遭中国霸凌反制，就已经埋下对中国政府反感的种子。
  <br/>
  <br/>
  张钰：“瑞典虽然是比较小的国家，但是民族自尊心、国家自尊心和对民主人权的态度，还是很积极的。（瑞典两个政党）他们在国会提出来，要宣布这个大使（桂崇友）为不受欢迎的人。”
  <br/>
  <br/>
  张钰指出，他印象所及，瑞典曾在1990年代及2000年后分别驱逐过中国派驻瑞典的外交官，一次是指控中国驻瑞典的教育参赞涉及间谍活动；另一次则是涉及监控流亡瑞典的维吾尔人。
  <br/>
  <br/>
  2015年，瑞典籍的书商桂民海在泰国消失被押往中国，后来出现在中国官方媒体上认罪。从那之后中国与瑞典的关系就一直起伏不断。中国民众去年在瑞典旅游的不文明行为，遭瑞典媒体大肆报导与讽刺后，引发“辱华争议”，后来甚至演变成中国与瑞典官方间你来我往的外交照会与交涉。
  <br/>
  <br/>

  同样居住在瑞典的华裔作家万之则担心，在瑞典反移民右翼风潮兴起的形势下，瑞典政界在桂民海一案上的做法以及中国的反制方式，都让他不禁摇头。
  <br/>
  <br/>
  万之：“我觉得，西方和中国在处理这些问题上，都比较好笑。中国这种外交方式现在也是非常好笑。但是，共产党关起来的人都是英雄吗？你认为，瑞典政府敢把他（桂从友）赶走吗？这是不可能的事儿。这变得跟毛泽东一样，‘凡是敌人拥护的，我就反对’。”
  <br/>
  <br/>
  笔名万之的陈迈平关注中国民主化发展多年。他还感慨，在国际政治中，不只是经济议题可以交易，像他一样的异议人士，在需要的时候就成为西方与中国交易的一张牌，不需要时被一脚踢开，这是国际政治的现实。
  <br/>
  <br/>
  尽管这样，瑞典政府仍在为遭关押的桂民海透过各种外交渠道争取他早日获释。美国国务院亚太助卿史达伟（David Stilwell）日前就指出，瑞典驻华盛顿大使近来提出与他会面的请求，就是为了讨论桂民海的案子。但他并未说明具体细节。
  <br/>
  <br/>
  法新社报导，前瑞典驻北京大使林戴安（Anna Lindstedt）日前被控以中间人身分安排一场未经授权的会面，让桂民海的女儿与两名身分不明的男子会面，试图让桂民海获得释放。林戴安目前面临审判，最重恐遭判10年徒刑。
  <br/>
  <br/>
  截至发稿，瑞典外交部未回覆记者置评的请求。
  <br/>
  <br/>
  桂从友接受瑞典媒体《今日工业报》访问时说，桂民海不但“还活着”，还“活得很好”，强调桂民海是因涉嫌非法向境外提供机密资讯，才遭关押，他声称，中国是依法处理。
  <br/>
  自由亚洲电台记者郑崇生华盛顿报导 责编：申铧 网编：郭度
