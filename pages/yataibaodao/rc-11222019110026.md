 <p>
  美国总统特朗普22日受访时说，他和香港民众及中国国家主席习近平同在，对于是否签署国会高度共识通过的《香港人权及民主法案》，则未置可否。与此同时最新民调显示近6成美国民众认为，香港应该独立，为何美国民间如此高度关注香港？
  <br/>
  <br/>
  美国国会参、众两院火速通过《香港人权及民主法案》与禁止美国企业出口如催泪弹的非致命武器给香港的《保护香港法案》，21日这两个法案已送到白宫。特朗普22日接受福克斯电视台访问时，当主持人问他是否会签署，让法案生效，他回答自己同时与香港人民及习近平站在一起，因为习近平是他的“好朋友”，对于是否签署法案，未置可否。
  <br/>
  <br/>
  《华盛顿邮报》解读认为，特朗普可能动用否决权。
 </p>
 <div>
 </div>
 <div>
  特朗普还再次将香港议题与贸易谈判挂勾，宣称要不是因为他，习近平早就出动邻近香港的解放军，“14分钟内消灭香港”。他告诉习近平，若犯下这种“严重错误”，会对美中贸易协议产生严重影响。
 </div>

  按照美国立法程序，美国总统在收到国会送达法案的隔日凌晨起算，在不含星期天的10天内，若总统不动用否决权（Veto），法案就会自动生效为法律。
  <br/>
  <br/>
  而关于香港的这两部法案，从22日起算，扣除感恩节国会休会期，如果特朗普不动用否决权，最迟在12月11日，就会生效成法律。
  <br/>
  <br/>
  若总统动用否决权，法案送回国会后，如果参、众两院都有达2/3的议员投票同意推翻总统的否决，那么法案就会生效成为法律。
  <br/>
  <br/>
  长期关注香港局势的前美国在台协会主席卜睿哲（Richard Bush）21日受访时分析，包括香港示威者与北京当局，也可能对这个法案有误解和反应过度；他还说就算变成法律，美国行政部门的执行程度，一向有弹性存在，不是没有回旋空间，关键还是在特朗普身上。
  <br/>
  <br/>
  卜睿哲：“现在反而是他（特朗普）的立场比较两难，因为，如果他动用总统权力否决（法案），在国会两党高度共识通过的情况下，总统的否决很可能会被国会推翻，所以，他可能以他自己的方式来行事。”
  <br/>
  <br/>
  曾在香港生活的卜睿哲，对香港的忧心有来自个人的生活经验，他最近还撰写文章指出，香港这座城好像已唱起“安魂曲”，世人可能正在见证“香港的终结”。

  对于一般美国民众来说，遥远的香港却也很能感同身受。美国爱默生学院（Emerson College）最新民调显示，有高达61%的美国民众关注香港的抗议局势，还有59%的美国民众认为，香港应独立于中国之外。
  <br/>
  <br/>
  为什么美国民众如此关切香港局势？平时会关注国际新闻的美国人莫赫（Marcus Moohead）告诉自由亚洲电台，因为NBA事件，让他更加关注香港。
  <br/>
  <br/>
  莫赫：“我会说是因为NBA在香港议题上的发言、立场，以及他们的意见，让我意识到中国的影响力有多大，甚至让美国的组织、公司与商业活动，都受左右。”
  <br/>
  <br/>
  让他最沮丧的是，美国是个有言论自由的国家，NBA火箭队经理莫雷（Daryl Morey）支持香港，不应该受到湖人队明星球员詹姆斯（LeBron James）的抨击，NBA后来态度改变，让他见识到美国在商业利益与捍卫价值观中的前后不一，更让他对香港人的勇气感到敬佩。
  <br/>
  <br/>
  而尽管中国官方全面发动攻势，抨击《香港人权及民主法案》，习近平22日在北京却罕见展现低姿态指出，中国没有准备去取代谁，追求民族伟大复兴的中国梦，绝不是“霸权梦”。
  <br/>
  <br/>
  他还说中国希望在公平、平等基础上，与美国达成第一阶段贸易协议，双方目前仍积极避免贸易战，但若有必要，中方也会反击。
  <br/>
  <br/>
  不过，特朗普受访时回应，他并不喜欢习近平用“公平”协议这个字眼，也再次抨击，是中国长期以来不公平对待美国。
