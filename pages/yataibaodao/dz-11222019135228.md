 <p>
  四川石渠县温波乡两名藏人本周四在当地派出所前高呼“西藏独立”口号、抛撒传单展开示威抗议活动，随即被警方拘捕。
  <br/>
  <br/>
  四川省甘孜州石渠县温波乡境内再度发生一起示威抗议活动，两名示威者本周四（21日）将“西藏独立”传单抛撒在乡派出所前，一起高呼“西藏独立”等口号进行示威，随后一批警察立即赶到现场，将他们强行带走。

  印度南部西藏色拉寺的甘孜石渠籍僧人、境外石渠同乡会新闻发言人强巴云丹本周四告诉本台， 石渠县境内最新发生的这起示威抗议活动再度把当地藏人对政府的不满推向高潮：“根据多方消息来源证实，11月21日当地时间下午约2点30分，石渠县温波乡温波村的云丹和拉杰（别名：曲杰）两位年轻藏人在温波乡派出所前抛撒大量写满‘西藏独立’字样的传单，并接连高呼‘西藏要独立’等口号进行示威，抗议中共当局对石渠在内的全藏地实施高压统治、残暴践踏藏人人权、剥夺宗教信仰及语言使用权。他们将当地藏人长期对当局暴行所累积的怨恨及愤怒一并发泄出来，希望能引起外界的关注。”
  <br/>
  <br/>
  强巴云丹表示，云丹和拉杰在进行示威前，分别制作了小视频，表达对西藏精神领袖达赖喇嘛的无比崇敬与思念，也表达对自由的无限渴望与向往：“为了这起示威活动，他们早已做好准备，不仅在纸上用红字和黑字写满‘西藏独立’的标语传单，还特别用达赖喇嘛尊者的法像、思念尊者的歌曲‘慈悲的上师：嘉瓦益西诺布（藏人对达赖喇嘛的尊称，即如意之宝）’、‘西藏独立传单’，再加他们各自的照片后每人做成一部小视频，并分别撰写了名为《自己的方向》和《话语》的短文，与视频一起分享在他们各自的微信朋友圈里。”

  <br/>
  有关短文内容方面，强巴云丹介绍说：“云丹分享在微信朋友圈的短文是《自己的方向》，内容写道‘为我们民族的福祉利益而牺牲生命的英雄们，以及为我们民族的自由独立而坚持抗争的斗士们，你们面对杀戮和暴虐，勇往直前！这些我都有听过。如今在我们温波，那些为独立抗争的同胞们被带进黑暗囚牢，令我无比心痛。我也将毫不迟疑地随后而来，同胞们！’ 拉杰分享在微信朋友圈的短文是《话语》，内容写道‘我虽无写作才能，也无精湛文笔，更无渊博学识，但我有肩负起维护民族尊严之使命。’”
  <br/>
  <br/>
  云丹和拉杰示威被捕之前，石渠县温波乡境内在本月7日出现大批写有“为西藏独立抗争”字样的传单后，当地温波寺经师为主的6名僧人先后遭到拘捕，此后经师在本周一得以获释，其余僧人至今仍被关押在石渠县城里。
