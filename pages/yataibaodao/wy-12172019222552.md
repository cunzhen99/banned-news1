 <p>
  台湾正在如火如荼进行的总统大选和中国大陆一党独大的专制独裁形成了鲜明的对照。很多大陆人想知道，台湾究竟是怎样走向民主的呢？美国斯坦福大学胡佛研究院于在周二的研讨会上正式宣布将于明年2月对公众开放的台湾前总统蒋经国的日记，也许能够提供一些对这个问题的答案。
  <br/>
  <br/>
  在蒋经国的领导下，台湾政府先后于1987、1988年宣布开放党禁和报禁，带领台湾走向了民主化。但蒋经国做出这种决策的真实动机一直众说纷纭。胡佛研究院即将公布的《蒋经国日记》给探索这个决策的成因带来了良机。
  <br/>
  <br/>
  但蒋经国日记从1937年开始记，最后停在了1979年12月。那么，他在此前的日记中对台湾前途的思考到底是什么指向？已经阅读了部分日记的胡佛研究院东亚部主任林孝庭告诉本台记者：
  <br/>
  <br/>
  “他希望还是一个由国民党来带领的渐进式的，而不是一下实现西方的那种民主模式。他认为，这样整个政局的走向是比较稳定的。（他做了）大量的本土化工作，把台籍的精英囊括到党国体制里来，使这个政府在台湾更具有代表性。”
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  他介绍说，在1979年之前的几次选举中，已经有台湾本土的青年人，包括异议人士，进入了政坛。70年代的台湾已经开始具有了一定的民主气象。
 </p>
 <p>
  不过，哈佛大学的中国问题专家戈迪溫（Steven Goldstein）教授则在研讨会上指出，把蒋经国想象成一个民主派是错误的：
  <br/>
  <br/>
  “我认为他那个时候首先想到的是如何让台湾存活下来。台湾已经失去了和美国原有的关系，在国际上失去了合法性，那还剩下什么呢？必须要为台湾争取一些支持。他是想真的建立一个两党制的民主体系，让国民党被击败吗？我不认为是这样。”
  <br/>
  <br/>
  1979年是蒋经国记录日记的最后一年。在当年，蒋经国及其领导下的台湾也正面临着严峻的考验。尤其是美国在年初宣布与台湾断交，让台湾面临的国际形势逆转。本次研讨会上公布的《蒋经国日记》片段，记述了他当时的心情。
  <br/>
  <br/>
  蒋经国在这篇日记中以正楷毛笔字，以中国传统的竖行方式写着，“美国大使于十六日清晨时，谓有极紧急事要求来见。果不出所料，通知美国将于六十八年一月一日承认共匪，同时与我断交。当即以严肃的态度向其提出最严重的抗议，内心愤恨痛苦。”
 </p>

 <p>
  来自加州大学伯克利分校的中国问题专家高隶民（Thomas Gold）教授在七十年代多次前往台湾。他在研讨会上介绍了七十年代末台湾不断发生的各种政治抗议。他认为，这些形势也在推动蒋经国走向民主：
  <br/>
  <br/>
  “他当时知道周围正在发生什么，他也会想到自己的历史角色：这个时候，应该让台湾取得正面的进展，在国际上获得认同。这才是对台湾、或者对整个中国是有益的。”
  <br/>
  <br/>
  在政治形势发生急剧变化的同时，台湾的经济也在70年代末开始起飞。旧金山台北经济文化办事处处长马钟麟在会议上盛赞蒋经国在任上的经济成就，尤其是他当时提出并主导的十大建设工程，进一步释放了经济活力。
 </p>

 <p>
  林孝庭认为，即将公布的日记可以在这方面提供佐证：
  <br/>
  <br/>
  “台湾经济到60年代晚期、70年代初期，已经遇到一个瓶颈。台湾的基础建设，港口、铁路都不够用了。蒋经国这个时候的想法就是，台湾的经济要再上一个台阶，就需要大刀阔斧地进行改革。一旦日记公开后，这方面的讯息就会呈现出来。“
  <br/>
  <br/>
  林孝庭还介绍，这次蒋经国日记将全部公开。今年去世的中国共产党元老李锐的女儿李南央正在胡佛研究院做研究员，她非常认同这种处理蒋经国日记的方式：

 <p>
  “从历史学家的角度来说，对这种重要的历史人物来说，他的日记应该是全面公开的。从我的角度来说，日记，尤其是其中有私人生活的部分，能够帮助还原一个立体的历史人物。”
  <br/>
  <br/>
  台湾国史馆馆长陈仪深还到研讨会现场支持胡佛研究院，以学术研究为目的公布《蒋经国日记》的做法。《蒋经国日记》是由蒋经国三儿媳蒋方智怡于2004年转交胡佛研究院暂存50年，目前台湾政府正争取蒋经国日记和同样存于胡佛研究院的蒋介石日记回到台湾。
