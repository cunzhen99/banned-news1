 <p>
  自从《香港人权与民主法案》2019年10月在美国国会众议院获得全票通过之后，11月19日晚，该法案又在参议院获得全票通过，如今已送交白宫给总统特朗普签署。如此一来，其成为美国法律便已成定局，指日可待。
  <br/>
  <br/>
  按照美国的宪法与法律规定，该法案送交白宫后，从第二天起，总统必须在10天内作出是否签署的决定。总统一旦签字就立即生效，成为美国的法律。总统在10天内即使不签字，只要他不行使否决权，则该法案也会自动成为美国的法律。退一万步讲，即使总统行使否决权，该法案就会被退回国会，进入下一个程序：参众两院再行投票。只要投赞成票率达到三分之二以上，则该法案无需再由总统签字，即可自动成为美国的法律。由于白宫已经暗示总统会签署该法案，所以其被否决的可能性几乎为零。
  <br/>
  <br/>
  因此，在美国国会参众两院都是全票通过的情况下，已无人能阻止《香港人权与民主法案》这部体现美国的伟大、闪耀着正义光芒的法案变成美国的法律。
  <br/>
  <br/>
  如此高票地通过一项法案，在美国的立法史上也是极为罕见的。由此可见，中共政权的倒行逆施不仅使中国人民怨声载道，也已经点燃了美国朝野和民间共同的怒火，使得人神共怒了。
  <br/>
  <br/>
  我知道一些朋友担心，美国政府是否将真正对中共使用这部法律。我认为不必担心，既然各界的共同努力能够推动美国立出这样一部正义之法，我们的共同努力也必将能促使美国政府作出正义之举——运用这部法律惩恶扬善，打击中共暴政。
