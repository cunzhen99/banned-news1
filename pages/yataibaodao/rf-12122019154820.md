 <p>
  台湾的2020年总统大选倒数计时只剩下1个月。美国国务院亚太助理国务卿史迪威（David Stilwell）12日在华盛顿表示，美国一直对台湾定期的民主选举过程被某种程度的干预或介入有所关切，他还重申美国的既定立场强调，美国希望两岸能通过对话解决分歧，避免威胁暴力。
  <br/>
  <br/>
  敲锣打鼓用台语喊着当选（即：冻蒜）加油声，寻求连任的台湾总统蔡英文，面临国民党总统参选人韩国瑜及亲民党总统参选人宋楚瑜的挑战，台湾的选战正打得火热。
 </p>
 <p>
  在华盛顿，美国的关心也来得即时。作为国务院亚太事务的重要推手，史迪威12日在华盛顿智库战略暨国际研究中心（CSIS）一场回顾美中关系历史的研讨会上，回答自由亚洲电台记者提问时，在没有点名提到中国的情况下，表达美国对台湾定期的民主过程可能受外力影响的担忧。
  <br/>
  <br/>
  史迪威：“这场选举，是台湾民主的定期过程，每4年就会举行一次，而很不幸地，过去4年，两岸出现不该有的紧张情势。美国对于台湾的内部事务遭某种形式的介入或干预，一向都有所关切；谈到美国的《台湾关系法》与美中“三个公报”，美国希望能确保台海两岸双方通过对话解决分歧，不要有暴力、压迫或威胁的情况出现，这也是美国对台湾这次选举的期待。”
  <br/>
  <br/>
  史迪威的说法，是华盛顿在台湾选举前的最新表态，就两岸关系他重申美国一贯的立场。在会中的演说，回顾美中关系发展历史时，史迪威也不忘细数美国过去是如何帮助中国，从国民党时代的中华民国打赢第二次世界大战谈起，到共产党建立中华人民共和国后，美国又如何一步步协助中国走向世界，包括协助中国取得世界银行、亚洲开发银行的援助，还有美国引领中国进入世界贸易组织。
 </p>

 <p>
  史迪威一再强调，美国没有要遏制中国的发展，美国欢迎中国和平崛起；中国应该要自己想一 想，到底希望一个什么样的美中关系。
  <br/>
  <br/>
  史迪威：“事实是，我们知道美中之间有竞争，并不代表冲突就无法避免。我认为，认知到竞争反而能避免冲突，也就是在冲突前就告诉你哪条线在哪，别跨过这条线。这也是我们现在的政策，在必要的时候，美国必须明确表态，坚持中国必须体认与尊重美国的关切与利益。”
  <br/>
  <br/>
  史迪威还特别提到，美国一些非政府组织近来被中国形容为“黑手”，介入香港局势，这不是事实。
  <br/>
  <br/>
  而在人权问题上，他则指出，美国促进人权的立场是一致的，却又坦承美国过去确实有“自我审查”的情况存在。他强调，在美国总统特朗普的这届政府中，美国决定不能再这么做，这也是为什么除了香港外，美国在新疆问题上也积极发声。
