 <p>
  美中两国科技竞争日益激烈之际，美国愈发重视厘清本国科研项目的资金来源以及和中国的关系。美国一家研究所因为没有披露中国的资助，不得不向政府支付巨额罚款。
  <br/>
  <br/>
  本周四，美国司法部宣布，范安德研究中心(Van Andel Research Institute)因未向美国政府报备旗下两名研究员接受中国政府资助，将支付550万美元以达民事和解。
 </p>
 <p>
  <a class="external-link" href="https://www.rfa.org/mandarin/yataibaodao/junshiwaijiao/wy-11192019112212.html">
   <br/>
  </a>
 </p>
 <p>
 </p>
 <p>
  美国司法部表示，2012-2019期间，这个位于美国密歇根的生物医学研究所有两位研究人员在申请和接受美国国家卫生研究院资助的同时，也接受了来自中国的资金。
  <br/>
  <br/>
  美国政府称，其中一位研究员在上海一所研究机构担任领导，该机构是范安德研究中心和中国科学院上海药物研究所的合作成果。中国机构还曾在给范安德研究中心的一封信中说，这名研究人员接受了中国“千人计划”的慷慨支持。
  <br/>
  <br/>
  美国政府还表示，范安德研究中心在2018年6月已经知晓了中国资金的存在，却知而不报，在发表成果的新闻通稿中删除了这一项资金来源。2018年11月，国家卫生研究院专门发邮件对资助来源表达关切，但是范安德研究中心仍然没有披露实情。
  <br/>
  <br/>
  美国国家卫生研究院(NIH)要求，申请资助的机构需要披露所有的资金来源，包括其他资助（Other Support disclosures）和外国成分（Foreign Component disclosures）。范安德研究中心的做法被认为是违反了“反虚假申报法”（False Claims Act）。
 </p>
 <p>
  <br/>

 <p>
  11月，美国参议院曾发布调查报告，讨论中国的“千人计划”对美国科研事业的威胁以及美国各方的准备不足。“千人计划”是中国于2008年启动的海外高端人才引进计划，以帮助当局实现建造科技强国的战略目标。
  <br/>
  <br/>
  2018年，一个为美国通用电气公司工作的“千人计划”参与者被美国逮捕后，中国政府在网站上删除了该计划的名单。今年5月，埃默里大学的华裔研究员夫妇因没有充分公开在中国的工作情况而被开除，其中一人也得到了“千人计划”的资助。
  <br/>
  <br/>
  据《华尔街日报》披露，范安德研究中心这两位研究人员是Huaqiang Xu和Jiyan Ma，他们在基因和神经科学研究的项目上收到了美国国家卫生研究院超过3百万的资助。目前，两人都已经辞职。
  <br/>
  <br/>
  自由亚洲电台记者薛小山华盛顿报道 责编：申铧 网编：郭度
