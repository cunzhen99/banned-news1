 <p>
  国际社会持续关注香港局势。近日，全球再有近万人发表联合公开信，批评香港警方暴力执法。其中，有超过3700位各国学者参与联署，包括多位左派学者。
 </p>
 <p>
  这封发布在英国人权组织“香港监察”（Hong Kong Watch）网站的联署信谴责香港警察的暴力滥权问题，例如“对大学校园里的学生使用不合比例的报复性暴力”，使用“新法西斯语言”，称呼示威者为“蟑螂”等。
 </p>
 <div>
 </div>
 <div>
  联署信呼吁香港大学挺身维护学术自由，并要求香港政府成立“由一名公正法官领导的独立调查委员会”调查警察暴力，对相关人员进行追责。
  <br/>
  <br/>
  “香港监察”组织主席、英国保守党人权委员会副主席罗杰斯(Benedict Rogers)就公开信向记者介绍道：“有人将这封联署信交给了‘香港监察’刊登。那么多知名学者相信香港警察确实对民众使用过度武力，并决定为此发声。这说明香港警暴问题极为严重。我们希望这封公开信得以传播，并且引发更多人的关注。”
 </div>

   港警暴力 左派学者也看不下去
  </b>
  <br/>
  <br/>
  这次参与联署的学者中有不少知名学者，其中包括女性主义理论家朱迪斯‧巴特勒（Judith Butler）、实验心理学家史蒂芬·平克（Steven Pinker）、哲学家格雷林（A. C.  Grayling）、普林斯顿法学教授劳勃‧乔治（Robert George）、“当代语言学之父”乔姆斯基（Noam Chomsky）、前希腊财长、知名经济学家瓦鲁法基斯（Yanis Varoufakis）、哲学家与心理分析理论家齐泽克（Slavoj Zizek）等。
  <br/>
  <br/>
  香港“反送中”抗争运动持续将近半年，警民之间的冲突越演越烈。各国政府及人权机构除了广泛谴责香港警方暴力镇压示威者的举动外，同时也会敦促抗议者保持冷静，以和平的方式表达意见。

  但记者留意到，这封公开信批评香港警方过度使用武力的同时，并没有提到少部分示威者的暴力行为，也没有作出相关呼吁。
  <br/>
  <br/>
  值得注意的是，联署公开信的人中不少是提倡非暴力的左派学者。比如知名哲学家齐泽克就是被视为当今激进左派的重要旗手。
  <br/>
  <br/>
  对此，联署信签署人之一、旅美学者滕彪告诉记者:“我想这封联署信是对香港抗争者很有力的支持。当然有一些媒体、政府官员在表述的时候倾向于‘各打五十大板’，既要批评警方的暴力，也要批评抗争者的暴力，这个听起来好像没错，但实际上是有问题的，因为99%的暴力源于警方，是警方的举动把一部分抗争者逼得越发激进。”

  在被问道这是否意味着国际社会开始警觉到香港局势的恶化，主要是香港政府与警察所为时，罗杰斯特别提到说，香港警方以催泪弹、橡胶子弹、胡椒喷雾、甚至实弹等不对等的武力来回应示威者的和平诉求。
  <br/>
  <br/>
  他认为，在越来越多有关抗议者在被拘留期间遭受殴打、虐待和强奸的消息传出后，香港警察极其野蛮和不合理的行为使外界明白，示威者的一些暴力行为是对警方暴力的回应。
