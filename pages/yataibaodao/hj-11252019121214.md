 <p>
  在刚刚过去的香港区议会选举中，泛民主派取得压倒性胜利。那么，港府和中央政府将如何应对“亲北京”派大败？这一创下历史最高投票率的选举，又将对持续的抗争运动起到什么样的影响和作用？本台记者韩洁一直在跟踪这方面的发展，下面请韩洁介绍相关情况。
  <br/>
  <br/>
  主持人：韩洁，香港这次的区议会选举结果和以外有很大不同，可以请你介绍一下吗？
  <br/>
  <br/>
  韩洁：好的。以前建制派和泛民主派所占的席位比率将近7比3。这次却反过来，泛民主派占了85%的席位。18区共452个议席中，泛民主派取得343席，亲北京阵营只有54席。
 </p>
 <p>
  <br/>
  主持人：我们留意到18区里面有17个区都是由民主派获胜，例如以福建人居多的北角。为什么会发生这样的明显的转变呢？
  <br/>
  <br/>
  韩洁：我就刚才你提到的这个问题采访了香港市民黄先生，他是这么看这次的选举大翻盘。
  <br/>
  <br/>
  “我想这次的投票结果是由‘反送中’运动驱使，香港人有了民主觉醒。过往市民比较政治冷感，大多数人选择弃权，年长的人会因在建制派的游说下投票。现在，年轻人主要以理念同政治立场为取态，都以实现民主为主；老一辈是要以维持社会运作及秩序为取态，所以会投票给建制派。”
  <br/>
  <br/>
  我要强调的是，星期天有294万人投票，投票率高达71.2%，突破以往记录。所以外界形容这次选举是一次变相公投，反映出了香港人的真实民意。
  <br/>
  <br/>
  主持人：这次的香港区议会选举可谓意义重大。那么，结果会如何影响香港未来的政治走向？
  <br/>
  <br/>
  韩洁：首先肯定是避免了建制派垄断的局面。香港时事评论员桑普接受我采访的时候就分析说，“不会有那么多‘大小白象’工程。每一个区的小额拨款和咨询渠道掌控在香港民众的手里”。
  <br/>
  <br/>
  再来就是大家最关注的明年的特首选举。我们知道负责推选香港特首的“香港选举委员会”中，有117名委员来自区议会，民主派在这次选举中拿下过半席位，加上本地的商界就可以左右大局了。
 </p>
 <p>
  <br/>

 <p>
  主持人：我们既然提到这个特首的选举，就不得不提一下中央政府对香港区议会选举的反应了吧？
  <br/>
  <br/>
  韩洁：港府和中央政府急跳脚了。连中国外交部长王毅都“呛”港人说，无论发生什么事，香港是中国的一部分，任何企图搞乱香港的企图都不可能得逞。可见，北京官方会加强“治港”力度。
  <br/>
  <br/>
  我想特别说明一下，香港政制及内地事务局局长聂德权去年6月的时候就提到，正在研究居住内地港人的投票权安排（让居住在内地的香港人可以在内地投票），目标是2020年的立法会换届选举可以实施。有网友精辟地评论说，这意味着“十四亿人随时可以换上特殊的‘在内地生活的香港人’的身份”，此举是“洪水倒灌”。
  <br/>
  <br/>
  主持人：看来后续还有很多值得观察的地方。我们刚才说到，“反送中”抗争运动在区议会选举中扮演了很重要的角色，这次的结果会不会对运动产生正面作用？
  <br/>
  <br/>
  韩洁：桑普就告诉我，这场运动还是会继续下去，“选举是和理非影响政局的最后手段。催泪弹会从此绝迹吗？如果政府还是无视选举结果，我们还要怎么样去容忍一个暴政？香港的民意和国际的声援都在我们这一边，毅力、恒心、坚持就是一切。希望学生不要牺牲自己的生命；精神上不要崩溃；不要被仇恨淹没了自己的理想，我相信我们会闯出一条新的道路。”
  <br/>
  <br/>
  所以我想，区议会选举算是给运动打了一只强心针吧，但恐怕选举也会激起官方更大的镇压。香港理工大学还有学生被围困，门口已经聚集议员和大批民众了，警察也在场戒备。所以我们保持关注，留意接下来的发展。
  <br/>
  <br/>
  主持人：好的，谢谢韩洁。
  <br/>
  <br/>
