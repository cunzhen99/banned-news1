 <p>
  中国欧盟商会近日发布报告指出，由于中国国企占据了“一带一路”建设的主导地位，大多数欧洲企业并没有从项目中获益。
  <br/>
  <br/>
  作为提升中国外交、政治和经济实力的一部分，“一带一路”自称已经发展成为“世界级项目”。用中国国家发改委新闻发言人孟玮的话说，截至去年10月底，中国已与137个发达和发展中国家以及30个国际组织，签署了近200份“一带一路”合作文件。
  <br/>
  <br/>
  作为“一带一路”的一大终点站，欧洲市场显然占据了举足轻重的地位。但中国欧盟商会周四发布的报告却指出，欧洲企业目前在“一带一路”中，充其量只是边缘角色，机遇非常有限。
  <br/>
  <br/>
  报告说，在132家受访的欧洲在华企业中，只有20家企业表示，它们真正参与了“一带一路”的相关项目。其中，只有两家企业是通过公开信息参与其中的，其余的企业大多依靠了商业合作伙伴或与当地政府的“紧密关系”，才得以参与到“一带一路”当中。
  <br/>
  <br/>
  中国欧盟商会主席伍德克(Jörg Wuttke)在新闻稿中指出：“与真正的多边发展战略相比，中国国有企业在一带一路的建设中所获得的合同数量比例过高，背后的原因是国家补贴和廉价融资。”
 </p>

 <p>
  他强调，欧洲急需应对这种“中国模式”的出口，以保持欧企在第三方市场的竞争力。
  <br/>
  <br/>
  报告指出，“一带一路”的招投标和采购流程缺乏清晰透明度，把中外企业的不平等竞争延伸到了“一带一路”的沿线国家，导致中国国企几乎包揽了从融资到原材料供应，再到建筑工程服务等领域的“一条龙”业务。
  <br/>
  <br/>
  中国外交部发言人耿爽周四批评这份报告“有失偏颇”。他的理由是，包括德国西门子公司(Siemens)在内的不少欧企，已经“积极”参与到了“一带一路”的建设当中。
