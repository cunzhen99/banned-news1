 <p>
  美国首都华盛顿保守派重要智库传统基金会（Heritage Foundation）22号举行年度总裁俱乐部会议，蓬佩奥面对理念相同的支持者，大谈美国价值，他多次强调总统特朗普（Donald Trump）领导的政府，在外交政策上始终如一，努力彰显美国追求民主与自由的决心，而中国则成了与美国价值的最大对照组。

  他说，中国充其量是美国的“战略竞争者”，但中国试图输出与美国完全相反的意识形态，更企图影响美国企业的言论自由与经营，美国职篮NBA近来在香港议题上的表态，就是一例。
  <br/>
  <br/>
  蓬佩奥说: “中国以高压手段及贪腐的方式治国，对美国来说，在一些有歧见的事情发声，现在更显重要，事实必须要得到阐述。而显然地，我们已经取代了NBA，扮演着对中国说实话的角色。”

  台下一片掌声，为蓬佩奥赞好，因为蓬佩奥不点名批评的对象，是NBA湖人队球星、有詹皇之称的詹姆斯（LeBron James）。詹姆斯日前批评挺香港的火箭队经理莫雷（Daryl Morey）是“被误导或没读书、搞不清楚状况”。他的谈话，在美国舆论引发争议，外界质疑詹姆斯双重标准，只会“对美国人大声吼，对中国跪舔” 。特朗普也曾在白宫揶揄NBA不敢表态支持香港，是在向中国示好。
  <br/>
  <br/>
  蓬佩奥致词强调，说实话与真相的重要，而他坦言，自已过去在担任国会众议员期间，对中国的崛起坐大、逐渐威胁美国利益与价值的认识不够，也做得太少。他提到华为的例子指出，现在包括欧洲国家也逐渐了解，华为崛起造成的挑战与风险是“更大且更深层次的”。
  <br/>
  <br/>
  蓬佩奥在另一个保守派智库哈德逊研究所新成立的播客（pocast）平台受访时，讲得更直白。他点名批评中国，在香港议题上破坏自身承诺，对新疆问题也直言不讳。

  蓬佩奥：“我们已经看到了中国共产党的核心问题，就是总是破坏承诺。习近平说不会将南海军事化，但他食言。在香港问题上也是，我们得确保中国不破坏“一国两制”的承诺。至于新疆问题，情况真的很糟，那确实是属中国主权的内政事务，但中国正在从事大规模的人权迫害。”
  <br/>
  <br/>
  特朗普政府中批评中国态度坚定的除了蓬佩奥，副总统彭斯也在美国的中国政策上扮演重要角色。
  <br/>
  <br/>
  彭斯预计24日将于威尔逊中心再次发表针对中国的演说。这是他去年针对中国的尖锐演讲后，再次针对中国发表主题演讲，而这正值美中贸易谈判及香港局势发展仍没有定数之时，彭斯的讲话会以何种声调呈现，会是美中关系下阶段发展的最新观察指标。
