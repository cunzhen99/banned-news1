 <p>
  香港民主党刚刚当选区议员的黄国桐，11月26日因公务前往澳门，被澳门以“可能危害澳门社会安全”为名拒绝入境。黄国桐当日与另一名曾在抗议运动中曾经被捕的港人一起被遣返。
  <br/>
  <br/>
  上周末香港的区议会选举，黄国桐代表民主党参加九龙太子社区的区议员选举，并成功以两千五百多票胜出。本周二，本身是律师的黄国桐，因公务前往澳门开会，却在入境时被拒绝进入澳门。正在乘船返回香港的黄国桐对自由亚洲电台表示，澳门有关当局拒绝他入境的理由是怀疑他可能破坏澳门安全。
 </p>
 <div>
 </div>
 <div>
  “在拘留室内有另外一位人士，也是被拒绝入境的。我是被扣了两个多小时，然后我们两人被遣送出境，不许进澳门。理由是：有明显的证据显示我们两人企图进入澳门特别行政区，从事危害澳门公共安全和公共秩序的活动。”
  <br/>
  <br/>
  黄国桐表示，澳门边境人员把他带到一间闻讯室内，两个多小时后把他遣返香港。在当选区议员之后，黄国桐曾和约六十名当选区议员一起前往香港理工大学，要求警方撤销对学校的封锁，并与另外四名议员一起进入理工大学查看情况。
  <br/>
  <br/>
  美联社星期二报道说，香港理工大学副校长卫炳江当日带人进入理工大学，发现校园内的抗争者已经基本撤离。他们组成的七个小组在校内巡查，只发现了一名身体虚弱的女性抗议人士。
  <br/>
  <br/>
  黄国桐认为，校方接管理工大学的日常管理之后，警方就没有理由继续围困大学校区。当天，理工大学校方发表声明，要求警方撤除包围。
  <br/>
  <br/>
  他透露，周二另一名曾在抗议行动中被捕的香港居民邱先生，也被澳门拒绝入境。邱先生表示，他9月被警方以非法聚集的罪名拘捕过一次，没想到已不能入境澳门。
  <br/>
  <br/>
  “我是9月29日被警察抓过，说我是非法聚集，这个月29日上庭。我只是到澳门去玩、吃饭，就只是这些而已。”
  <br/>
  <br/>
  上周末的区议会选举，结果泛民主派获得压倒性的胜利，取得85%的议席，而支持香港特区政府的亲中党派和团体议席，比上一届减少了80%。不过，香港英文的《南华早报》分析认为，香港的抗议者可能不会因为一次选举胜利而降低对政府的压力。
  <br/>
  <br/>
  黄国桐表示，区议会选举只是香港市民抗争的其中一个环节，目前香港市民士气高涨，大家会继续抗争。
  <br/>
  <br/>
  “选举只是要在制度内进行抗争，两个方法是并排的，街头抗争和制度抗争不离不弃。说因为选举胜利就会放弃街头抗争，百分之一百不会的。”
  <br/>
  <br/>
  同船返回香港的邱先生表示，自己并非前线的所谓勇武派，只是经常上街参与和平游行抗议。他也认为，在五大诉求达成之前，香港市民不会放弃抗争。
  <br/>
  <br/>
  “如果不公平当然会要继续抗争。选举是基本法白纸黑字写在那里的，不能抵赖说不给。就像澳门赌场都有赌场的规则，输了钱说不给你，那肯定是不行的。”
  <br/>
  <br/>
  周一和周二，香港金融区的核心中环，继续有大批市民中午午餐时间聚集在街头呼喊口号。《南华早报》认为，虽然泛民主派在区议会选举获得胜利，但香港的街头抗争活动，恐怕还将持续一段时间。
  <br/>
  <br/>
  自由亚洲电台记者石山华盛顿报道。 责编：申铧 网编：郭度
