  阿齐兹：“现在用你的手机搜索一下中国正发生什么事，了解他们是如何设立集中营，把无辜穆斯林关起来。”
  <br/>
  <br/>
  就是这段中国在新疆设置集中营关押穆斯林的指控，让她的TikTok帐号被封号1个月。不过这个视频11月26号开始在社交媒体推特上热传。11月27日，阿齐兹在个人推特上指出，她的Tiktok帐号已恢复使用，查询她的TikTok帐号，还能看见她那则夹带关注新疆人权的短视频。截至发稿为止，已突破50万人转发。
  <br/>
  <br/>
  TikTok是抖音的海外版，非常受美国年轻世代欢迎，包括美国知名歌手爱莉安娜·格兰德（Ariana Grande）也是活跃用户。
  <br/>
  <br/>
  <b>
   人权人士不怀疑中国的长臂管辖
  </b>
  <br/>
  <br/>
  然而，就在美国政界从国会到外资审议委员会（CFIUS）都对这个中国企业专为海外市场打造的社交媒体平台心存怀疑之际，美国使用者谈论新疆再教育营的视频，究竟是不是遭到TikTok的中国母公司字节跳动审查而一度下架？美国的人权组织“声援维吾尔人运动“（Campaign for Uyghurs）创办人阿巴斯（Rushan Abbas） 告诉自由亚洲电台，她一点都不怀疑中国官方会对中资企业在海外长臂管辖。
  <br/>
  <br/>
  阿巴斯：“中国试图影响全世界，中国不只伸长手臂到海外的中国人，甚至还企图影响世界其他国家的人民。”
  <br/>
  <br/>
  阿巴斯说，这也是为什么她不使用微信或TikTok的原因。
  <br/>
  <br/>
  TikTok在美国年轻世代的影响力，已经可以与社交媒体巨头脸书一争高下。美国陆军甚至利用TikTok发布征兵广告。美国国会表达安全关切后美国陆军部长麦卡锡（Ryan McCarthy）21日表示，美军已开始对TikTok进行安全评估。

 <p>
  陆军发言人22日则证实，已指示陆军学员不得着军装使用TikTok，因为这一软件处理用户信息的方式“令人担忧”。
  <br/>
  <br/>
  洛杉矶的独立学者吴祚来告诉自由亚洲电台，美国政府的担忧是有道理的，因为，连美国苹果公司要在中国做生意，都得把中国的手机用户资料库放在中国，更何况是有中国血统的公司，而关键还不只是内容审查，最让人担用的是用户在手机上的个资被搜集。
  <br/>
  <br/>
  吴祚来：“他对美国的公司都这么严厉控制，你可以想像，他对中国的公司会进行什么样的控制，这个是防不胜防。所以，美国必须要在信息产业上、从华为开始，到抖音，都必须要升级对他的监管。”
  <br/>
  <br/>
  截至发稿，TikTok并未回覆记者查询。
  <br/>
  <br/>
  <b>
   字节跳动否认暂停账号与新疆视频有关
   <br/>
  </b>
  <br/>
  针对阿齐兹的个案，TikTok母公司字节跳动告诉《纽约时报》，是因为阿齐兹之前发布一段包含恐怖组织基地组织首领奥萨马·本·拉登（Osama bin Laden）图像的视频，违反TikTok针对恐怖主义内容的政策。
  <br/>
  <br/>
  TikTok 10月时的书面声明指出，TikTok美国用户数据全部储存在美国国内，并在新加坡备份。数据中心完全位于中国以外的地区，数据不受到中国法律的约束。
  <br/>
  <br/>
  声明还说，在内容审查方面， Tiktok没有移除任何与中国相关的敏感内容，也从未被中国政府要求移除任何内容。并表示“如果被要求，我们也不会那么做。”
  <br/>
  <br/>
  不过，英国《卫报》9月份曾披露 TikTok 内部文件显示他们有内容审查指南， 如“天安门”、“藏独”及“法轮功”，都是遭封杀的敏感词。
