 <p>
  武汉肺炎病毒的蔓延已造成湖北省六个城市相继封城。目前已经证实，这一新型冠状病毒来自被人食用的野生动物。但自2003年非典之后，中国政府曾下大力整顿野味市场。为什么野味市场仍然屡禁不止，并最终导致了本次病毒爆发呢？
  <br/>
  <br/>
  <b>
   武汉肺炎祸起野味市场
  </b>
  <br/>
  <br/>
  在周二的国务院新闻发布会上，中国科学院院士、中国疾病预防控制中心主任高福已经表示，本次武汉肺炎病毒的来源是当地一家海鲜市场非法销售的野生动物。众多线索指认，这家市场就是武汉华南海鲜市场。虽然称为海鲜市场，但民众可以在这里买到大量野味。
  <br/>
  <br/>
  国家卫健委高级别专家组组长钟南山院士对媒体表示，病毒可能来源于竹鼠或獾一类的动物。但北京大学等院校的研究人员22日在《医学病毒学杂志》在线发表的一篇研究论文称，蛇是最有可能带有2019-nCoV新型冠状病毒的野生动物。
  <br/>
  <br/>
  武汉疫情与野味之间的关系曝光后，不少网友纷纷把这一事件与2003年非典爆发相提并论，指责政府管理失职，没有有效约束野味的销售，才酿成此祸。
  <br/>
  <br/>
  一位不愿意透露真实姓名的先生曾在广东生活过几年，他告诉本台记者，
  <br/>
  <br/>
  “长江以南区域，野味市场比较大，而且当地市场的管理也就是睁一只眼、闭一只眼。”
  <br/>
  <br/>
  <b>
   中国野味市场的乱象
  </b>
  <br/>
  <br/>
  在加拿大滑铁卢大学进行中国食品安全研究的司振中博士分析了这种市场乱象，
  <br/>
  <br/>
  “这毕竟是一个小规模的、灰色的市场，（政府）并没有完全的监管，而且政府也没有相关的严格的数据，存在这种情况也没有办法。”
  <br/>
  <br/>
  中国对野生动物养殖和销售有严格的制度规定，普通商户必须有《水生野生动物经营许可证》、《水生野生动物驯养证》等许可证才能进行经营，并且还有诸多限制条件。例如：野生娃娃鱼不能直接以食用为目的销售，而必须是人工养殖的第二代之后，才能食用开发。
  <br/>
  <br/>
  2003年非典之后，食品市场进行了大力的清理整顿。当时一度认为，导致非典流行的萨斯病毒来自人们食用的野生动物果子狸。广东政府为此下大力气整顿食品安全市场，新出台了野生动物养殖业的措施，表示要依法查处未经批准养殖、引进和出售野生动物的活动。
  <br/>
  <br/>
  2017年，广东省又首创了活禽市场的“1110”防疫制度（即：一日一清洗，一周一消毒，一月一休市，过夜零存栏），被农业部在全国推广。
  <br/>
  <br/>
  <b>
   中国人喜好野味的社会文化因素
  </b>
  <br/>
  <br/>
  虽然如此，依然挡不住中国人对野味的喜好，庞大的需求使野味市场一直在暗地里运作。司振中认为，中国人对野味的喜好源自诸多社会文化因素，
  <br/>
  <br/>
  “野生动物的消费是一种社会地位的体现，中国人聚餐的时候，希望能够显示自己的经济能力，野生动物价格比较高，比较稀有。”
  <br/>
  <br/>
  同时，中国人传统上认为野生动物来自纯天然，没有食品添加剂等等。另外，中医下药也采用野生动物，让很多中国人认为，野味能增强体质。
  <br/>
  <br/>
  司振中指出，要杜绝野生动物传染的病毒，最好的办法就是全面禁止野味的销售。
  <br/>
  <br/>
  目前，在中国诸多大城市普遍存在活禽市场，但这些市场的管理不一定很规范。司振中此前已公开表示，中国活禽市场的主要问题之一，就是需要更好地去执行既有的食品安全制度。
  <br/>
  <br/>
  本次武汉肺炎疫情扩散后，病毒的发源地武汉华南海鲜市场已于近日关闭。但据红星新闻网报道，武汉市市场监督管理局曾于2019年9月25日通报称，武汉市区两级政府部门开展野生动物市场专项整治行动，未发现违法经营行为。此时据武汉确诊第一例新型肺炎病毒感染病例，还不到三个月。
  <br/>
  <br/>
  (记者：王允 编辑：申铧） 网编：郭度
