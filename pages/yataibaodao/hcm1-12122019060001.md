 <p>
  香港浸会大学学生会长方仲贤日前质疑民进党，只想用香港人的鲜血换取台湾人的选票，在台湾政坛掀起波澜。前国民党主席朱立伦趁势回应，国民党应该支持《难民法》。长期关注人权议题的民进党立委尤美女则反讽，国民党长期不关心人权，现在愿意来关心香港是一件好事。
  <br/>
  <br/>
  朱立伦11日回应媒体提问时表示，支持香港所有人民追求民主人权，是国民党一贯坚持的立场。他们不希望有任何政党或个人，想要把香港争取民主人权的过程，当作提款机、当作选举战略，而是真正关心香港人民、关心香港民主。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  朱立伦强调，所以对于《难民法》的通过，国民党应该予以支持，《难民法》躺在那边那么多年，而民进党今天若真正关心香港民主，就应该积极推动这样的法案。
 </p>
 <p>
 </p>
 <p>
  <iframe frameborder="0" height="620" scrolling="no" src="https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRFAChinese%2Fvideos%2F447968629220693%2F&amp;show_text=0&amp;width=622" width="622">
  </iframe>
 </p>
 <p>
  <br/>
  朱立伦12日再度提及，台湾加入两公约（公民权利和政治权利国际公约）之后，在人权议题上，台湾就有义务，但是有太多的政治考量，最近因为香港的议题，大家才突然觉得这个议题真的非常重要。
  <br/>
  <br/>
  朱立伦：“没想到我们现在的执政党，号称最支持香港民主人权的，原来只是一个吃香港人豆腐或者是只是关心的假议题，把它当作提款机。如果真的那么重视的话，那两公约当中最重视的难民法我们为什么不能支持？”
 </p>
 <p>
 </p>
 <p>
  长期关心中国与香港人权议题的民进党立委尤美女12日接受本台访问时指出，民进党跟国民党最大的不同，就是所追求的价值不同。民进党追求的就是民主、自由、法治、人权；香港为了追求他们的自由与民主，学生们前仆后继地持续六个月，民进党当然是支持香港。
  <br/>
  <br/>
  尤美女：“国民党一直在追求所谓的拚经济，对人权这一块从来就不关心。当然朱立伦他们现在愿意来关心香港议题，这是一件好事，我们不希望他只是流于‘口惠而已’。”
  <br/>
  <br/>
  尤美女批评，朱立伦根本不清楚《难民法》，就算《难民法》通过，香港人也不适用，她附和蔡英文总统日前提出，香港要用的是《港澳条例》，而不是《难民法》。尤美女说，香港的问题还是发生中的事，所有需要协助的个案，台湾政府都已经尽到最大的努力去协助。
  <br/>
  <br/>
  <strong>
   港生来台入学或转学方案出炉 各校外加10%名额
  </strong>
  <br/>
  <br/>
  事实上，台湾的教育部11日已跨部会推动各大学专案办理招生试办计划，协助港生以新生入学或转学方式来台读书，台湾、成功与中兴大学已通过核准，另有清华、交通大学、台北医学大学等校申请中。
 </p>
 <p>
 </p>

  <br/>
  <br/>
  根据台媒苹果日报报导，明年2、3月为联合分发报名期，各校可供10％外加名额，港生可循此管道来台就学。以成大公布“香港专案转学”为例，不收转学报名费，另搭配“协助香港地区大学学生紧急纾困助学金”，录取者可申请机票、学杂费、住宿费等补助。
  <br/>
  <br/>
  自从香港“反送中”运动以来，台湾基进党也投入协助港人来台安置事宜。基进党主席陈奕齐12日接受本台访问。
  <br/>
  <br/>
  陈奕齐：“香港人他们的不幸，从过去雨伞运动累积到今天‘反送中’，突然间让台湾人民看到‘一国两制’是这么可怕，从来不是让蔡英文政府捡到枪，而是让台湾人民突然警觉、警示到‘一国两制’下的处境是这么可怕。蔡英文何德何能可以介入香港‘反送中’运动，这是香港林郑特首还有律政司司长郑若骅他们搞出来的，可能是北京授意，这跟蔡政府有什么关系？”
  <br/>
  <br/>
  <strong>
   协助香港 林荣基：可理解台政府只能说不能做
  </strong>
  <br/>
  <br/>
  从香港逃亡至台湾超过半年的香港铜锣湾书店前店长林荣基接受本台访问时认为，朱立伦只是顺势捡起枪，在炒作方仲贤的声明罢了，更何况方仲贤已经发文道歉表示后悔，那就表示这把枪都已经坏了，捡起坏掉的枪能有什么作用？
  <br/>
  <br/>
  林荣基：“一个政党是提出好好保护台湾的民主自由，有一个是认同大陆所谓的’统一’，你关注台湾你会很清楚，国民党是什么想法。他的说法对年轻人影响不大，对老一辈的台湾人还是有一点影响，但是不大。这很明显是为了总统选举而说的，带出来那个风向往那个方向走而已。”
  <br/>
  <br/>
  林荣基说，他能理解台湾政府只能做不能说的情况。对于香港在意的《难民法》现在不可能通过，从他的角度来看，台湾其实非常保守，他有很强的保护性，要台湾政府现在很快通过《难民法》，蔡英文和民进党的选票流失是很大的。
 </p>
