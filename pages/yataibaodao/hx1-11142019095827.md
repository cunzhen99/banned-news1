 <p>
  台湾，国民党总统参选人韩国瑜周四与外媒会面，对香港警民对峙升高，他提出“反港独、双普选”六字解方。韩国瑜提醒北京当局，民主自由不是洪水猛兽，是普世价值。不过被问到是否后悔进香港中联办？韩国瑜并没有正面答覆。
  <br/>
  <br/>
  韩国瑜去年当选高雄市长后，曾进入北京驻港官方机构香港中联办进行拜访，引发香港人民反感。加上六月份香港反送中运动百万人上街，韩国瑜被问到此事表示不清楚。相较蔡英文多次公开声援香港抗争者，韩国瑜在反送中议题上明显失分。
 </p>

 <p>
  距离台湾总统大选不到2个月，韩国瑜14日和外媒会面前预告锁定两岸政策和香港问题。特别在香港议题上，他花了五分多钟阐述立场，有备而来展现夺回话语权的企图。
  <br/>
  <br/>
 </p>
 <p>
  <iframe frameborder="0" height="500" scrolling="no" src="https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRFAChinese%2Fvideos%2F2471316273194841%2F&amp;show_text=0&amp;width=622" width="622">

 <p>
  <b>
   韩国瑜：反港独、双普选能为香港降温
  </b>
  <br/>
  <br/>
  韩国瑜对升高的香港警民对峙提出解方：“大概六个字、六个字，香港的问题、核心问题基本上就全部表述清楚了：‘反港独、双普选’。香港未来领导者特首，直接港人直选；香港的议员，直接直选。然后有基本法，反对香港独立。也就是‘反港独’，北京政府放心；‘双普选’香港市民放心。”
  <br/>
  <br/>
  韩国瑜指出，过去支撑香港繁荣和国际金融地位的有四个柱子，就是法治、效率、自由化和廉节的政风。看到反送中令香港动乱不安，年轻人肉体受到重大伤害、丧失生命，他心里非常不舍。
  <br/>
  <br/>
  韩国瑜说，北京当局时常对外面讲“心灵契合”，如果北京当局、香港政府，要达到跟香港人民“心灵契合”，就应该立刻订定给港人“双普选”，相信香港情势就能因此迅速降温。
 </p>

  韩国瑜说：“我也要呼吁北京，民主自由不是洪水猛兽，民主自由是普世价值。”
  <br/>
  <br/>
  韩国瑜说，“中华民国”在台澎金马实施总统、国会等民意机关由人民直接选举至今数十年，台湾社会繁荣稳定，也没搞台独。相信北京给香港自由民主，也能像台湾一样。
  <br/>
  <br/>
  至于被问到一旦香港抗争者流亡台湾，他对通过难民法的态度。韩国瑜则没有回应。

 <p>
  韩国瑜指出，警察连香港的大学都进去抓人了，这违反了学术自由。大学是神圣不可侵犯的殿堂。他还提到警察开枪，香港局面如果持续恶化，对北京当局来说，“一国两制”会受到严厉的挑战。
 </p>
 <p>
  美联社记者问韩国瑜“有没有后悔去中联办？”  因为他过去踏进过在香港的中联办，被贴上亲中标签。
  <br/>
  <br/>
  韩国瑜说：“如果今天去中联办吃了一顿饭，我就抛弃了台湾所有自由民主的价值，抛弃了中华民国，那中华民国跟自由民主的价值，也太没有行情了。”
  <br/>
  <br/>
  韩国瑜还反批民进党的官员去见国台办官员、去中联办，都没事，但是国民党的人这么做却被骂“卖台”，这是双重标准。
 </p>

  <b>
   韩国瑜：两岸关系提“两个不要怀疑”
  </b>
  <br/>
  <br/>
  对于两岸问题，韩国瑜提出“两个不要怀疑”：“第一个不要怀疑，你可以怀疑太阳明天会不会从东边起来，不要怀疑共产党一定有收复台湾的决心 。第二个不要怀疑，你可以怀疑太阳明天会不会从东边起来，但是你不要怀疑台湾两千三百万人强烈追求自由民主的决心。”
  <br/>
  <br/>
  《经济学人》杂志记者提问，中国大陆领导人习近平今年初的谈话，将“九二共识”和“一国两制”划等号，国民党没有接受“一国两制”，若执政要怎么跟北京开始对话？
  <br/>
  <br/>
  韩国瑜重申“九二共识，一中各表”是两岸关系定海神针。路透社追问，中国大陆领导人习近平提的是“共同努力谋求国家统一的九二共识”，韩的“九二共识”定义是否包含习的“国家统一”？
 </p>

  韩国瑜只重申：“我们对九二共识的界定，非常清楚，一中各表，我们是中华民国在台澎金马，我们有自己完整的宪政体制，这方面是非常确定的。”
  <br/>
  <br/>
  韩国瑜强调，马英九的“九二共识，一中各表”，没有出卖中华民国，他沿续马总统的这个基础不会改变，不会掀起惊涛骇浪。他自认有信心当选总统，以“和”为目标与态度，能恢复两岸对话。
  <br/>
  <br/>
  <b>
   韩：北京一天不放弃武力犯台 没有条件签和平协议
  </b>
  <br/>
  <br/>
  至于习近平今年初抛出“一国两制”台湾方案。韩国瑜说：“邓小平的‘一国两制’，在台湾完全没有市场。港澳的‘一国两制’如果有人认为要套在中华民国台澎金马的头上，他是大错特错或是说别有居心。”
 </p>
 <p>
  如果国民党重新赢得执政，是否要和中国大陆签订和平协议？韩国瑜说：“两岸和平协议，在中共没有放弃武力犯台情况下，我个人认为还没有这条件，所以国防政策要强化，台湾两千三百万人如果没有共识，没法整合。”
 </p>
