 <p>
  自从6月9日，香港百万人上街反送中至今整整三个月了。中共在香港的不断施暴，使得世人......，尤其是更多的香港市民进一步认清了中共专制独裁政权，反民主自由的反动本性。这更加坚定了港人，必须争取到双真普选的决心。没有双真普选，其它的即使得到了，也都可能随时失去。那样一来，香港将彻底沦陷到中共的魔爪之中，港人未来的命运将和内地人民如今正在遭遇的一样，黑监狱、黑头套、绑架失踪、殴打软禁等状况，即使可能形式有所不同，但实质上将会相差无几。
  <br/>
  <br/>
  最近有关香港，包括要求将中国共产党列为恐怖组织的多个白宫签名，全都早已超过了所需要的100,000数字。在未来的60天内，白宫将必须对此作出公开回应。最近港人更是直接表达出，要美国出手帮助香港，制止中共在港的暴行。今天美国国会已经复会了，很多人期待《香港民主与人权法案》马上被排上国会两党的议事日程，使之尽快通过成为法律，从而起到协助港人摆脱困境的作用。
  <br/>
  <br/>
  美国无愧于侠义之国的称号。不管你是否认同这样的说法，无论世界任何地方发生天灾或人祸时，大家都期待美国带头出手相助或解决问题，就是美国是希望灯塔的最好证明。尤其是其，成功的民主榜样性作用。如今的香港也不例外地有着同样的期待，而且这样的期待多数人都认为是合理的。也许一些美国人自己不这样认为，也正是因为这些年对美国在全球重要性的认识不够，而让美国失去了很多决断和做得更好的机会。也许你对美国以前政客的不坚持美国立国之本的原则，甚至出卖这些原则很失望，对其绥靖政策深恶痛绝，但美国就是美国，一旦意识到了问题的存在就必定能够纠正它。当然，凡有人的地方就有的问题另当别论。
  <br/>
  <br/>
  诚然，外因是要通过内因起作用的。中国的民主之路还是要靠我们中国人自己勇敢地去走出来。但是毫无疑问，美国会起到灯塔般的指引和推动作用。只有我们自己去做了，别人才有机会去帮助我们。另外，不要以为防火墙真的那么强大，使得墙里的人对香港发生的事一无所知，其实很多觉醒的公民时刻在网上与中共对视着。“我不说，但是我知道”，用来形容目前国内网上的一片肃杀之气再合适不过了，“于无声处听惊雷”。
  <br/>
  <br/>
  也许美国有关香港的反应将会成为照亮国人心灵的希望之光，世界上认清中国邪恶本质的人们和香港人民一起期待着。
