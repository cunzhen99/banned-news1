 <p>
  11月26日，中国国务院发布通知，规定第二轮土地承包到期后再延长三十年。有学者告诉本台，不断延续土地承包制仍然是一个粗糙、将就的方法，是脚踩西瓜皮，走一步算一步。
  <br/>
  <br/>
  国务院发布的《关于保持土地承包关系稳定并长久不变的意见》指出，保持土地集体所有、家庭承包经营的基本制度、农户依法承包集体土地的基本权利长久不变，同时保持农户承包地稳定。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  自实行家庭承包经营以来，中国已经先后两次延长承包期限。第二轮土地承包从1997年开始，到2027年底到期。此次延长之后，现有的土地承包关系将延续至2057年底。
  <br/>
  <br/>
  美国三一学院经济系终身教授文贯中认为,提前十年左右宣布这一政策，有一定的积极意义：“至少给农民一个预期，使用权还是归农民，如果农民要在土地里面进行一些投资、经营计划，就有一个比较长的时间实行计划。”
  <br/>
  <br/>
  北京宪政学者陈永苗指出，延长土地承包制可能是为集体土地流转做准备，缓解经济下行的压力：“经济下行可能要印发货币，货币一出来，如果没有新的土地入市，可能会出现通货膨胀，这个蛋糕的盘子未必有以前大，但是它需要有新的蛋糕。”
  <br/>
  <br/>
  <b>
   三权分置 放活经营权
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>
  <div class="image-inline captioned" style="width:1200px;">

 </p>
 <p>
  中国国务院发布的意见书还提出，要完善落实农村土地所有权、承包权、经营权三权分置的体系。
  <br/>
  <br/>
  中共中央曾于2016年首次以文件形式确认了这一政策体系，在集体所有权不变的基础上，将原有的承包经营权拆分成三权，允许自愿有偿转让土地经营权。
  <br/>
  <br/>
  文贯中认为这一做法毫无新意，实际上80年代开始就在做的事，通过30年多的实验证明是一个失败：
  <br/>
  <br/>
  “有效率的农民试图通过经营权转让获得土地（经营），单独都要给农民签合同。如果你没有发财，那就给几百块。如果你发财了，这些农民都会来咬你一口，‘所以你现在不能给我500块了，要给我5000块’。所谓契约精神，在农村是非常淡的。而且这种经营权得不到法律保障，农民说要回去就要回去。”
  <br/>
  <br/>
  文贯中说，“在全世界，都是用所有权（私有）来杜绝这个问题，使交易成本大大下降，一次性买断。”
  <br/>
  <br/>
  对于众人关心的农民进城落户是否会影响承包土地，该规定提出，原有的土地承包权也不会改变，但农户可以依法自愿有偿转让。
  <br/>
  <br/>
  文贯中指出，进城落户和土地承包权不挂钩，这是对农民在城市很难有铁饭碗的补偿，但其实不如改革户籍制、或者由政府向农民工提供失业津贴、帮助其再就业，也避免了土地资源的浪费。
  <br/>
  <b>
   <br/>
   土地承包制非长久之计？
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>
  <div class="image-inline captioned" style="width:1000px;">
   <div style="width:1000px;">

   </div>
   <div class="image-caption">
    <span style="width:1000px;">
     中国农业部长韩长赋。(Public Domain)
    </span>
    <span class="copyright">
    </span>
   </div>
  </div>
 </p>
 <p>
  农业部部长韩长赋去年在答记者问时表示，延长土地承包制与第二个百年战略构想在时间上高度契合--2050年前后，中国将建成社会主义现代化强国，经济结构、社会结构，包括城乡关系、工农关系都会发生大变化。
 </p>
 <p>
  但是，文贯中却指出，农村土地承包制只能是权宜之计，无益于解决三农问题，最终应该实施土地私有制，或者允许农民从强制性的土地集体所有制退出：
  <br/>
  <br/>
  “从（19）78年到现在已经是40年了，三农问题越来越严重。我可以预期，你再搞30年也是没用的。”
  <br/>
  <br/>
  他阐述道，强制性的土地集体所有制阻碍了生产效率，无法促进一批务农能力和意愿较强、效率较高的农民最大化地发挥优势，也妨碍其他农民加入第二、第三产业。
  <br/>
  <br/>
  “中国已经是世界第二大经济体，但是留下一个又穷又低效的农业，一个破破烂烂的农村，和相对贫困化的农民。” 文贯中说。
  <br/>
  <br/>
  陈永苗认为，延长农村土地承包制总体上无法帮助社会财富和土地利益的重新分配：
  <br/>
  <br/>
  “城乡结合处可以卖钱的地，实际上土地租金已经吃完了，中国的土地财政已经用完了，没有什么太大的空间；农民种的地，粮食（价格）涨不上去，工农业剪刀差还在，人家也没有太多种地的欲望。”
  <br/>
  <br/>
  陈永苗强调，不断延续土地承包制仍然是一个粗糙、将就的方法：“长久之计大家就别想了。现有的政治框架下不可能解决这个问题。现在就是脚踩西瓜皮，走一步算一步。”
  <br/>
  <br/>
  农村土地集体所有制是中国特色，起源于大跃进时期。1982年的宪法修订进一步明确，城市土地属于国有，农村和城市郊区的土地，除非法律特别规定，都属于集体所有。
  <br/>
  <br/>
  自由亚洲电台记者薛小山华盛顿报道 责编：申铧 网编：郭度
 </p>
