 <p>
  台湾调查局21日公布调查结果指出，日前有国民党总统候选人韩国瑜粉丝散播蔡英文重要幕僚陈菊涉挪用公款500亿的假讯息，最早出现于“香港凤凰新闻社”网站 。这家媒体是在中国香港登记。
  <br/>
  <br/>
  一名台湾女韩粉今年8月在脸书分享假讯息，指称总统府秘书长陈菊挪用公款至少500亿，被陈菊控告诽谤。台北地院日前依诽谤罪判杨女拘役50日，或者缴纳罚金5万元。
  <br/>
  <br/>
 </p>
 <p>
 </p>
 <p>
  调查局指出，假讯息防制中心于今年9月调查多起假讯息案，内容皆为脸书用户转传，指调查局人员爆料陈菊贪污500亿元，以作为民进党竞选资金；经溯源调查后发现，相关内容最早于今年4月9日刊载于香港凤凰新闻社网站，随后出现在中国“希尼”论坛、脸书“Howard Fang”个人网页等，并逐渐散布至其他社群媒体及粉丝专页。
  <br/>
  <br/>
  调查局表示，原贴文内容并无“本文系调查局官员传出”等文字，但自今年4月12日，Howard Fang脸书贴出该则假讯息后，即出现提及调查局等字眼，自此转传到各粉丝团的贴文内容，也都加上调查局的名义。
  <br/>
  <br/>
  <b>
   凤凰新闻社官网：拥护中共
  </b>
  <br/>
  <br/>
  被台湾调查局指涉散播陈菊假讯息的“香港凤凰新闻社”，其官网介绍该公司2015年11月在中华人民共和国香港特别行政区注册，是继凤凰卫视、凤凰网、凤凰通讯社之后，在香港崛起的第四大凤凰系列媒体机构，并且拥护中国共产党和中国政府的领导。“凤凰新闻社台湾总社”脸书公开表态支持韩国瑜。
 </p>

  <b>
   调查局：凤凰新闻社在中国辖区 但无法证实有中共资金
  </b>
  <br/>
  <br/>
  至于凤凰新闻社是否有来自中国方面的资金？台湾调查局资通安全处副主任刘家荣表示无法证实：“这个也有难度，因为它目前是香港凤凤新闻社，他的资金的查缉，目前已超越我们国家司法管辖的范畴，所以目前我们也没有具体的证据可以证明有中共的资金介入。”
  <br/>
  <br/>
  不过刘家荣说：“但是我们可以证明它是来自于中共的辖区香港，而且是散播对秘书长陈菊的一些不实的讯息。我们发布的目的是要澄清，希望国人不要被不管是大陆或香港一些不实讯息给误导或欺骗上当。”
  <br/>
  <br/>
  刘家荣说，目前无证据显示凤凰新闻社和凤凰卫视有任何关联。
  <br/>
  <br/>
  针对此个案，研究假讯息的台北大学犯罪学系助理教授沈伯洋21号接受自由亚洲电台采访说，将伺服器设在香港、马来西亚、新加坡攻击很常见，这是典型假讯息制造手法，重点散播管道是透过脸书、line或其他社交媒体，影响力的大小主要看这则“假讯息”是 “很假”还是“普通假”？
  <br/>
  <br/>
  以陈菊这则假讯息而言，沈伯洋说，假讯息产生效果是针对人性，有些人对陈菊本来就不满，假设他过去大量制造对陈菊不满的消息，这时丢出这样的假消息，只要对陈菊不满的都会信。
  <br/>
  <br/>
  沈伯洋：“他的目标群众就是本来就对陈菊不满的人，至于本来没有这个想法的人不太可能会直接受到这个影响。但是要说服本来就讨厌陈菊的人，要他不相信跟陈菊有关的假新闻 ，难如登天，因为他们看到就很开心，他们自动会转发，因为他们的意识型态就在那里，这是投其所好的新闻。”
  <br/>
  <br/>

  <b>
   学者：共产党将七、八成制造假讯息重心转到香港 影响舆论
  </b>
  <br/>
  <br/>
  沈伯洋分析，台湾也有自己的内容农场，两党都会制造假新闻互相攻击，原本数量也差不多，这是选战策略。但如果有中国共产党的加入，以民进党为目标的假新闻会比较多。
  <br/>
  <br/>
  不过沈伯洋发现，中国介入着力比较多的是立法委员选举，希望 扶植在各县市跟中国有往来的立法委员当选，蓝绿阵营的都有。地面上的谣言战，攻击很多的是民进党政府的不作为。
  <br/>
  <br/>
  沈伯洋对中共介入台湾总统大选，提出有趣的观察：“他们（中国共产党）对台湾，尤其是总统（竞选）的部份，有点半放弃的状况，因为以目前的民调，他们连要不要支持韩国瑜都不确定。他们攻击太过份的话可能会制造反效果，再加上香港现在又很忙，（蔡英文、韩国瑜）这个民调又拉的比较开，都是因素。”
  <br/>
  <br/>
  沈伯洋认为，中国共产党现在将七成、八成制造假讯息的重心，从针对台湾移到香港：“要怎么纠正其他的国家对香港负面的报导，是他们现在在着力的地方，他要的导向就是希望大家都假装中立，就是暴徒也很暴力、港警也很暴力，大家都很暴力，就是中立。所以你看现在国际媒体报导的方式好像变的有点中立，但其实很怪，我认为都是受到中国铺天盖地的影响。尤其你去观察他们用的推特和脸书的帐号，几乎全部、至少超过一半以上，（发的帖）都跟香港有关，都没有在针对台湾。  ”
  <br/>
  <br/>
  沈伯洋还说，台湾提升对假讯息的意识和防范，也会使制造和散播假讯息者减低对台湾这方面的投入。
  <br/>
  <br/>
  调查局指出，假讯息防制中心日前已查处多件中国记者张希达等散布假讯息案件，包括伪称“蔡英文卖台证据影片”及“蔡英文总统为中国共产党培植在台潜伏卧底人才”。调查局提醒民众在选举期间，散播假讯息，恐触犯选举罢免法有关意图使人当选或不当选的罪名，被判五年以下有期徒刑。
