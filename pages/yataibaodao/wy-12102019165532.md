 <p>
  美国国会在两个星期之内先后通过了针对中国维吾尔人和香港的人权法案，受到世界广泛关注。欧盟也于日前宣布将制定欧洲版的《马格尼茨基法》，严惩侵犯人权的行为。但在国际人权日，有美国国会议员表示，中国人权还有很多工作要做。
  <br/>
  <br/>
  美国国会众议院在12月10日国际人权日这天召开了中国人权研讨会。众议院外交事务委员会亚太小组资深共和党议员约霍(Ted Yoho, R-FL)在会上表示，
  <br/>
  <br/>
  “习近平要求中国共产党无所不用其极地去压迫人民。虽然中国共产党自身的性质已经越来越被世人所知晓，但他们大量打压人权的做法还被掩盖着。我们要做的就是提醒他们，这些做法应该停止。”
 </p>
 <p>
 </p>
 <p>
  实际上，出席本次研讨会的各位众议员和证人主要关心的问题，就是美国要如何做，才能影响中国共产党，使其停止打压人权。
  <br/>
  <b>
   <br/>
   美国政府应该如何做？
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>

   <div style="width:1632px;">

 </p>
 <p>
  华盛顿智库新疆问题专家郑国恩（Adrian Zenz）介绍了新疆强迫劳工生产的产品融入了国际产业链的情况。他就此向美国政府建议，
  <br/>
  <br/>
  “我呼吁对新疆强迫劳役的制度和手法，及其与美国商业公司的关系做深入调查。在对其中涉及的打压人权的做法找到证据后，下一步也应该在商业领域有所动作。”
  <br/>
  <br/>
  美籍维族人法卡特·乔达特(Ferkat Jawdat)讲述了他已经近两年无法和留在新疆的母亲取得联系。他感谢美国国会众议院通过了《维吾尔人权政策法案》，他同时还呼吁美国政府能继续支持美国的维吾尔人团体以及自由亚洲电台在维护维吾尔人权方面的工作。
  <br/>
  <br/>
  香港城市大学学生会副主席邵岚（Joey Siu）介绍了香港反送中运动的最新情况。她强调，香港政府正在制造一场危机。她呼吁美国政府在通过《香港人权与民主法案》之后，进一步向中国政府施加压力，
  <br/>
  <br/>
  “我希望美国政府能领导世界各国，共同确保政府遵守国际社会在人权方面的准则；我也建议美国政府能参与到国际社会的调查行动中，揭露香港政府对市民施加的野蛮行为。”
  <br/>
  <br/>
  非政府组织人权观察中国部主任索菲·理查森评价说，特朗普政府在人权方面采取了很多大胆的措施，相比以往历届政府都更加积极。但她也指出，包括美国国务院等政府部门，还可以采取更主动的方式来应对中国的人权问题。
  <br/>
  <br/>
  <b>
   中国向美国输出其人权标准
  </b>
  <br/>
  <br/>
  研讨会也特别关注了中国政府正把他们的人权标准输出到美国。约霍议员举例说，在上一届国会期间，他曾与台湾的代表举行会晤，但中国大使馆曾辗转告知他，希望他不要参与这次会晤。他感叹说，这是发生在美国的土地上，让人难以想象。
  <br/>
  <br/>
  人权观察中国部主任索菲·理查森（Sophie Richardson）指出，中国共产党正在侵蚀美国的大学，
  <br/>
  <br/>
  “应该要求美国的大学采取必要的措施，以确保校园里的学术自由不受中国政府的影响。但实际上，我们感觉到，美国五十个州的州立大学系统对这个问题还比较麻木。”
  <br/>
  <br/>
  她提到，人权观察曾向美国五十个州立大学发出信函，提醒他们中国政府在这方面的做法。约霍议员当即表示，将在这一项目上继续跟进，促使这些大学重视这个问题。
  <br/>
  <br/>
