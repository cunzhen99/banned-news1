 <p>
 </p>
 <p>
  台湾公立的中山大学惊爆有10多位涉及两岸研究的学者的电子邮箱被黑客入侵、监控长达3年。台湾政府表示，将进一步了解类似的情况，是否发生在其他的大学。
  <br/>
  <br/>
  台湾中央通讯社7日报导，台湾南部高雄的中山大学有10多位教授的电子邮箱被监控，包括政治学研究所所有教授，以及部份活跃于两岸、台湾公共事务的亚太所与政经系教授。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  中山大学政治研究所教授廖达琪接受自由亚洲电台采访，证实确有此事，她自己也感到相当惊讶。
 </p>
 <p>
  <br/>
  <b>
   受害教授：有老师收到
  </b>
  <b>
   “
  </b>
  <b>
   奇怪的信
  </b>
  <b>
   ”
  </b>
  <b>
   才曝光
  </b>
  <br/>
  <br/>
  廖达琪说：“我也是今天才知道我可能也是被暗中监督、窥探隐私的对象，新闻报导出去我才晓得。要不然平常我的信件都是助理先过滤，所以我没有特别感受。”
 </p>
 <p>
 </p>
 <p>
  <div class="image-inline captioned" style="width:622px;">
   <div style="width:622px;">
    <img alt="台湾中山大学政治研究所教授廖达琪（图源：百度百科）" src="https://www.rfa.org/mandarin/yataibaodao/gangtai/hx2-11072019092706.html/0df431adcbe.jpg" title="台湾中山大学政治研究所教授廖达琪（图源：百度百科）"/>
   </div>
   <div class="image-caption">
    <span style="width:622px;">
     台湾中山大学政治研究所教授廖达琪（图源：百度百科）
    </span>
    <span class="copyright">
    </span>
   </div>
  </div>
  <br/>
  廖达琪说，她认为学校网路应可挡掉可疑邮件，只要被归类到垃圾邮件，她一概不打开看。
  <br/>
  <br/>
  廖达琪说：“当然会觉得蛮可怕的，为什么我们的资安完全做不到这个事情（防堵），这不是很奇怪？”
  <br/>
  <br/>
  廖达琪透露，因为有老师收到奇怪的信件，向校方反应，这件事才曝光。
  <br/>
  <br/>
  至于所谓“奇怪的信”的内容，廖达琪说：“借用学校一个高层的信箱，叫他们去跟高层喝咖啡。后来他们发现是假的。那位高层并没有发这封信。”
  <br/>
  <br/>
  廖达琪谴责入侵的骇客，希望学校网路邮件在信息安全上能提供更高保障。
  <br/>
  <br/>
  <b>
   校方：清查有
  </b>
  <b>
   85
  </b>
  <b>
   个可疑帐户
  </b>
  <b>
   IP
  </b>
  <b>
   位置在香港、中国和美国
  </b>
  <br/>
  <br/>
  中山大学图资处处长赖威光接受自由亚洲电台采访分析说，骇客寄这封假信件的用意，是为扩大监看对象：“可能是一个钓鱼的邮件，他寄给第三者的时候，第三者可能还没有被他侵入，第三者点进去以后，可能就被他入侵。”
 </p>
 <p>
 </p>
 <p>
  <div class="image-inline captioned" style="width:808px;">
   <div style="width:808px;">
    <img alt="台湾中山大学图资处处长赖威光 （图源：中山大学图书馆－国立中山大学官网）" src="https://www.rfa.org/mandarin/yataibaodao/gangtai/hx2-11072019092706.html/lwg.jpg" title="台湾中山大学图资处处长赖威光 （图源：中山大学图书馆－国立中山大学官网）"/>
   </div>
   <div class="image-caption">
    <span style="width:808px;">
     台湾中山大学图资处处长赖威光 （图源：中山大学图书馆－国立中山大学官网）
    </span>
    <span class="copyright">
    </span>
   </div>
  </div>
  <br/>
  赖威光说，这位老师很警觉，认为他和那位高层不熟，对方应该不会直接写信给他，之后向那位高层求证，对方确实没发信给他，他随即通知校方和调查局展开调查。
  <br/>
  <br/>
  赖威光指出，清查结果，有85个帐户潜伏在系统里，异常存取，学校已全数清除。
  <br/>
  <br/>
  赖威光说：“这些异常的帐户IP都是来自香港、大陆、美国，这些帐户的异常行为就是他们会去看不是属于他们个人的档案，会去看别人的档案，我们就把它们停了。”
  <br/>
  <br/>
  <b>
   Open Webmail
  </b>
  <b>
   被全台湾校园广泛使用
  </b>
  <b>
  </b>
  <b>
   恐有更多学校受害
  </b>
  <br/>
  <br/>
  中山大学使用的Open Webmail，是一套开放原始码的免费电子邮件信箱程式，被台湾校园广泛使用。外界认为应不只中山大学受害。
  <br/>
  <br/>
  台湾主管两岸事务的大陆委员会发言人邱垂正7号在例行记者会上表示，中共极权专制长期打压不同意见，近年来，更将监控系统伸进校园里，透过监控师生的行为，扩大其政治的掌握力度，对于人权的侵害日益严重，目前国际社会已严重关切这件事情，并采取应对。
  <br/>
  <br/>
  邱垂正强调，建构国家安全防护网非常重要，特别针对两岸文教交流，会请教育部进行了解，并要求教育部和相关学校积极维护老师们的资讯安全。
  <br/>
  <br/>
  <br/>
  自由亚洲电台记者 夏小华  台北报导 责编：陈美华、申铧  网编：瑞哲
 </p>
