 <p>
  11月12日，美国国会参议院外交关系委员会主席、共和党人吉姆·里施（Jim Risch）在一个讲话中，强调了美欧双方联手对抗中国的挑战、建立跨大西洋应对方案的重要性。他还表示，参议院正在推动有关香港的立法。
 </p>
 <div>
 </div>
 <div>
  在演说中，里施主要探讨了美国和欧洲要维系他们共同建立的公正互惠、尊重人权的国际体系，联手应对中国的不公平贸易、不透明且充满腐败的投资、威胁他国主权的政治干涉、主导新兴科技的勃勃野心，以及对人权、自由等基本价值的重新定义。
 </div>

 <div>
  里施在美国华盛顿智库战略和国际研究中心举行的研讨会上介绍说，他的祖辈分别来自德国和爱尔兰，他将很多欧洲人视为远亲。然而今天，美国和欧洲面临一个混乱的世界，最大的威胁即来自中国：
  <br/>
  <br/>
  “多年以来我们第一次见证一些重大竞争对手的复兴：尽可能扩张的俄罗斯，中东的伊朗以及无处不在的中国。如果你不信，请周游世界，你会看到中国遍布各地的脚印。中国确实是我们全球性的对手而且代表了对美国和欧洲共同利益的重大挑战。”
  <br/>
  <br/>
  里施预计，中国很快将不再满足于主导政治或经济叙事，而是决心要制定标准，通过重新定义人权的概念或者主宰新兴科技如何被运用：“中国首要的外交目标是在2025年之前把自己置于国际体系的中心，并且确保国际体系根据自己的价值观、原则和目标运行。”
  <br/>
  <br/>
  欧盟在今年3月的一份声明中指出，把中国定性为关键技术领域的“经济竞争对手”和政治上的“系统性竞争对手”。外界普遍认为，这是欧盟首次联合调整对华政策。
  <br/>
  <br/>
  在新兴科技方面，里施强调要公平公正的竞争环境，警惕中国主导规则制定的野心和对知识产权的窃取。他对丹麦和捷克做出表率、拒绝华为5G技术表示赞赏,但是德国等国的犹豫不决让人担忧：“如果有足够多的北约国家允许华为进入（电信领域），我担心北约内部的情报共享将大打折扣并受到负面影响。”
  <br/>
  <br/>
  华盛顿战略和国际研究中心资深副总裁海瑟尔·康莱质疑美国是否为欧洲提供了华为之外的可靠选项。里施没有直接回答，而是强调中国在倾举国之力发展5G技术，美国很难做到，但是使用华为技术和设备带来的潜在危险高于一切。

  中国持续不断对知识产权的窃取让里施感到很无奈和愤怒，他说，“在我的家乡爱荷达州，美光科技（Micron Technology, Inc.）是全世界第二大记忆芯片制造商，其芯片技术被中国的国企所窃取。这发生在中国政府将获取芯片技术列为国家的头等大事之后。中国公司将偷来的技术申请了专利并且反而在中国法庭状告美光科技。这种偷窃不胜枚举。欧洲公司比如荷兰的艾司摩尔（ASML）也有相似的遭遇。”
  <br/>
  <br/>
  据荷兰《金融日报》报导，半导体设备制造商艾司摩尔(ASML)被几名研发部的中国雇员窃取商业机密，泄露给中国政府，致使公司亏损数亿美元。荷兰执政党随即呼吁政府在5G网络建设中禁用华为设备。
  <br/>
  <br/>
  在国际贸易方面，里施认为美国和欧洲的当务之急是推进世界贸易组织（WTO）的改革，此外也要加强对投资的筛查以及对关键技术的出口控制。
  <br/>
  <br/>
  美国总统特朗普于今年7月提出世贸组织应进行改革，解决中国等富有经济体可以自我定位为发展中国家从而享受特殊待遇的问题。中国外交部曾对此回应道，中国坚持发展中国家定位，不是回避应尽的国际责任，而是在主张发展中国家的基本权利。
 </div>

  里施也强调了加强关键性基础设施建设，特别是确保欧洲港口安全的重要性：“我们必须关注港口的网络空间安全，因为港口愈发依赖于机械化处理和数字化工业基础设施。充分理解中国获得欧洲港口将威胁跨大西洋地区的安全，是北约的当务之急。”
  <br/>
  <br/>
  目前，中资企业运营或持股十个以上的欧洲港口。习近平于11月10日至12日开展对希腊的国事访问，中希两国就中远海运集团投资希腊最大港口比雷埃夫斯港6亿美元的项目达成协议。
  <br/>
  <br/>
  关于如何共同应对来自中国的挑战，里施列出了三个指导性要点：第一，美国和欧洲要不能短视地把经济利益置于政治和安全考量之上。第二，美国和欧洲必须重新树立维护共同利益的坚定承诺，不要重蹈2014年北约对俄罗斯的挑衅准备不足的覆辙。第三，美国和欧洲必须认识到这是一场国际竞争，不仅为了美欧的公司利益抗争，也要努力在中国市场建立一个公平竞争的普遍规则和标准。
  <br/>
  <br/>
  《香港人权与民主法案》已于10月16日在美国众议院无异议通过，关于何时会在参议院表决，作为该法案的共同提案人和强烈支持者，里施表示他们在努力推进，本周三就有一场会议将会讨论该法案的日程。
  <br/>
  <br/>
  12月3日至4日，北约峰会将在伦敦召开。里施透露，一份针对中国的报告将在会上发布。
  <br/>
  <br/>
