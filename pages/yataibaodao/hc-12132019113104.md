 <p>
  美中两国政府周五相继表示，双方已达成了第一阶段贸易协议。美方将取消原定两天后生效的新一轮中国商品关税，中方则宣布将会取消部分美国商品关税、购买美国农产品、改善国内市场环境等等。不过，这项协议尽管历尽艰辛才达成，在正式签署之前仍存在变数。
  <br/>
  <br/>
  中国国务院新闻办公室周五深夜举行新闻发布会，证实了美中两国已经达成第一阶段贸易协议。商务部副部长兼国际贸易谈判副代表王受文在会上宣布了这条消息。
 </p>
 <p>
  <br/>

 <p>
  “经过中美两国经贸团队的共同努力，双方在平等和相互尊重原则的基础上，已就中美第一阶段经贸协议文本达成一致。”
  <br/>
  <br/>
  他介绍说，这份协议文本分为知识产权、技术转让、食品和农产品、金融服务、扩大贸易等九个章节。另外，美方还同意分阶段取消对华产品加征关税，实现了加征关税由升到降的转变。
  <br/>
  <br/>
  美国贸易代表办公室同日发布声明说，美中两国已经达成了一项“历史性的、可执行的”第一阶段贸易协议。这项协议要求中国进行经贸领域的结构性改革，也包括中方将在未来几年购买大量额外的美国商品和服务的承诺。
  <br/>
  <br/>
  旅美经济学者秦伟平认为，这项协议的很多细节尚未公布，其分量也是有限的。
  <br/>
  <br/>
  “我认为这项协议是美中双方基于各自的政治和经济需要，达成的框架性协议。它覆盖的类别很广，包括美国非常关注的知识产权保护方面，但是相关细节可能会非常少。我认为这是一项原则性指导协议。”
  <br/>
  <br/>
  美国原定本周日对包括手机、笔记本、玩具在内的1560亿美元的中国商品加征15%的关税，但美国总统特朗普周五在推特上表示，因为双方已经达成了协议，美国将不会加征新一轮关税。美方官员还在声明中表示，美国将保留对2500亿美元左右的中国商品加征的25%的关税，但美国将把此前对1200亿美元左右的中国商品加征的15%的关税下调至7.5%。
  <br/>
  <br/>
  <b>
   中方对协议细节闪烁其词
  </b>
 </p>

 <p>
  中国财政部副部长廖岷在新闻发布会上指出，鉴于美国已经承诺将取消部分关税，并会加大对中国商品的关税豁免力度，中方也将考虑不实施原定本周日生效的对美国商品加征关税的措施。记者尚不清楚“考虑不实施”这样的含糊措辞，是否意味着中方尚未正式决定取消对美商品加征新一轮关税。
  <br/>
  <br/>
  记者注意到，农产品采购是此前第一阶段贸易谈判的重点之一。特朗普表示，他要求中国采购500亿美元的美国农产品，但并未指明一个采购时间表。这个采购额相当于中国2017年的美国农产品进口额的两倍多。有业内人士指出，中国在短期内实现这样的采购目标并不现实。
  <br/>
  <br/>
  当被问及中方能否证实协议要求中国明年将从美国进口500亿美元的农产品，中方官员仅表示，协议实施后，中国将大幅增加从美国进口的农产品，但并未透露具体数字。
  <br/>
  <br/>
  秦伟平分析说，中国政府之所以没有公布农产品采购额，表明当局对这个采购目标仍然持有保留态度。
  <br/>
  <br/>
  “一方面，中国市场能不能消化掉500亿美元这么多的农产品？另一方面，中国有没有这样的购买能力？我们知道中国经济已经不处于上行周期了，官方公布的GDP数据都表明经济在下滑，而实际情况比官方数据可能还要惨。”
  <br/>
  <br/>
  此外，美国官员曾多次强调，两国达成的贸易协议必须包括执行机制，以确保中方履行承诺。美国贸易代表办公室周五发布的这份声明说，这项协议建立了一套强有力的争端解决机制，确保双方能够快速有效地实施并执行相关协议内容。
  <br/>
  <br/>
  值得注意的是，当被美国媒体问及中方能否证实这项协议包括严格的监督执行机制，中方官员周五并没有作出直接回应。中国财政部副部长廖岷表示，当务之急是签署并落实这项协议，至于第二阶段贸易谈判何时开始，将取决于第一阶段协议的落实情况。
  <br/>
  <br/>

   签署前仍存变数

 <p>
  美国加州企业家袁建斌指出，直到两国正式签署协议前，其文本仍然可能有变。
  <br/>
  <br/>
  “我认为在双方签署协议前，仍然会存在变数。这项协议受到了美国大选和经济环境的制约。对于中国来说，当局也受到了国内其他力量的一些制约。”
  <br/>
  <br/>
  中国商务部副部长王受文透露，双方将各自尽快完成法律审核、翻译校对等必要程序，并就正式签署协议的具体安排进行协商。
