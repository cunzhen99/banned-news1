 <p>
  12月10日是国际人权日，加拿大多伦多丶渥太华丶温哥华丶维多利亚等多个城市在刚过的周日都举行游行和集会，表达反抗中共打压人权，声援香港人持续抗争。
  <br/>
  <br/>
  8日中午，温哥华支联会在温哥华美术馆外举行的国际人权日集会活动，约400人参加，以响应《世界人权宣言》和国际人权日精神，反对中国继续打压人权民主，呼吁国际社会能制定类似美国的《香港人权及民主法案》和《维吾尔人权法案》，温支联主席李美宝表示，10日国际人权日当天，将有两位加拿大参议员将提出动议呼吁联邦政府要落实《马格尼茨基法案》来制裁违反人权的中国和香港官员，这是一个好的开始。
 </p>
 <p>
 </p>
 <div>
 </div>
 <div>
 </div>
 <div>
 </div>
 <div>
  “后续还有很多工作要做，我们要吁请更多国会议员支持动议，希望特鲁多政府能真正拿出行动来因应香港当前的困境。”
  <br/>
  <br/>
  此外，一批年轻的挺香港人士则在温哥华市中心竖立了一棵「催泪弹圣诞树」，让来往行人挂上小卡片，来悼念与祝福那些力争香港民主的人民。活动发起人陈艾玛说，小卡片上写着「催泪弹，中国制造」，希望更多市民知道香港政府已对当地人民射发了超过1万枚催泪弹，他们还在现场播放香港抗争活动和警方使用暴力执法的视频，以唤起加拿大市民对香港问题的关注。
  <br/>
  <br/>
  “路人看到有圣诞树，就会好奇说你们在干嘛？然後他们会看丶会去听，因为我们现场都有人拿着宣传品去介绍香港发生了什麽事。”
  <br/>
  <br/>
  长期支持民主运动的沈升表示，历史证明独裁专制的国家终究会垮台，别误以为现在中国经济强大就让其他国家禁声，但中共的野心霸权最终残害了全人类，所以国际社会不可能坐视不理。
  <br/>
  <br/>
  “(中国)经济输出的同时，它还在搞一种价值观的输出。它用反人类的活动来输出它的那种毒性，所以你不去碰它，它会来侵涉你的。这种毒瘤是人类必须面对的，必须根除的，不然就不堪设想。”
  <br/>
  <br/>
  在多伦多则有20个公民社会组织发起「国际人权日集会游行」，他们一致敦促总理特鲁多要展现领导能力，应对中国政权在中国境内和境外丶计划地压制人权的问题，组织者之一的港加联会长冯玉兰表示：“中国共产党是自由世界的最大威胁之一，加拿大需要站出来反对中共的压迫。”多伦多大学士嘉堡分校学生会会长丶西藏人齐美拉姆(Chemi Lhamo)也说，“加拿大不能再沉默了，因为今天西藏丶台湾丶香港和维吾尔族面对的情况，明天可能会是我们的处境。”
  <br/>
  <br/>
  自由亚洲电台记者  柳飞  / 责编：安克 网编：郭度
