 <p>
  美国国务卿蓬佩奥在柏林墙倒塌30周年前夕重返德国，批评中国共产党正打造“新独裁主义”，对中国人民的高压统治手段和当年的东德有“令人震惊的相似之处”。柏林围墙引发的当年东欧共产国家的解体浪潮，有可能在中国出现吗？
  <br/>
  <br/>
  1989年柏林围墙倒塌前夕，才20多岁的蓬佩奥是派驻西德的美军成员，协防西德抵抗高墙另一边的东德。

 <div>
  30年后，当年的美国大兵已是国务卿，来到统一后的德国首都柏林发表演讲。蓬佩奥回望过去冷战时民主与独裁的对峙，让他深有感触。他说，自己当时也没预料到铁幕与高墙的崩塌速度如此之快。蓬佩奥再次将炮口，对准中国。
  <br/>
  <br/>
  蓬佩奥：“中国正在创造一种新的威权主义，是一种这世界已经很久没见过的威权主义。中国共产党镇压自己人民所使用的方式与策略，和前东德‘有让人惊恐的相似之处’。”
  <br/>
  <br/>
  蓬佩奥在美国于北约的重要盟邦德国呼吁，西方自由国家应要团结一致，抵挡来自中国、俄罗斯和伊朗政府的威胁。他尤其提醒德国，在5G发展上，应该对中国企业华为投标德国的基础建设有所警醒。
  <br/>
  <br/>
  他还说，美国和盟邦不该把当年捍卫而来的自由与胜利，视为理所当然，要了解我们正与不自由国家进行价值观的竞争。
  <br/>
  <br/>
  <b>
   柏林墙的倒塌，美丽的错误
  </b>
 </div>
 <div>
  <b>
  </b>
  <br/>

 <div>
  分隔东、西柏林的围墙，是在中国天安门的血腥镇压后的1989年11月9日轰然倒下。
  <br/>
  <br/>
  穿越东德波尔豪摩史特斯检查站（Bornholmer Strasse）的车辆按响喇叭庆祝，在柏林围墙西边的人们手拉手，帮助东边的人翻越围墙，呼吸自由的空气。这些历史的珍贵瞬间，都是在当时东德政府官员夏波夫斯基（Gnter Schabowski）念出“私人赴国外旅游，现在不必有先决条件就可申请”后发生。
  <br/>
  <br/>
  美联社当年驻东德分社社长赖莫尔德（Frieder Reimold）重返柏林围墙，回想当年的那一晚，他至今仍难忘自己难以相信的感受。一场本以为是极为无聊的记者会，却让他见证改变历史的重大一刻。
  <br/>
  赖莫尔德：“（夏波夫斯基念的）消息非常重要，因为那表示，墙突然就要消失了，当时我也花了一点时间确认自己听到的是真的，墙真的要倒了吗？那堵致命地、该死的墙，真的要倒了？人们真的可以自由穿越了吗？放宽私人旅行，没有任何条件限制是什么意思？我还一度怀疑这是不是东德政府玩什么花样，会不会不算数？“
  <br/>
  <br/>
  历史的重大一刻就这么发生，东德民众如潮水般涌向边界，却不见东德军方拦阻。后来众多历史资料则显示，当时夏波夫斯基宣布的规定，其实是误读消息，造就了历史行的改变。本该隔天才实施的新规，原来是有其他相关细节限制。
  <br/>
  <br/>
  <b>
   “苏东波效应”何时到中国？
  </b>

 <div>
  这样美丽的错误，后来导致前苏联、东欧及波兰民主化的“苏东波效应”。“苏东波”有没有可能在21世纪出现于中国？六四事件的参与者、法国塞尔奇蓬多瓦兹大学教授张伦告诉自由亚洲电台，历史的长河漫漫，需要耐心，而他始终怀抱希望。
  <br/>
  <br/>
  张伦：“中共之所以今天能够在意识形态上宣传他的统治，并且有一些资源支撑这样的管制，事实上是得益于柏林墙的倒塌、得利于（柏林墙倒塌后的）这一波全球化。这波全球化反过来现在有新的变化，中共还会不会有那样的资源支撑他的墙？那是另外一个问题。西方可能都是在进入一个调整期，面对中国的一个调整期。我们在一个转型期，可能我们也不必急着立刻下结论。”
  <br/>
  <br/>
  向前的历史车轮能不能带动中国改变，悲观与乐观的看法都有。高龄89岁、柏林围墙倒塌当时的美国国务卿贝克近来在华盛顿的一场纪念柏林墙倒塌30周年的活动上说，民主国家必须持续高举自由与人权的价。30年前美国这样的坚持，引领世界与盟邦推倒铁幕，现在，也必须这样做。

  <br/>
