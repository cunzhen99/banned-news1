 <p>
  香港地方区议会选举，泛民主派压倒性胜出后，美国方面纷纷做出反应。美国总统特朗普（Donald Trump）26日说，相信中国国家主席习近平可以妥善处理香港局势。与此同时，美国国务卿蓬佩奥（Mike Pompeo）以及多位国会议员都对香港民主派表达了祝贺。长期关心中国民主法治发展的美国学者孔杰荣（Jerome A. Cohen）则说，期盼这是香港局势转好的开端。
 </p>
 <p>
 </p>
 <p>
  在白宫椭圆形办公室签署关于美国原住民的法案后，特朗普被问到对香港区议会选举结果的看法，他不忘提到美中贸易谈判进展，以及他与习近平的好情谊。
  <br/>
  <br/>
  特朗普：“我们与香港人民站在一起。你们也知道，我和习近平的关系良好，而非常重要的美中贸易谈判、可说是有史以来最重要贸易协议的谈判，现在到了最后关头，我想，会非常顺利的。但同时，我们也希望香港局势能有好的发展，我想习主席能办得到，我了解他，我知道他可以办得到。”
  <br/>
  <br/>
  《香港人权及民主法案》已经送到白宫，对于这项两党高度共识通过的法案，特朗普仍未明确表态是否签署生效。
 </p>
 <p>

 <p>
  而在国务院记者会上，国务卿蓬佩奥则强调，是否让法案生效，属总统权限，但他主动祝贺香港人民完成区议会选举的民主实践，也再次向北京喊话，要兑现“一国两制、高度自治”的承诺。
  <br/>
  <br/>
  蓬佩奥：“我们祝贺香港人民，在11月24日完成了自由、公平及和平的区议会选举。美国持续支持香港人民追求的民主价值及基本自由，这同时也是得到一国两制制度架构所保障的。”
  <br/>
  <br/>
  对香港选举表达祝贺的,还少不了美国国会里香港的坚定支持者，包括众议院议长佩洛西（Nancy Pelosi）与参议院多数党领袖麦康奈尔（Mitch McConnell），都在选举结束后第一时间推特发文表达祝福。
  <br/>
  <br/>
  然而，正躺在特朗普总统桌子上的香港法案让北京芒刺在背，不只高声反对，还有大动作施压。
  <br/>
  <br/>
  新华社26日报导，中国外交部副部长郑泽光25日召见美国驻华大使布兰斯塔德（Terry Branstad），就《香港人权及民主法案》提出交涉和抗议。报导还称，中方告诉布兰斯塔德，若无法阻止涉港法案成法，美国后果自负。
 </p>

  <b>
   孔杰荣：料中共内部激辩香港   忧民主海啸蔓延
  </b>
  <br/>
  <br/>
  而长期关注中国民主法治发展的纽约大学法学院教授孔杰荣接受自由亚洲电台访问时则表示。特朗普当然应该签署让法案生效，这是美国表态支持香港人民追求民主自由的重要象征。
  <br/>
  <br/>
  他对香港选举结果感到深受鼓舞，也乐观期盼这是解决香港当前危机的开端，关键仍在中南海内部不同意见的辩论，他们能不能解放思想，让香港的“一国两制”能成为坚守法治、迈向民主的良制示范。
  <br/>
  <br/>
  孔杰荣：“我相信，习近平现在在内部听到的意见，一定是极为冲突的。一定有一派人会告诉他，就放手给香港一些他们所要的！但另一派一定说，千万别这么做，因为一但放手，中国国内民众也会想要更多。共产党内部的争辩在于，一派会说，再等个27年吧！等到‘50年不变’的承诺到期，就不用再担心香港了；但是，另一派的关切会是，继续放任香港局势不管，受害的是共产党自己，会让香港问题比新疆还严重！这在内部一定有很大争论。”
  <br/>
  <br/>
  孔杰荣表示，在选举过后，包括北京、香港特区政府的反应都还算克制。他认为，这是解决香港局势的积极讯号；另一方面，香港民众透过投票，表达意见，证明民主仍是有效运作的机制，“数人头”是可以取代上街“比拳头”的暴力抗争。
  <br/>
  <br/>
  他说，尽管香港争取民主仍有很长的路要走，但他引用英国诗人亚历山大·蒲柏（Alexander Pope）的名言：“希望会在心头永远悸动着：人类从来不曾，却始终希冀蒙福。”（Hope springs eternal in the human breast：Man never is，but always to be， blest.）期待香港明天会更好，能进而影响中国。
  <br/>
  <br/>
  自由亚洲电台记者郑崇生华盛顿报导 责编：申铧 网编：郭度
