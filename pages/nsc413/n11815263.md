 【大纪元2020年01月23日讯】（大纪元记者周慧心综合报导）武汉新型肺炎持续扩散至中国大陆多个省市，今天（23日）半天增44个确诊病例，使全国确诊病例增至613人，其中重症95宗，死亡人数累计17人。武汉市宣布进入全面战时状态，今天上午10时起“封城”，全市公共交通暂停营运，市民如无特别原因不能离开。
</p>
<p>
 今天（1月23日）截至下午2时30分，半天内共有9个省市公布增加44个确诊病例，分别是：
</p>
<p>
 浙江新增17宗病例，其中5宗在杭州，该省累积至27宗病例；
 <br/>
 湖南再新增5宗病例，该省累计增至9宗；
 <br/>
 四川卫健委通报新增3宗病例，该省累计增至8宗；
 <br/>
 福建再多3宗病例，全省累计增至4宗；
 <br/>
 辽宁新增1宗病例，全省累计增至3宗；
 <br/>
 安徽新增8宗病例，累计增至9宗，其中六安、滁州、阜阳及亳州为当地首例确诊病例；
 <br/>
 贵州新增2宗病例，全省累计增至3宗；
 <br/>
 广西新增确诊3宗病例，全省累计确诊5宗；
 <br/>
 黑龙江新增2宗病例，全省累计2宗病例。
</p>
