 <p>
  中国最近出台的要求政府机构用国产科技产品替换外国货的措施引发广泛关注。在美中两国关系紧张的情况下，双方之间的科技战似乎越演越烈。
  <br/>
  <br/>
  继美国政府下令对中国电信巨头华为和中兴公司实施禁令后，中国也采取相应反制措施。
 </p>
 <p>
  <br/>
  <b>
   三年内全面使用国产科技产品
  </b>
  <br/>
  <br/>
  美国《华尔街日报》9日的报道引述消息人士透露，中国政府去年曾要求政府机构以及关键基础设施企业从2019年到2021年间，按照30%、80%、20%的比例，将信息技术采购合同分配给中国国内的供应商。
  <br/>
  <br/>
  对此，美国圣汤姆斯大学国际研究中心教授叶耀元分析说，北京官方此举的目的有两个。
 </p>
 <p>
  “第一就是给西方世界发出‘我不需要再依赖你们技术’的讯号。第二就是，在贸易战之下，中国国家主席习近平不愿对美国低头的话就需要采取一些措施。把外国科技产品替换就是和中国国内民众说不用担心贸易战，我们爱用国货；我们不需要跟别人买；我们自己有。”
  <br/>
  <br/>
  报道引述中信建投证券股份有限公司估计，至2022年，将有约2,000万到3,000万台外国产计算机设备或被替换成国产货。
  <br/>
  <br/>
  美国纽约“美中科技文化交流协会”会长谢家叶接受本台记者采访时说，从上述数字能看出中国对外国科技的依赖程度。
  <br/>
  <br/>
  “我们要知道中国国内企业和政府部门计算机，尤其是软件部分都是进口的，在中国的使用率应该说是蛮大的。”
  <br/>
  <br/>
  <b>
   国产自我研发能行吗？
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>
  <div class="image-inline captioned" style="width:1500px;">
   <div style="width:1500px;">
    <img alt="华为技术有限公司（AFP）" src="https://www.rfa.org/mandarin/yataibaodao/jingmao/xql-04182019101507.html/000_Hkg7939617.jpg" title="华为技术有限公司（AFP）"/>
   </div>
   <div class="image-caption">
    <span style="width:1500px;">
     华为技术有限公司（AFP）
    </span>
    <span class="copyright">
    </span>
   </div>
  </div>
 </p>
 <p>
  中国近年来不断加强国内技术研发以摆脱对美国技术的依赖。尤其在美中贸易战爆发后，两国在科技方面的“较量”也越发激烈。夹在这场科技竞争中的中国企业难免遭遇不少挑战。
  <br/>
  <br/>
  《华尔街日报》的报道提到，美国政府早前对中兴通讯实施的制裁让中国意识到，虽然中国在信息科技领域取得不少进步，外国技术依旧是中国科技行业主心骨。
  <br/>
  <br/>
  尽管美国之后取消了对中兴的禁令，但华为公司、晋华集成、海康威视等中国公司却被列入出口管制的“实体名单”，需要和包括谷歌、高通、英特尔等科技公司暂停部分合作。大规模的技术断供迫使这些中国企业另觅出路。例如，为了替代安卓系统，华为自我研发的操作系统鸿蒙今年8月正式登场，成为华为存亡的关键技术。
  <br/>
  <br/>
  叶耀元认为，中国目前拥有一定的资源来进行研发，但“闭门造车”般的政策存在长远问题。
  <br/>
  “当中国企业开发出这个技术后是在国内市场处于垄断状态。在这种情况下，是无法确保这个开发出来的技术是最好的。可能刚开发出来能力不错，但由于缺乏竞争关系，长时间之后这技术就会输给别人家的技术。”
  <br/>
  <br/>
  谢家叶也表示：“从短期利益来看，使用国内的产品好像是给本国企业撑腰。但这些技术是否足够先进？对使用者来说是不是得心应手？不进口外国技术的话，你是否能了解未来科技的发展趋势？”
  <br/>
  <br/>
  从过去几十年的观察来看，中国一直通过复制外国科技来进行自我研发。谢家叶对中国在与外界切断交流后，是否可以靠自身力量发展出新的技术持怀疑态度。
  <br/>
  <br/>
  （记者：韩洁 责编：申铧 网编：郭度）
  <br/>
  <br/>
  <br/>
 </p>
