 <p>
  1.    香港公立何传耀纪念中学中五学生曾志健在10月1日的荃湾冲突中被警员开实弹射中，左胸受伤，胸骨骨折，一度生命垂危。民间记者会2日以「自由之心不死　奈何以枪毙之」为主题召开记者会，形容警方发射实弹是开了「谋杀的第一枪」，香港已进入战争状态。
  <br/>
  <br/>
  港警10.1开枪实弹伤人之后，台湾独派大老辜宽敏警告北京：“一颗子弹就可能让共产党倒台。”
  <br/>
  <br/>
  2.    本周一（9月30日）中共建政70周年日前夕，中共政治局常委习近平带领全体常委到毛泽东纪念堂朝拜，并向烈士纪念碑献上花篮及默哀。这是习近平上任后第二次朝拜这位中共元老。
  <br/>
  <br/>
  但之前的9月25日北京举行的一次表彰会议上，习近平给包括张志新，荣国团等文革政治迫害死难者在内的278名个人颁发了“最美奋斗者”证书。
  <br/>
  <br/>
  紧接着，新华网27日发文《1966年：“文化大革命”十年内乱开始》，批毛发动的文革“非常严重” “他当时对党和国家的政治状况的错误估计，已经发展到非常严重的程度”。文革“不是、也不可能是任何意义上的革命或社会的进步”。
  <br/>
  <br/>
  3.    中国狱中维吾尔族学者伊力哈木·土赫提(Ilham Tohti) ，9月30日获得欧洲理事会2019年哈维尔人权奖。欧洲理事会宣布，哈维尔人权奖在欧洲理事会秋季复会首日举行的特别仪式上，共同授予中国维吾尔族学者伊力哈木·土赫提和致力于巴尔干地区民族和解的“青年人权运动”（YIHR）。

  4.    中国著名自由派学者、北京大学法学教授贺卫方的微信账号周四遭永久封号。
  <br/>
  <br/>
  中央社报道，贺卫方的微信账号的封号类型是“限制登陆，不可解封，”被封原因是“传播恶性谣言等违法违规内容”。
  <br/>
  <br/>
  2017年5月，贺卫方的个人微博账号被禁言，但他至今仍然拥有近184万名粉丝。
  <br/>
  <br/>
  《纽约时报》去年的一篇专题报道形容他是一位“在政治风暴中等待时机的异见学者”。贺卫方当时表示，他目前连在学术刊物上发表论文都变得困难，而过去五年尤其压抑。
  <br/>
  <br/>
  5.    十一阅兵结束后，天安门广场首次出现「一国两制」方阵，香港花车上竖立着香港特区基本法，游行队伍边举着「明天会更好」的牌子，台湾花车的大银幕不断播出「实现伟大民族复兴 推进祖国和平统一」的字幕。
  <br/>
  <br/>
  根据先前媒体报导，参加「一国两制」方阵的港澳台人士约有2000人，多为在内地升学的港澳学生。
