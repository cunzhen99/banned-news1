 <p>
  《联合国人权宪章》作为一份旨在维护全人类每个人都应该享有的基本权利的文件，于1948年12月10日在法国巴黎签署，并在全球推行，至今70多年了。当时参与起草的中华民国的代表张彭春、美国的爱里诺· 罗斯福、加拿大的约翰·彼得福·汉弗莱、法国的瑞纳·赛缪尔·卡心、黎巴嫩的查尔斯·马利克等作者如今已经辞世，物是人非了。我们无法知道，他们对《联合国人权宪章》今天所取得的成效和如今世界面临的人权挑战作何感想。可是我们却知道，中共党国体制利用现代科技践踏人权，肯定是他们当时没有想到的新的人权问题。
  <br/>
  <br/>
  去年七月，我有幸受邀参与了纪念《联合国人权宪章》实施70年的纪录片在纽约的拍摄。该纪录片从全球的角度，向世人展示了目前世界各地的人权状况。其德语版已经于去年12月10日的“国际人权日”按时完成制作并发布，其英文版名叫《Inviolable：The Fight for Human Rights》， 于本周二也就是2019年10月1日在纽约公开放映。我接受德国驻联合国大使 Ambassador Christoph Heusgen 之邀，一起欣赏了这部震撼人心的影片。之后由Christoph 主持、提问，和参与该片采访与制作的德国最著名电视主持人Claus Kleber、该片的导演Angela Andersen、“人权观察”总执行人及非洲代表一起，就《人权宪章》已经起到的重要作用，取得的成效及目前与未来面临的挑战和工作重点应该是什么等问题，进行了对话讨论。

 <p>
  70多年，在人类发展的历史长河中，不过是转瞬之间的事。可是对于具体的许多个人来讲，却是从起点到终点的一生经历的时间。尽管联合国由于它的低效、腐败与比较容易被中共这样的大独裁者拉帮结伙进行破坏等，有许多需要改革、完善的地方，目前饱受批评，甚至有人呼吁解散它。可是就联合国人权宪章而言，70年来，毋庸置疑地在对全球许多国家的法律制定和人权保护、人道救助上，都起到了很大的作用。
  <br/>
  <br/>
  我个人认为，目前的人权迫害归纳起来，最主要也是最重要的两点是由于制度性的社会不公和有组织犯罪所造成。从方法论的角度，要解决制度性社会不公造成的人权迫害，可以从建立公平，公正的社会制度 - 推行民主，建立宪政法治着手解决；而解决有组织犯罪的难点，则是在于如何解决犯罪集团不断阻止、破坏人们建立公平的社会制度。尤其是像中共这样的反人类组织掌握了国家政权后，为了维护其极权专制，不断以国家的名义利用国家机器持续的、有针对性地进行大规模的人权迫害，阻碍国家实现民主。要解决这个问题，就必须打击独裁者。
  <br/>
  <br/>
  因此，民主国家联手在全球推行民主，协助建立宪政法治体系，特别是打击专制独裁，消灭反人类政权是达到《联合国人权宪章》想要达到的目的的最直接和从根本上解决人权问题，无法回避的途径与办法。只要专制依然存在，人权就得不到真正的尊重与保障。
  <br/>
  <br/>
  总之，专制不除，人民难安。人类去除野蛮的极权专制、实现真正维护每个人人权的革命尚未成功。面对极权对人权的无底线侵犯，此时时刻，世界各国人民必须携手努力才是正确的选择。
