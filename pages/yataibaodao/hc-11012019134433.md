 <p>
  美国国防部长最近在澳大利亚出席外交会议时说，中国正通过一系列手段破坏印太地区稳定，而美国认为任何一个国家都不能支配整个地区。在这样的背景下，澳大利亚国防部长周三表示，中国在这一地区的影响力显然在上升，但中国仍是一个重要的合作伙伴。
 </p>
 <p>
  自从特朗普就任美国总统后，华盛顿开始频繁使用“印太地区”来取代“亚太地区”，并通过立法等手段强化其在这一地区的领导地位。新上任不久的美国国防部长埃斯珀 (Mark Esper)8月在美澳年度部长级会议上说，美国坚决反对中国在印太地区令人担忧并具有侵略性的破坏稳定的行为，引发中方的驳斥。
  <br/>
  <br/>
  当时出席了这场会议的澳大利亚国防部长琳达*雷诺兹(Linda Reynolds)周五在华盛顿保守派智库哈德逊研究所(Hudson Institute)表示，近十年来，印太地区的确出现了诸多安全风险，而中国军队现代化进程也格外引人注意。
  <br/>
  <br/>
  “朝鲜已经开发并试验了各种核武器和弹道导弹。我们还注意到，尽管各国公开承诺不会在南海争议水域进行军事化，这个趋势仍在持续。”
  <br/>
  <br/>
  2015年以来，中国军队展开了一场全面革新。当局除了进行大面积军备升级，也对军事结构进行了调整。习近平在中共十九大报告中提出，中国应力争在2035年基本实现国防和军队现代化，到本世纪中叶把中国军队全面建成世界一流军队。
  <br/>
  <br/>
  雷诺兹说，为了应对这些区域风险，美澳两国必须提升它们的军事威慑力。澳大利亚政府2016年初发布国防白皮书，计划在未来20年间投入近2000亿澳元(现约合1350亿美元)，用于扩充国防规模和装备。
  <br/>
  <br/>
  她表示，印太地区的局势正在发生深刻变化，伴随而来的各种安全风险为这片区域增添了不少火药味。
  <br/>
  <br/>
  “横跨三大洋，澳大利亚所处的战略环境正受到前所未有的挑战。另外，这些挑战超出了传统军事意义的范畴，包括经济胁迫、外交干预、雇用民兵、网络攻击等等。”
 </p>

 <p>
  雷诺兹强调，澳大利亚地处印太地区心脏部位。它西邻印度洋，东至太平洋，南靠南冰洋。就在亚太地区的各个角落，众多区域经济体将成为全球发展的重要引擎。如果按购买力平价(PPP)衡量的话，十年之内，全球前五大经济体中的四个可能就来自印太地区，包括中国、印度、日本和印度尼西亚，而澳大利亚一向是、并将持续是印太区域繁荣的受益者。
  <br/>
  <br/>
  与美中贸易近期受挫形成鲜明对比的是澳大利亚与中国越来越紧密的经贸伙伴关系。数据显示，中澳1972年建交时，两国总贸易额仅为8600万美元，而去年这一数字已经超过1500亿美元。值得注意的是，澳大利亚对中国有着近600亿美元的贸易顺差，向中国大规模出口包括矿石、煤炭、黄金在内的自然资源。
  <br/>
  <br/>
  正因如此，雷诺兹指出，澳大利亚寻求与中国建立更深入的合作关系，而这也会有利于提升区域安全。
  <br/>
  <br/>
  “中国的经济活力和中澳贸易的互补性促使我们两国自然而然地成为了经贸伙伴，这确保了两国各自的繁荣。这也包括在那些能为区域带来积极影响的领域中，扩展并深入我们的合作关系。”
  <br/>
  <br/>
  她还指出，除了中国之外，澳大利亚还特别重视与日本、印度尼西亚以及美拉尼西亚岛群的经贸和外交关系。在谈到与小国的关系时，雷诺兹强调，澳大利亚始终坚持尊重国家主权的原则，并确保其投资不会导致这些国家落入债务陷阱。
  <br/>
  <br/>
  不过，堪培拉近期也对中国释出另一个信号，体现出了两国的微妙关系。澳大利亚外交部长玛丽斯*佩恩(Marise Payne)周三表示，澳大利亚会追究中国在人权上应担负的责任。她指出，在这些敏感问题上保持沉默并不符合澳大利亚的国家利益。
  <br/>
  <br/>
  自由亚洲电台记者家傲华盛顿报道  责编：申铧  网编：洪伟
