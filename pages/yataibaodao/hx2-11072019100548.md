 <p>
  美国在台协会（AIT）7日在台北举行“美台安全合作对话”公共论坛。协会处长郦英杰表示，台湾面对北京威胁，并不孤单。台湾的国防安全研究院董事长霍守业演讲则说，台湾没有失职，台湾为了安全的存在尽了责任。
  <br/>
  <br/>
  美国在台协会7日在台北举办的“AIT@40数位对话”第三场公共论坛，聚焦在“安全合作”。
  <br/>
  <br/>
 </p>
 <p>
 </p>
 <p>
  美国在台协会处长郦英杰致词指出，中华人民共和国对台湾构成的不仅仅是军事威胁。除了在台海从事挑衅意味愈发浓厚的军事活动外，北京还针对台湾进行了各式各样非传统型态的威胁：包括一连串旨在破坏大众对民主体制信任感的不实讯息；在外交上施压，挤压台湾的国际空间，并限缩台湾参与国际社会的空间；另外还对台进行经济胁迫。
 </p>
 <p>
 </p>
 <p>

   AIT

  <b>
   处长：面对北京非传统威胁 需要非传统的应对
  </b>
 </p>
 <p>
  郦英杰认为，这样的非传统威胁，需要非传统的应对措施回应。美国方面向来鼓励并支持台湾努力采取创新及不对称的做法，使台湾及整个印太区域对此类威胁具备更强的抵御能力。
  <br/>
  郦英杰提到，推动美台安全合作并不仅限于美国与台湾之间。若能有理念相近国家与民主社会参与其中，将传达一个强烈讯息，那就是“凸显台湾并不孤单”。美台“全球合作暨训练架构”（GCTF）就是一个很好的例子。这类理念相近国家的支持，并不仅仅对台湾有益，也是维护美国自由且开放的印太区域这一愿景的关键因素。
  <br/>
  <br/>
  郦英杰强调，美台关系并非基于共同威胁，而是奠基于对共享价值的严正遵守之上，包括民主、包容、自由及法律之前人人平等，都是美台共享价值。今天的论坛证明了民主可以是政策创新的泉源，这与另一种限制自由、违反基于规则的国际秩序并遏阻创新的治理体系，形成鲜明对比。
  <br/>
  <br/>
  台湾的外交部常务次长曹立杰致词时指出，台湾跟美国也致力强化非传统领域的安全合作，共同举办20多场的工作坊，主题涵盖人道援助、网路安全、媒体识读。
 </p>
 <p>
 </p>
 <p>

  <br/>
  台湾的国防安全研究院董事长霍守业发表专题演讲强调，台湾现在很安全地存在，就是美台合作的结果。
 </p>
 <p>
  霍守业说：“我要告诉美国朋友，在美台安全合作上，我们台湾没有失职。因为我们今天安全地存在、好好的活着，我们尽到了我们的责任，我们也知道美国朋友给了我们很大的帮助，他们也尽了他们能够尽的责任。”
  <br/>
  <br/>
  霍守业表示，台、美两国“唇齿相依”，台湾在东亚第一线，处在前哨的位置，后面有一个强大力量支持，让台湾站在前面不畏风雨，就是美台合作的最佳写照。
  <br/>
  <br/>
  霍守业还提到，中共除了军事威胁，还有所谓的锐实力、网路战、资讯战、商战、舆论战、新闻战、法律战、统战，台湾希望继续在美国的支持下，能面对这些威胁屹立不摇。
  <br/>
  <br/>
  <b>
   台湾政委唐凤：假讯息“污染”言论环境 破坏对民主信心
  </b>
  <br/>
  <br/>
  与会的行政院政务委员唐凤接受媒体访问提到这次美国锁定“安全合作”作为论坛主题的重要性。
 </p>
 <p>
 </p>
 <p>
  <div class="image-inline captioned" style="width:916px;">

    </span>
    <span class="copyright">

   </div>
  </div>
  <br/>
  唐凤说：“这次讨论安全上的互相合作，网友们有一个共识是说，因为双方都深切珍惜民主自由跟人权价值，所以其实大家的合作是建立在捍卫共同理念上。资讯安全其实是一个很好的例子，因为如果我们不能确保底层网络不被渗透、不被侵犯、不被窃听的话，其实我们很难在上面发展言论自由、新闻自由和集会结社自由。”
 </p>
 <p>
  被问到台湾如何打击假讯息以保障言论自由，尤其台湾总统大选将在2个月后举行。
  <br/>
  <br/>
  唐凤说：“有非常多的朋友在社群网路扩散一些让人看了会对民主制度比较没有信心的言论。这并不是针对特定候选人、党派、特定政治主张，相反地，我现在才跟一些国外的研究者讨论这个问题，他们用的词是‘pollution’，就是把这个言论环境污染掉，让大家都不想讨论公共议题。这样的话，自然就让民主变得比较没有意思。”
  <br/>
  <br/>
  唐凤指出，在人人都是“直播主”的自媒体时代，如何促成更健康、不破坏民主的公共议题管道“就事论事”？台湾正在学校、社区加强推动媒体素养，希望营造一个“与恶的距离不要那么短”的社群网络环境。
  <br/>
  <br/>
  自由亚洲电台记者 夏小华  台北报导 责编：陈美华、嘉远  网编：瑞哲
 </p>
