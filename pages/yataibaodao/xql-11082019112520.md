 <p>
  已经持续了一年的美中贸易纠纷，最近频频传出双方达成阶段性协议的消息。不过，有关美国是否同意把已经开征的关税撤销的问题，仍然存在极大争议。
  <br/>
  <br/>
  美中贸易战已经进行了16个月，对两国经济都产生了伤害。一个月前，美中高级别谈判之后，传出两国可能签署一个第一阶段协议，以缓解经贸上的紧张关系。按照原来的计划，美中两国领导人应该在智利举行的亚太经合组织峰会上见面并作最后拍板。然而智利因政局动荡宣布放弃主办该次峰会。
 </p>
 <div>
 </div>
 <div>
  美国白宫发言人格里沙姆(Stephanie Grisham)星期四对媒体透露，美国对与中国达成协议“非常乐观”。
  <br/>
  <br/>
  中国商务部则表示，美中两国已同意分阶段取消关税。路透社说，一位不具名的美国官员证实，分阶段取消关税，是两国第一阶段协议的重要一环。
 </div>
 <div>
  <br/>

 <div>
  在此之前，中国一直坚持美国必须取消之前加征的关税，才会与美国达成协议。因此最新的官方表态，显示中国作出了让步妥协。
  <br/>
  <br/>
  但是美国白宫内部对取消关税存在很大争议。中央社报道说，特朗普总统重要幕僚白邦瑞（Michael Pillsbury）表示，分阶段取消关税问题，哪些关税？什么时候取消？美方尚未有具体共识。他认为中方有些一厢情愿，可能是为了安抚国内的强硬派。
  <br/>
  <br/>
  《纽约时报》说，特朗普政府内部支持撤关税的人认为，成功达成协议，可为总统带来政治胜利；而反对者警告说，在中国未作更多实质承诺前就撤回关税，意味着向中国屈服。
  <br/>
  <br/>
  白宫贸易顾问、被视为鹰派人物的纳瓦罗（Peter Navarro）说，以撤销关税达到达成协议的目的，在白宫内部“目前尚无共识”，而唯一能够拍板决定的，是特朗普总统本人。
