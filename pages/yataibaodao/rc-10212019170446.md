 <p>
  总部位于华盛顿的人权组织公民力量 21日向有“香港坦克人”称号的牧师安东尼，以及香港非政府组织民间人权阵线（简称：民阵）颁奖。美国共和及民主两党都有议员参加颁奖典礼，重申美国国会支持香港人民为民主自由而战的立场。
  <br/>
  <br/>
  香港“没有领袖”的“反送中”运动迈入第5个月后，持续获得国际社会高度关注。公民力量21日在美国华盛顿国会山庄举行第14届族群青年领袖研习营之际，将今年的“公民力量奖”颁给香港“坦克人”安东尼及民阵。
  <br/>
  <br/>
  今年八月，安东尼在香港街头的一次抗议示威遭到警方袭击时，他手持雨伞，只身向香港警察下跪，企图拦阻警方暴力执法，却遭到港警一脚踹开的影象与照片，在网络上广为流传，让人们想起1989年在北京长安街上，只身阻挡坦克的王维林。安东尼因此获得“香港坦克人”的称号。
 </p>

 <p>
  公民力量创办人杨建利说，对曾参与1989年天安门事件的他而言，安东尼的勇气始终让他震撼与感动。而低调的安东尼没有出席颁奖仪式，而是由香港大专学界国际事务处的张昆阳代他领奖，并代为宣读得奖感言。
  <br/>
  <br/>
  “有谁会想到，这个普通市民的小故事，能登上这样的国际大舞台被讲述。但愿将这个奖项中所有的赞美与荣耀，归给每一位香港公民。”
  <br/>
  <br/>
  另一个获奖者是2002年成立的香港人权织民阵。民阵是雨伞运动以及这次反送中运动中，多次大规模游行集会的组织者。民阵召集人岑子杰不久前遭不明人士攻击受重伤，无法出席。代替他领奖的民阵前副召集人梁颖敏接受自由亚洲电台访问时说，岑子杰已经出院，但仍需要休养恢复，而这个时间点颁奖给民阵，她认为，对所有参与反送中运动的人来说，是非常大的激励。
 </p>

 <p>
  梁颖敏：“这个recognition（认可）不只是对民阵的认可，这是对所有香港人，不怕危险与辛苦站出来，坚持这个运动的认可。”
  <br/>
  <br/>
  长期以来就支持与关注中国民主化运动的美国联邦众议员民主党的麦高文（Jim McGovern）和共和党的史密斯（Chris Smith），也都出席颁奖典礼。虽然两人分属不同政党，但却异口同声地希望联邦参议院能快马加鞭将众院已经通过的《香港人权与民主法案》排入议程并投票，好尽快让美国总统签署成为法律 。
  <br/>
  <br/>
  麦高文：“通过《香港人权及民主法案》，是重要一步，但不会就此结束，更不是我们与那些在前线奋战的香港人民团结同在的终点。现在，是时候要求联邦参议院尽他们的职责，并且施压特朗普政府做正确的事。”
  <br/>
  <br/>
  史密斯：“我们是不会放弃的。众院已经通过了《香港人权与民主法案》。我们希望参院能快点排期、投票通过，然后送交给特朗普总统。而我相信，当法案呈给总统后，他会签署、让法案生效的。”
  <br/>
  <br/>
  参议院已于16日收到众院版的《香港人权与民主法案》。根据参院在官网上公布的工作议程，《香港人权与民主法案》排在第243案。按排期，本周参院应会优先处理如排在第2及第4顺位的美国海岸防卫队及国防部的预算案。
  <br/>
  <br/>
  截至发稿，参院公布的议程共有248个众院已通过的法案，排上参院工作日程表。
  <br/>
  <br/>
  按美国立法程序，法案须分别在参、众两院全院通过后，并协调出一致版本，才能交总统签署生效。
  <br/>
  <br/>
  公民力量主办的这届族群青年领袖研习营有来自中国大陆、香港、台湾、澳门等地20多 位青年人权工作者参加。他们将就人权工作接受培训和见习，并与美国国会议员互动交流。
  <br/>
  <br/>
