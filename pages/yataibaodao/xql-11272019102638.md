 <p>
  向澳大利亚投诚的中国间谍王立强，已经在中、港、台和澳大利亚引起轩然大波。中国政府称王立强是一名已经被判刑的诈骗犯，所持证件都是伪造的。官媒环球网更公布了一段据称是2016年福建省光泽县法院的庭审录像。不过有评论人士认为，这段模糊的经过剪辑的录像，疑点甚多。
  <br/>
  <br/>
  环球网公布的这段2016年的2分34秒庭审录像，被称为王立强的被告人，大部分时间背对镜头。中央社报道说，整段录像中法官、书记官、检察官、法警和律师都有出现，而被告人唯一的正面特写却画面模糊，虽然样貌与在澳大利亚投诚的王立强有几分相像，但真实性仍待辨认。
  <br/>
  <br/>
  <b>
   中国政府的说法疑点颇多
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>

 <p>
  录像明显经过大幅剪辑，内容主要为被告基本资料确认，以及法官宣判及被告人最后陈述。被告人最后诈骗罪成立，法官判处他一年三个月徒刑，缓刑一年半，并罚款一万元人民币。
  <br/>
  <br/>
  在美国的时政评论人士李洪宽认为，这段录像可信度不高。
  <br/>
  <br/>
  “以前这种录像都有正面特写，它这个没有，很模糊，因为背面很容易造假。而且声音也不清楚，所以没有什么可信度。”
  <br/>
  <br/>
  上周末，澳大利亚媒体公布王立强的采访，透露了他在香港为一家名为中国新创投资的企业工作，而该企业实际为中国军方情报机构。过去几年，王立强负责了一些在香港和台湾进行渗透的工作，主要是通过收买社交媒体，影响当地的舆论，以达到控制选举的目的。另外，他也透露曾参与香港铜锣湾书店老板李波的绑架工作。
  <br/>
  <br/>
  报道在港台和澳大利亚甚至西方世界引起高度关注。中国政府立即否认王立强的指控，并反控王立强是诈骗犯，甚至出示了相关判决书。
  <br/>
  <br/>
  美国评论人士横河表示，有关的判决书，在中国政府做出声明之前无法查到。
  <br/>
  <br/>
  “这个事出来第一天我就说，中国会开始抹黑、编故事。而且这个判决书，网友在网上查过，在官方公布前法院记录里没有，后来才出现的。”
  <br/>
  <br/>
  <b>
   台湾议员：王立强只是化名

 <p>
  台湾媒体报道说，台湾桃园市议员王浩宇证实，王立强案被正式报道之前，澳大利亚媒体曾找他核实情况，并透露这位叛逃的间谍，真实姓名并非王立强，王立强只是一个化名。
  <br/>
  <br/>
  不过到目前为止，所有采访过王立强的媒体，以及澳大利亚反情报机关都没有证实这一消息。
  <br/>
  <br/>
  李洪宽认为，王立强所说的中国新创公司应该确实是中国军情机关设立的企业，而王立强也确曾为他们工作，但他本人可能并非正式的谍报人员。
  <br/>
  <br/>
  “他未必有正式职位。这个间谍应该打个引号，但他可能确实做过这些事，很多细节都很清楚。”
  <br/>
  <br/>
  横河表示，谍报工作中决策和执行并非一个部门，王立强很可能只是一个中间的传信人而已。
  <br/>
  <br/>
  “他很可能是一个传信人，西方叫做‘cut off”,中国人叫保险丝。因为执行的那些人，不能知道真正老板是谁。”
  <br/>
  <br/>
  他也认为，王立强所揭露的事实部分经过澳大利亚方面的核实，因此具有相当的可信度。而由于中国新创公司老板向心和龚青目前都被台湾方面调查，相信日后会有更多的内幕被揭露出来。
